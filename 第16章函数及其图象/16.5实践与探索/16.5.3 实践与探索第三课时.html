16.5 实践与探索

16.5.3 实践与探索（第三课时）

学习目标

  1. 了解现实生活中变量间的近似函数关系，能根据数据用图象法求近似的一次函数表达式。
  2. 体会用经验数据拟合函数模型的思想方法，感受近似计算在实际问题中的应用。
  3. 进一步理解数形结合思想，培养分析和解决实际问题的能力。

课题导入

同学们坐过火车吗？细心观察会发现，铁轨之间都留有一道小小的缝隙。这是为什么呢？

原来，金属也会"热胀冷缩"——温度升高时体积会变大，温度降低时体积会缩小。如果铁轨不留缝隙，夏天受热膨胀，铁轨就会挤压变形，甚至引发事故！

那么，一种合金材料从 $-40^{\circ}\mathrm{C}$ 到 $60^{\circ}\mathrm{C}$，它的体积究竟会发生怎样的变化？这种变化能用我们学过的函数来描述吗？今天我们就来做一回"小小科学家"，用实验数据寻找近似的规律。

复习回顾

已知一次函数 $y = kx + b$ 图象上的两点 $(x_1, y_1)$、$(x_2, y_2)$，可列出方程组求出 $k$ 和 $b$。

待定系数法的关键：两点确定一条直线。

🔍 新知探索

问题3

为了研究某合金材料的体积 $V$ ($\mathrm{cm}^3$) 随温度 $t$ ($^{\circ}\mathrm{C}$) 变化的规律，对一个用这种合金制成的圆球测得相关数据如下：

$t/^{\circ}\mathrm{C}$$-40$$-20$$-10$$0$$10$$20$$40$$60$
$V/\mathrm{cm}^3$$998.3$$999.2$$999.6$$1000$$1000.3$$1000.7$$1001.6$$1002.3$

能否据此寻求 $V$ 与 $t$ 之间的函数关系式？

分析　在平面直角坐标系中描出各对数值所对应的点，我们发现，这些点大致位于同一条直线上，可知 $V$ 与 $t$ 之间近似地符合一次函数关系。我们可以用一条直线去尽可能地与这些点相贴近，求出近似的函数关系式。

如图16.5.4所示的就是一条这样的直线，较接近的点可考虑取 $(10, 1000.3)$ 和 $(60, 1002.3)$。你也可以将直线稍稍挪动一下，换上其他适当的两点，试一试。

O $t/^{\circ}\mathrm{C}$ $V/\mathrm{cm}^3$

图16.5.4

试一试，求出近似的函数关系式。

取点 $(10, 1000.3)$ 和 $(60, 1002.3)$，设 $V = kt + b$。

$$\begin{cases} 10k + b = 1000.3 \\ 60k + b = 1002.3 \end{cases}$$

两式相减得 $50k = 2$，$k = 0.04$。

代入得 $b = 1000.3 - 0.4 = 999.9$。

所以近似的函数关系式为：$V = 0.04t + 999.9$。

📐 概括总结

我们曾采用待定系数法求得一次函数和反比例函数的表达式。但是现实生活中的数量关系是错综复杂的，在实践中得到的一些变量的对应值，有时很难精确地判断它们有怎样的函数关系，需要我们根据经验分析，进行近似计算和修正，列出比较接近的函数关系式。

在问题3中，如果把直线稍稍挪动，取另外两点（如 $(-20, 999.2)$ 和 $(40, 1001.6)$），求出的近似函数关系式会有什么不同？

取 $(-20, 999.2)$ 和 $(40, 1001.6)$：

$$k = \frac{1001.6 - 999.2}{40 - (-20)} = \frac{2.4}{60} = 0.04$$

$$b = 999.2 - 0.04 \times (-20) = 999.2 + 0.8 = 1000.0$$

得 $V = 0.04t + 1000.0$，与之前的 $V = 0.04t + 999.9$ 非常接近，说明拟合结果比较稳定。

用图象法求近似函数关系式的步骤：

① 描点：在坐标系中描出各组数据对应的点；

② 拟合：观察这些点是否大致在一条直线上，用直线去贴近这些点；

③ 取点：在直线上选取两个较接近的点，用待定系数法求出近似表达式。

小刚观察了学校新添置的一批课桌椅，发现它们可以根据人的身高调节高度。他测量了一套课桌椅上的四档高度，得到如下数据：

椅面高 $x/\mathrm{cm}$$37$$40$$42$$45$
桌高 $y/\mathrm{cm}$$70$$75$$78$$82.5$

请你和同学一起讨论，研究 $y$ 与 $x$ 可能满足什么函数关系。

在坐标系中描出点 $(37,70)$、$(40,75)$、$(42,78)$、$(45,82.5)$，这些点大致在同一条直线上。

取 $(37,70)$ 和 $(45,82.5)$，设 $y = kx + b$：

$$k = \frac{82.5 - 70}{45 - 37} = \frac{12.5}{8} = 1.5625$$

$$b = 70 - 1.5625 \times 37 = 12.1875$$

近似函数关系式：$y = 1.5625x + 12.1875$（或取整为 $y \approx 1.56x + 12.2$）。

  • 数学建模：将实际问题中的数据抽象为函数模型，用一次函数近似描述变量关系。
  • 数据分析：通过对实验数据的观察和分析，判断变量间可能存在的函数关系。
  • 应用意识：体会数学建模在科学研究、工程技术中的重要作用。

课堂小结

  1. 现实问题中的数据往往存在误差，变量间的关系通常是近似的。
  2. 用图象法求近似函数关系式：描点 → 拟合直线 → 取点 → 待定系数法。
  3. 选取的点不同，得到的近似表达式可能略有差异，但应力求接近实际情况。
  4. 近似计算和修正是科学研究中常用的方法。

  1. 整理课堂笔记，归纳用图象法求近似函数关系式的步骤。
  2. 根据问题3中的数据，取另外两组不同的点求近似函数关系式，并与 $V = 0.04t + 999.9$ 比较。
  3. 调查生活中的一组相关数据（如身高与鞋码、气温与用电量等），尝试用一次函数近似描述它们的关系。
