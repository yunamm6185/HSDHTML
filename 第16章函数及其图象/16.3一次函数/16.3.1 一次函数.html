16.3.1 一次函数

🎯 学习目标

1理解一次函数和正比例函数的概念
2能正确判断一个函数是否为一次函数或正比例函数
3理解正比例函数与一次函数的关系
4能根据实际问题建立一次函数关系式

🚗 暑假出行计划

前面我们学习了函数的概念，知道函数是描述变量之间对应关系的数学工具。在生活中，很多问题的两个变量之间存在着线性关系：

  • 汽车匀速行驶时，行驶路程与时间的关系
  • 弹簧挂上重物后，伸长量与重物质量的关系
  • 每月固定存钱，存款总额与月份的关系

这些都可以用我们今天要学习的一次函数来刻画！

📐 实际问题探究

📌 问题1 —— 汽车行驶问题

暑假里小明的爸爸带领全家去北京自驾游。汽车驶上A地的高速公路后，小明发现汽车匀速行驶的速度是95 km/h。已知A地直达北京的高速公路全程为285 km，小明想知道汽车从A地驶出后，距北京的路程和汽车在高速公路上行驶的时间有什么关系，以便根据时间估计自己距北京的路程。

分析：汽车距北京的路程随着行车时间的变化而变化。要想找出这两个变化着的量之间的关系，并据此得出相应的值，显然，应该探究这两个变量之间的函数关系式。为此，我们设汽车在高速公路上匀速行驶的时间为 $t$ h，汽车距北京的路程为 $s$ km，则不难得到 $s$ 与 $t$ 之间的函数关系式：

$$s = 285 - 95t \quad \text{①}$$

📌 问题2 —— 弹簧伸长问题

弹簧下端悬挂重物，弹簧会伸长。弹簧的长度 $y$ (cm) 是所挂重物质量 $x$ (kg) 的函数。已知一根弹簧在不挂重物时长6 cm。在一定的弹性限度内，每挂1 kg重物弹簧伸长0.3 cm。求这个函数关系式。

解：因为每挂1 kg重物弹簧伸长0.3 cm，所以挂 $x$ kg重物时弹簧伸长 $0.3x$ cm。又因为不挂重物时弹簧的长度为6 cm，所以挂 $x$ kg重物时弹簧的长度为 $(0.3x + 6)$ cm，即有

$$y = 0.3x + 6 \quad \text{②}$$

这就是所求的函数关系式。（其中自变量 $x$ 的取值范围由问题的"弹性限度"确定）

💭 想一想

问题1、问题2中得到的两个函数关系式①和②有什么共同点？

共同点：

① 函数关系式都是用自变量的一次整式表示的

② 都可以表示为 $y = kx + b$ 的形式（①式可化为 $s = -95t + 285$）

③ 自变量的次数都是1

④ 表达式都是关于自变量的整式

📚 概念形成

📖 概括 —— 一次函数的定义

上述函数关系式都是用自变量的一次整式表示的，我们称它们为一次函数(linear function)。

一次函数通常可以表示为 $y = kx + b$ 的形式

其中 $k$、$b$ 是常数，$k \neq 0$

特别地，当 $b = 0$ 时，一次函数 $y = kx$（常数 $k \neq 0$）也叫做正比例函数(direct proportional function)。

⚡ 重要关系：正比例函数是一次函数当 $b=0$ 时的特殊情况

问题1、问题2中得到的函数，都是一次函数。

📝 练习
1

仓库内原有粉笔400盒。如果每个星期领出36盒，求仓库内余下的粉笔盒数 $Q$ 与星期数 $t$ 之间的函数关系式。

解：原有粉笔400盒，每星期领出36盒，$t$ 星期领出 $36t$ 盒。

余下粉笔盒数：$Q = 400 - 36t$

即函数关系式为：$Q = -36t + 400$

（其中 $t \geq 0$ 且 $400 - 36t \geq 0$，即 $0 \leq t \leq \frac{100}{9}$）

2

今年植树节，同学们种的树苗高约1.80 m。据介绍，这种树苗在10年内每年长高约0.35 m。求树高(m)与年数之间的函数关系式，并算一算4年后这些树约有多高。

解：设树高为 $h$ m，年数为 $x$ 年。

树苗原高1.80 m，每年长高0.35 m，$x$ 年长高 $0.35x$ m。

函数关系式：$h = 0.35x + 1.80$（$0 \leq x \leq 10$）

当 $x = 4$ 时：$h = 0.35 \times 4 + 1.80 = 1.40 + 1.80 = 3.20$ (m)

答：4年后这些树约有 3.20 m 高。

3

小敏的爸爸为小敏存了一份教育储蓄。首次存入1万元，以后每个月存入500元，存满3万元为止。试用函数关系式刻画存款增长的规律，并求经过多少个月可存满全额。

解：设存款总额为 $y$ 元，存入月数为 $x$ 个月。

首次存入10000元，每月存入500元，$x$ 个月存入 $500x$ 元。

函数关系式：$y = 500x + 10000$

存满3万元时：$500x + 10000 = 30000$

$500x = 20000$

$x = 40$

答：经过 40个月 可存满全额。

4

以上3道题中的函数都是一次函数吗？为什么？

答：都是一次函数。

理由：

第1题：$Q = -36t + 400$，符合 $y = kx + b$ 形式，$k = -36 \neq 0$，是一次函数

第2题：$h = 0.35x + 1.80$，符合 $y = kx + b$ 形式，$k = 0.35 \neq 0$，是一次函数

第3题：$y = 500x + 10000$，符合 $y = kx + b$ 形式，$k = 500 \neq 0$，是一次函数

这三个函数表达式都是用自变量的一次整式表示的，且自变量的系数都不为0，所以都是一次函数。

📖 读一读

📜 一次函数的发展简史

一次函数的概念源于人类对线性关系的认识。早在古埃及时期，人们就用比例关系来计算土地面积和粮食分配。17世纪，法国数学家笛卡尔创立了解析几何，将代数与几何联系起来，一次函数的图象——直线，成为研究变量关系的基本工具。

一次函数是最简单的函数类型，但它却是理解更复杂函数（如二次函数、指数函数）的基础。在物理学中，匀速直线运动的路程-时间关系就是一次函数；在经济学中，成本-产量的线性关系也常用一次函数描述。

🌍 生活中的一次函数

一次函数无处不在，看看这些例子：

📱
手机话费：月租费 + 通话费 × 通话时长（$y = kx + b$）
🚕
出租车费：起步价 + 单价 × 里程（$y = kx + b$）
💧
水费电费：基本费 + 单价 × 用量（$y = kx + b$）
🌡️
温度换算：$F = 1.8C + 32$
🏷️
折扣计算：实付金额 = 原价 × 折扣率（正比例函数，$b = 0$）

🧠 课堂小结

  • 一次函数：形如 $y = kx + b$（$k$、$b$为常数，$k \neq 0$），自变量次数为1
  • 正比例函数：形如 $y = kx$（$k \neq 0$），是一次函数当 $b = 0$ 时的特殊情况
  • 判断方法：① 是否为整式；② 自变量次数是否为1；③ 自变量系数是否为0
  • 实际应用：很多生活中的线性关系都可用一次函数表示

常数 $k$ 和 $b$ 决定着函数的不同特征。这就像人生——选择和起点同样重要。
