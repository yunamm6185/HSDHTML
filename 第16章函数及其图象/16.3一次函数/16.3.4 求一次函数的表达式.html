第16章 函数及其图象

16.3.4 求一次函数的表达式

🎯 学习目标

1理解待定系数法的思路和步骤
2能用待定系数法求一次函数表达式
3能从实际情境中建立一次函数模型
4理解"两点确定一条直线"的原理
🌟 情境导入

我们已经学习了一次函数的图象——它是一条直线．已知直线，我们能画出图象；反过来，已知直线上的一些信息（比如两个点的坐标），能不能求出这条直线所对应的一次函数的表达式呢？

本节课，我们就来研究这个问题——用待定系数法求一次函数的表达式．

🔁 知识回顾

复习：一次函数的一般形式为 $y = kx + b\;\;(k \neq 0)$，其中：

  • $k$ 是一次项系数，决定直线的倾斜方向和倾斜程度；
  • $b$ 是常数项，决定直线与 $y$ 轴的交点 $(0,b)$；
  • 确定一次函数表达式，实质就是确定 $k$ 和 $b$ 这两个系数的值．
📖 例题讲解

▶ 例 4  华氏温标与摄氏温标的换算

世界大部分国家都采用摄氏温标预报天气，但美国、英国等国家则采用华氏温标．在研究性学习活动中，某小组同学查阅到以下资料：

在 1 标准大气压下，把冰水混合物的温度规定为 0 摄氏度，记作 0℃；把沸水的温度规定为 100 摄氏度，记作 100℃．

在 1 标准大气压下，把冰水混合物的温度规定为 32 华氏度，记作 32°F；把沸水的温度规定为 212 华氏度，记作 212°F．

设某一时刻温度计上的华氏温度为 $y\,(°\text{F})$，摄氏温度为 $x\,(°\text{C})$，已知 $y$ 是 $x$ 的一次函数，试写出这个一次函数的表达式．

分析：已知 $y$ 是 $x$ 的一次函数，可设表达式为

$y = kx + b\quad(k \neq 0)$

问题转化为：根据已知条件，求出 $k$ 和 $b$ 的值．

📋 已知条件：

冰水混合物

$x = 0$ 时，$y = 32$

沸水

$x = 100$ 时，$y = 212$

将这两组值分别代入 $y = kx + b$，得到关于 $k$、$b$ 的二元一次方程组，即可求出 $k$ 和 $b$．

💡

想一想：式中的一次项系数 1.8 和常数项 32 有怎样的实际意义？

解 设所求一次函数的表达式为 $y = kx + b\;(k \neq 0)$，根据题意，得

{

$0 \cdot k + b = 32,$

$100k + b = 212.$

解这个方程组，得

{

$k = 1.8,$

$b = 32.$

所以，所求一次函数的表达式为

$y = 1.8x + 32.$

这种先设待求函数的表达式（其中含有待定系数），再根据条件列出方程或方程组，求出待定系数，从而得到所求结果的方法，叫做待定系数法（method of undetermined coefficient）．

💡 解释"想一想"：

在求得的函数的表达式中，一次项系数 1.8 表示摄氏温度 $x$ 每增加 1 摄氏度时华氏温度增加的度数；常数项 32 表示摄氏温度为 0 摄氏度时所对应的华氏温度的度数．

📌 待定系数法的步骤

设 → 代 → 解 → 写

1
设出含待定系数的表达式
→
2
代入已知条件
→
3
解方程（组）求系数
→
4
写出最终表达式

说明：正比例函数 $y = kx$ 只含 1 个待定系数，已知 1 个条件即可求出；一次函数 $y = kx + b$ 含 2 个待定系数，需要 2 个条件（即已知两点坐标）．

✏️ 做一做

🖊 做一做

已知一次函数 $y = kx + b$ 的图象经过点 $(-1,\;1)$ 和点 $(1,\;-5)$，求当 $x = 5$ 时的函数值．

解：设 $y = kx + b\;(k \neq 0)$，根据题意，得

{

$-k + b = 1,\quad\cdots①$

$k + b = -5.\quad\cdots②$

①＋② 得：$2b = -4$，所以 $b = -2$；

代入①得：$-k + (-2) = 1$，所以 $k = -3$．

∴ 一次函数的表达式为 $y = -3x - 2$．

当 $x = 5$ 时，$y = -3 \times 5 - 2 = -15 - 2 = \boldsymbol{-17}$．

💬 讨论

🗣 讨论

（1）在上面的"做一做"中，已知条件是一次函数图象上两个点的坐标，它反映了自变量 $x$ 与因变量 $y$ 的值之间怎样的对应关系？

（2）题目中并没有要求写出函数的表达式，解题时却通常首先求出函数的表达式，它在这里起了什么作用？

（1）两个点的坐标给出了两组 $x$ 和 $y$ 的对应值，也就是给出了 $k$、$b$ 满足的两个方程，从而可以唯一确定一次函数的表达式．这体现了"两点确定一条直线"的几何原理．

（2）虽然题目只要求求函数值，但先求出表达式可以建立起 $x$ 与 $y$ 之间的一般关系式，然后代入任意 $x$ 值都能快速求得对应的 $y$ 值，是一种先"建模"再"使用"的策略，体现了数学建模的思想．

📝 练习

📋 练习

1．已知一次函数的图象如图所示，写出这个函数的表达式．

（第 1 题）

解：由图可知，直线过点 $(2,\,0)$ 和 $(0,\,-3)$．

设 $y = kx + b\;(k \neq 0)$，代入两点坐标，得

{

$2k + b = 0,$

$b = -3.$

由第二个方程得 $b = -3$，代入第一个方程得 $k = \dfrac{3}{2}$．

∴ 函数表达式为 $y = \dfrac{3}{2}x - 3$．

2．写出两个一次函数，使它们的图象都经过点 $(-2,\;3)$．

解：设 $y = kx + b\;(k \neq 0)$，将 $(-2,\,3)$ 代入得 $-2k + b = 3$，即 $b = 2k + 3$．

取 $k = 1$，则 $b = 5$，得 $y = x + 5$；

取 $k = 2$，则 $b = 7$，得 $y = 2x + 7$．

满足条件的一次函数有无数个，写出其中两个即可．

3．温度计是利用水银（或酒精）热胀冷缩的原理制作的，温度计中水银（或酒精）柱的高度 $y\,(\text{cm})$ 是温度 $x\,(°\text{C})$ 的一次函数．某种型号的实验用水银温度计能测量 $-20°\text{C}$ 至 $100°\text{C}$ 的温度，已知 10℃ 时水银柱高 10 cm，50℃ 时水银柱高 18 cm．求这个函数的表达式．

解：设 $y = kx + b\;(k \neq 0)$，根据题意，得

{

$10k + b = 10,\quad\cdots①$

$50k + b = 18.\quad\cdots②$

② $-$ ① 得：$40k = 8$，所以 $k = 0.2$；

代入①得：$10 \times 0.2 + b = 10$，所以 $b = 8$．

∴ 函数表达式为 $y = 0.2x + 8\;\;(-20 \leq x \leq 100)$．

📝 课堂小结

本节课我们学到了什么？

  1. 用待定系数法求一次函数表达式的四步流程：设 → 代 → 解 → 写
  2. 确定一次函数 $y = kx + b$ 需要两个条件（已知两点）；确定正比例函数 $y = kx$ 只需一个条件
  3. 实际问题中，先从情境提取对应值，再用待定系数法建立函数模型
  4. 求出函数表达式后，可以代入任意自变量值求函数值，体现函数表达式的工具作用

📖 读一读

本节课我们学习的 待定系数法，其实是一种应用非常广泛的数学方法，不仅在中学数学中常常用到，在科学研究和工程计算中也是重要工具．

比如，天文学家在发现新的小行星后，想要推算它的轨道，通常会先假设轨道的形状是一个椭圆（椭圆的方程也是一种含有待定系数的表达式），然后把在不同时间观测到的几个位置数据代入方程，解出那些未知系数——这样就能完整地确定这颗小行星的运行轨道了！

再比如，工程师设计桥梁时，也要先根据经验假设梁的受力变形规律（通常也是一种一次或二次函数关系），再把实验中测得的数据代入方程，解出系数后，才能得到精确的变形公式，用于后续的设计计算．

看来，先假设形式，再由数据确定系数——这种"先猜后验"的思路，是科学家和工程师们也在使用的方法！

🎓 数学核心素养

📐 数学抽象

从温度计、华氏温度等实际情境中抽象出一次函数模型，建立数学与现实的联系．

🔍 逻辑推理

由两个条件列出方程组，推导出唯一确定的函数表达式，体现逻辑的严密性．

✏️ 数学运算

准确解二元一次方程组，正确求出待定系数 $k$ 和 $b$，计算过程规范严谨．

🌐 数学建模

将现实问题转化为函数模型，再利用模型解决问题，是数学建模的完整闭环．

💬 教师寄语

待定系数法的核心思想是"先设后求"——先假设答案的形式，再用已知条件"填满"未知量．这种思路不只用于函数，在更高级的数学中（如求多项式、方程的根）都会反复出现．学会了待定系数法，就打开了一扇数学建模的大门．
