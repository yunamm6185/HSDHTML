例1 已知一点求正比例函数

已知正比例函数 $y = kx$ 的图象经过点 $P(-3, 6)$：

(1) 求该函数表达式    (2) 点 $Q(2, -4)$ 是否在该函数图象上？

(1)

  • 设：$y = kx$（$k \neq 0$）
  • 代：∵ 图象过 $P(-3, 6)$，∴ $6 = k \times (-3)$
  • 解：$k = -2$
  • 写：∴ 函数表达式为 $\boxed{y = -2x}$

(2) 将 $Q(2, -4)$ 的坐标代入 $y = -2x$：
右边 $= -2 \times 2 = -4$ = 左边
∴ 点 $Q(2, -4)$ 在该函数图象上 ✓

例2 已知两点求一次函数表达式

已知一次函数 $y = kx + b$ 的图象经过 $A(-1, 8)$ 和 $B(2, -1)$ 两点，求这个函数的表达式。

  • 设：$y = kx + b$（$k \neq 0$）
  • 代：
      将 $A(-1, 8)$ 代入 → $-k + b = 8$ ……①
      将 $B(2, -1)$ 代入 → $2k + b = -1$ ……②
  • 解方程组：
      ② − ① 得：$(2k+b)-(-k+b) = -1-8$
      → $3k = -9$ → $k = -3$
      将 $k=-3$ 代入① → $-(-3)+b=8$ → $3+b=8$ → $b = 5$
  • 写：$\boxed{y = -3x + 5}$

💡 验证：把两个已知点代回验证！
A: $-3\times(-1)+5 = 3+5 = 8$ ✓   B: $-3\times 2+5 = -6+5 = -1$ ✓

例3 实际问题建模 — 弹簧问题

一根弹簧不挂物体时长度为 12 cm。挂上物体后，每增加 1 kg 物体，弹簧伸长 0.5 cm（在弹性限度内）。设所挂物体重为 $x$ kg，弹簧长度为 $y$ cm。

(1) 求 $y$ 关于 $x$ 的函数关系式

(2) 求自变量 $x$ 的取值范围

(3) 当弹簧长度为 18 cm 时，所挂物体重多少 kg？

(1) 分析数量关系：

  • 原长（常数项）：$b = 12$ cm
  • 每增加 1 kg 伸长 0.5 cm → 斜率 $k = 0.5$
  • 总长 = 原长 + 伸长量 = $12 + 0.5x$

∴ 函数关系式：$\boxed{y = 0.5x + 12}$（$x \geq 0$）

(2) 自变量的取值范围：
  • 实际意义：$x \geq 0$（重量非负）
  • 弹性限度：假设最大承重为 $m$ kg，则 $0 \leq x \leq m$
∴ 取决于具体弹性限度

(3) 当 $y = 18$ 时：$0.5x + 12 = 18$ → $0.5x = 6$ → $\boldsymbol{x = 12}$ kg

即当弹簧长 18 cm 时，所挂物体重 12 kg。

例4 综合应用 — 从图象获取信息再求表达式

已知一次函数 $y = kx + b$ 的图象如图所示（经过 $(-2, 0)$ 和 $(0, 3)$）。

(1) 求此函数的表达式

(2) 当 $-4 \leq x \leq 2$ 时，求 $y$ 的取值范围

(3) 当 $y \leq 3$ 时，求 $x$ 的取值范围

(1) ∵ 过 $(0, 3)$ → $b = 3$

过 $(-2, 0)$ → $-2k + 3 = 0$ → $k = \frac32$

∴ 表达式为 $\boxed{y = \frac32 x + 3}$

(2) ∵ $k = \frac32 > 0$ → 增函数，$y$ 随 $x$ 增大而增大

当 $x = -4$ 时，$y = \frac32\times(-4)+3 = -6+3 = -3$

当 $x = 2$ 时，$y = \frac32\times2+3 = 3+3 = 6$

∴ 当 $-4 \leq x \leq 2$ 时，$\boxed{-3 \leq y \leq 6}$

(3) 先求 $y = 3$ 时 $x$ 的值：$\frac32 x + 3 = 3$ → $x = 0$

∵ 增函数且 $y \leq 3$ → 需要 $x \leq 0$

∴ $\boxed{x \leq 0}$

📝 课堂练习

1. 已知 $y-2$ 与 $x$ 成正比例，且当 $x = 3$ 时 $y = 8$。求 $y$ 与 $x$ 的函数关系式。

"$y-2$ 与 $x$ 成正比例" → 设 $y - 2 = kx$（$k \neq 0$）

当 $x=3$, $y=8$：$8 - 2 = 3k$ → $k = 2$

∴ $y - 2 = 2x$ → $\boxed{y = 2x + 2}$

⚠️ 注意审题！不是 "$y$ 与 $x$ 成正比例"，而是 "$y-2$ 与 $x$" 成正比例。这是常见的易错点！

2. 如图，直线 $l$ 是一次函数 $y=kx+b$ 的图象，填空：

  $k =$ ______，$b =$ ______；当 $x = 30$ 时，$y =$ ______。

需要从图中读取两个关键点的坐标（通常是与两坐标轴的交点），然后使用待定系数法求解。

🧠 课堂小结

  1. 待定系数法四步：设 → 代 → 解 → 写
  2. 正比例函数只需 1 个条件（1个未知量 $k$）；一般一次函数需 2 个条件（2个未知量 $k$ 和 $b$）
  3. 实际问题的建模：先分析常量和变化率，再写出表达式
  4. 求完表达式后要验证——把已知条件代入检查是否正确
