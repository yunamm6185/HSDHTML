📖 一、函数增减性的定义

📌 增函数与减函数的描述

定义：

  • 对于函数 $y =$ (关于 $x$ 的表达式)，如果在某个范围内，任意取 $x_1 < x_2$，设对应的函数值分别为 $y_1$ 和 $y_2$，都有 $\boldsymbol{y_1 < y_2}$，则称这个函数在这个范围上是增函数（单调递增）。
  • 如果在某个范围内，任意取 $x_1 < x_2$，都有 $\boldsymbol{y_1 > y_2}$，则称这个函数在这个范围上是减函数（单调递减）。

核心思想：取两个自变量比较大小 → 比较对应的函数值大小。如果自变量越大函数值也越大，就是增函数；自变量越大函数值反而越小，就是减函数！

🔬 二、探究活动 — 用代数法证明一次函数的增减性

问题：证明一次函数 $y = kx + b$ 的增减性与 $k$ 的关系

我们需要分别讨论 $k > 0$ 和 $k < 0$ 两种情况。

1当 $k > 0$ 时

任取 $x_1 < x_2$，设 $y_1 = kx_1 + b$，$y_2 = kx_2 + b$。

因为 $k > 0$, 且 $x_1 < x_2$

所以 $kx_1 < kx_2$（不等式两边乘正数，方向不变）

因此 $kx_1 + b < kx_2 + b$

即 $\boxed{y_1 < y_2}$

✅ 结论：当 $k > 0$ 时，一次函数 $y = kx+b$ 是增函数（在整个实数范围内都成立）

2当 $k < 0$ 时

任取 $x_1 < x_2$，设 $y_1 = kx_1 + b$，$y_2 = kx_2 + b$。

因为 $k < 0$, 且 $x_1 < x_2$ → $x_2 - x_1 > 0$

所以 $k(x_2 - x_1) < 0$（负数 × 正数 ＜ 0）

即 $kx_2 - kx_1 < 0$

所以 $kx_2 + b < kx_1 + b$，也就是 $y_2 < y_1$

即 $\boxed{y_1 > y_2}$

✅ 结论：当 $k < 0$ 时，一次函数 $y = kx+b$ 是减函数（在整个实数范围内都成立）

📌 定理总结

对于一次函数 $y = kx + b$（$k \neq 0$）：

$k$ 的符号单调性直观理解
$k > 0$增函数（在所有实数上）“左低右高”
$k < 0$减函数（在所有实数上）“左高右低”
🔬 三、反比例函数的单调性证明

例1 证明反比例函数 $y = \frac{k}{x}$（$k > 0$）在 $(0, +\infty)$ 上的单调性

求证：当 $k > 0$ 时，$y = \frac{k}{x}$ 在 $(0, +\infty)$ 上是减函数。

证明：

任取 $x_1, x_2 \in (0, +\infty)$，且设 $0 < x_1 < x_2$。对应的函数值 $y_1 = \frac{k}{x_1}$，$y_2 = \frac{k}{x_2}$。

比较 $y_1$ 与 $y_2$ 的大小：

$$y_1 - y_2 = \frac{k}{x_1} - \frac{k}{x_2} = k\left(\frac{1}{x_1} - \frac{1}{x_2}\right) = k\cdot\frac{x_2-x_1}{x_1 x_2}$$

分析各项符号：
• $k > 0$（已知条件）
• $x_2 - x_1 > 0$（因为 $x_1 < x_2$）
• $x_1 x_2 > 0$（因为 $x_1 > 0, x_2 > 0$）

所以 $y_1 - y_2 > 0$ → $y_1 > y_2$

即任取 $x_1 < x_2$ 都有 $y_1 > y_2$，符合减函数定义。

∴ 当 $k > 0$ 时，$\displaystyle y = \frac{k}{x}$ 在 $(0,+\infty)$ 上是减函数 ✅

⚠️ 注意：反比例函数并不是在整个定义域（$x \neq 0$）上具有统一的单调性。它在每个单独的区间 $(-\infty,0)$ 和 $(0,+\infty)$ 内分别是单调的，但不能说在整个定义域上单调。
✅ 四、利用增减性解题

例2 利用增减性比较函数值的大小

(1) 已知 $y = 3x - 5$（$k=3>0$，因此是增函数）。若 $a+1 > a-2$，那么当 $x = a+1$ 时的函数值与当 $x = a-2$ 时的函数值哪个大？

(2) 已知点 $A(-1, y_1)$、$B(3, y_2)$ 在 $y = -2x + 7$ 的图象上，判断 $y_1$ 与 $y_2$ 的大小。

(1) 因为 $y = 3x-5$ 中 $k=3 > 0$，函数随 $x$ 增大而增大。

由 $a+1 > a-2$ 可知自变量 $x=a+1$ 大于 $x=a-2$，因此对应的函数值也更大。

所以 $\boxed{\text{当 }x=a+1\text{ 时的函数值} > \text{当 }x=a-2\text{ 时的函数值}}$。

(2) 因为 $y = -2x+7$ 中 $k=-2 < 0$，函数随 $x$ 增大而减小。

自变量 $-1 < 3$，所以 $x=-1$ 对应的函数值 $y_1$ 大于 $x=3$ 对应的函数值 $y_2$，即 $\boldsymbol{y_1 > y_2}$。

验证：$y_1 = -2\times(-1)+7 = 9$；$y_2 = -2\times3+7 = 1$ → 确实 $9 > 1$ ✓

例3 综合应用

已知一次函数 $y = (2m-3)x + m+4$，若 $y$ 随 $x$ 的增大而减小，且 $y$ 轴截距大于 -2，求整数 $m$ 的值。

条件分析：

  1. y 随 x 增大而减小 → 减函数 → $k < 0$
    → $2m-3 < 0$ → $m < \frac32$
  2. y 轴截距 > -2
    截距 = $m+4$（令 $x=0$）
    → $m+4 > -2$ → $m > -6$
  3. $k \neq 0$ → $2m-3 \neq 0$ → $m \neq \frac32$（已包含在条件1中）

综合得：$-6 < m < \frac32$

满足条件的整数 $m$ 有：-5, -4, -3, -2, -1, 0, 1

📝 课堂练习

1. 用定义证明：函数 $y = -4x + 10$ 在全体实数上是减函数。

任取 $x_1 < x_2$，设 $y_1 = -4x_1+10$，$y_2 = -4x_2+10$。

计算 $y_1 - y_2 = (-4x_1+10) - (-4x_2+10) = -4(x_1 - x_2) = 4(x_2 - x_1)$。

因为 $x_2 > x_1$，所以 $x_2 - x_1 > 0$，从而 $y_1 - y_2 > 0$，即 $y_1 > y_2$。

根据定义，当 $x_1 < x_2$ 时有 $y_1 > y_2$，所以函数 $y = -4x+10$ 是减函数。✓

2.已知反比例函数 $y=\frac{k}{x}$ 经过点 $P(2, -3)$：

  (1) 求 $k$ 并写出函数表达式   (2) 该函数在哪个区间上是增函数？

(1) 代入 $(2,-3)$：$-3 = \frac{k}{2}$ → $\boldsymbol{k = -6}$

表达式：$\displaystyle y = \frac{-6}{x}$

(2) 因为 $k = -6 < 0$

根据反比例函数性质：当 $k<0$ 时，在 $(-\infty, 0)$ 和 $(0, +\infty)$ 每个区间上都是增函数（注意不能说在整个定义域上单调递增，必须分开区间叙述）。

📖 数学阅读 — 单调性的价值

函数的单调性（即增减性）是函数最重要的基本性质之一，它帮助我们不画图也能分析函数的变化趋势。

  • 利用单调性可以方便地比较函数值的大小，解决许多实际问题中的最优化判断。
  • 单调性也是求解方程近似解和不等式的重要工具，例如利用单调性可以确定解的个数或范围。
  • 在未来的数学学习中（如高中），我们会用更丰富的方法研究更复杂函数的单调性，但“作差法”这种从定义出发的证明将是永恒的基石。
  • 正因为单调函数保持“一一对应”的关系，它在实际问题中常用于建立反函数模型。

今天我们用作差法从代数上严格证明了基本函数的单调性，这为今后学习函数分析奠定了严谨的基础！
