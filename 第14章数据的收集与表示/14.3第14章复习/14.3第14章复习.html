📡 §14.1 数据的收集

🔍 普查 vs 抽样调查

方式定义
普查对所有考察对象作的全面调查
抽样调查对部分考察对象作的局部调查
适用抽样：数量巨大 / 有破坏性 / 经费时间有限

📐 统计学基本概念

  • 总体：所要考察对象的全体
  • 个体：组成总体的每一个考察对象
  • 样本：从总体中取出的一部分个体
  • 样本容量：样本中个体的数量（无单位）

🎲 简单随机抽样

用抽签的办法决定哪些个体进入样本，使每个个体被抽到的机会相等。

💬 "简单随机抽样是一种很重要的抽样方法"

📁 数据来源分类

  • 一手数据：通过亲自调查、测量或实验直接获得的数据
  • 二手数据：查阅文献资料、互联网等间接获得的数据

⚠️ 使用二手数据时要注意数据的来源是否可靠

📊 频数与频率

⭐ 核心公式
$$\text{频率} = \frac{\text{频数}}{\text{总次数}} = \frac{f_i}{n}, \quad \text{所有频率之和}=1$$

🔢 频数 (Frequency)

每个对象出现的次数

  • 频数为非负整数
  • 所有频数之和等于总次数 $n$
  • 反映对象出现的绝对频繁程度

📉 频率 (Relative Frequency)

频数与总次数的比值（百分比）

  • $0 \leq \text{频率} \leq 1$
  • 所有频率之和恒为 1
  • 反映相对频繁程度，便于比较不同总量的数据

💡 关键提醒

当总次数不相等时，应比较频率而不是频数！

❌ 错误做法：甲班喜欢篮球的有20人，乙班有15人 → 甲班更喜欢
✅ 正确做法：比较两班的频率（百分比）

📈 §14.2 数据的表示

📊 条形统计图

  • 用长方形的高度表示数据大小
  • 各长方形的宽度相同
  • 能清楚地显示具体数值的比较

📈 折线统计图

  • 用折线连接各数据点
  • 适合展示数据随时间变化的趋势
  • 能直观反映增减变化情况

🥧 扇形统计图

  • 整个圆代表总体（100%）
  • 扇形面积 = 该部分所占百分比
  • 圆心角公式：
    圆心角 = 360° × 百分比

📉 频数分布直方图

绘制步骤：

  1. 计算极差（最大值 − 最小值）
  2. 决定组数和组距
  3. 确定分点，列出频数分布表
  4. 画频数分布直方图

✨ 特点：长条之间没有空隙，反映数据整体分布形态

⚠️ 三、易错点与常见误区
🚨 易错点 1：混淆普查与抽样调查的适用场景
并非所有问题都适合普查。当考察对象数不胜数、有破坏性损伤、经费时间有限时，应选用抽样调查。
🚨 易错点 2：样本容量忘记不带单位
样本容量是样本中个体的数量，是一个数字，不附带任何单位。
🚨 易错点 3：总次数不等时误比频数
当两组数据的总次数（样本容量）不同时，必须比较频率而非频数。
🚨 易错点 4：直方图的组距选择不当
组距过大则丢失信息，组距过小则过于琐碎。一般取 5~12 个组为宜。
🚨 易错点 5：扇形图圆心角计算错误
圆心角 = 360° × 该部分的百分比。注意百分比要用小数形式参与计算，例如32%要写成0.32。
🚨 易错点 6：被误导性统计图欺骗
纵轴起点不是0、纵轴单位长度不一致、使用三维效果等手法都可能夸大差异。阅读统计图时要保持批判性眼光！
📝 四、综合例题精讲

📌 例题 1：选择合适的调查方式

下列哪些考察适合用普查？哪些适合用抽样调查？请说明理由。

  1. 考察一片试验田里某种大麦的穗长情况；
  2. 考察一个班级中的学生对建立班级生物角的看法；
  3. 考察人们保护海洋的意识；
  4. 考察某批次灯泡的使用寿命；
  5. 审查某公司本年度的全部财务账目。
【解析】

(1) 抽样调查——大麦数量多，难以逐一测量，且抽样即可了解整体情况。

(2) 普查——班级人数有限（通常40~50人），可以对每个学生进行调查。

(3) 抽样调查——人群数量庞大，无法逐一调查。

(4) 抽样调查——测试灯泡寿命具有破坏性（测试后灯泡报废）。

(5) 普查——财务审计要求全面准确，且账目总量有限。

📌 例题 2：扇形统计图的综合计算

某市民服务热线某日收到的诉求分类占比如下表所示：

诉求类型求助咨询投诉建议其他
占比25%32.5%20%12.5%10%

已知求助类诉求为200件，求：

  1. 当日收到的总诉求数量是多少？
  2. 咨询类诉求的数量？
  3. 投诉类诉求所对应的扇形圆心角的度数？
【解】

(1) 总诉求数量 = $200 \div 25\% = 800$（件）

(2) 咨询类诉求数量 = $800 \times 32.5\% = 260$（件）

(3) 投诉类圆心角 = $360^{\circ} \times 20\% = 72^{\circ}$

答：总诉求800件，咨询类260件，投诉类圆心角为72°。

📌 例题 3：频数分布表的制作与分析

为了解八年级学生的课外阅读情况，随机抽取了50名学生，记录他们每周阅读时长（单位：小时）如下：

3, 5, 2, 7, 4, 6, 8, 3, 5, 4, 6, 9, 2, 4, 5, 7, 3, 6, 4, 8,
5, 3, 7, 4, 6, 2, 5, 8, 3, 4, 6, 7, 5, 3, 9, 4, 6, 2, 5, 8,
3, 7, 4, 5, 6, 3, 8, 4, 7, 5

完成以下任务：

  1. 计算极差，若分5组，确定每组的组距和分点；
  2. 列出频数分布表；
  3. 哪一组的人数最多？该组的频率是多少？
【解】

(1) 极差与分组：最大值=9，最小值=2，极差=$9-2=7$
分5组：组距 $\approx 7 \div 5 = 1.4$，取组距 = 2
分点：$2 \le t < 4$, $4 \le t < 6$, $6 \le t < 8$, $8 \le t \le 9$
（共4组已足够覆盖，或可调整为5组：$2\sim3$, $4\sim5$, $6\sim7$, $8\sim9$）

(2) 频数分布表：
组别$2\le t<4$$4\le t<6$$6\le t<8$$8\le t\le 9$
频数1316129
频率0.260.320.240.18

(3) 第2组（$4 \le t < 6$）人数最多，频数为16，
频率 = $16 \div 50 = 0.32$ = 32%
✏️ 五、巩固练习

📝 练习 1

下列调查中，最适合采用全面调查（普查）的是（   ）

A. 调查全国中学生的视力情况    B. 调查某品牌电视机的使用寿命

C. 调查"神舟飞船"零部件的质量    D. 调查你所在班级同学的身高

答案：D
解析：A和B适合抽样调查；C需要普查（质量要求高）；D班级人数有限，适合普查。但C和D相比，D更典型地体现"范围小→普查"，故选D。（注：C也常作为普查选项）

📝 练习 2

为了解某校2000名学生对学校食堂饭菜的意见，从中随机抽取200名进行调查。在这个问题中：

①总体是2000名学生；②个体是每名学生对食堂饭菜的意见；③样本是抽取的200名学生；④样本容量是200。

以上说法正确的个数是（   ）

A. 1个   B. 2个   C. 3个   D. 4个

答案：B（②④正确）
解析：①错——总体是2000名学生对食堂饭菜的意见（全体考察对象的属性）；②正确；③错——样本是抽取的200名学生的意见；④正确。

📝 练习 3

在一次"我最喜爱的体育项目"调查中，共有120人参加，其中喜爱篮球的频数为36人。

(1) 喜爱篮球的频率是多少？

(2) 如果在扇形统计图中表示篮球爱好者，对应的圆心角是多少度？

【解】
(1) 频率 = $36 \div 120 = 0.3 =$ 30%

(2) 圆心角 = $360^{\circ} \times 30\% = 360^{\circ} \times 0.3 =$ 108°

📝 练习 4

右图（假设）是一个频数分布直方图，其中第二组的频率为0.25，第四组的频数为8。如果总共有40个数据，那么第三组的频数是多少？

答案：需要更多信息
解析：已知总数据n=40，第四组频数=8（频率=8/40=0.2），第二组频率=0.25（频数=10）。剩余第一组和第三组频数之和=40−10−8=22。如果题目给出四组频率之和为1，还需知道第一组频率才能确定第三组频数。
（此题为训练思路，实际考试通常会给出更多条件）
