数学活动：无限循环小数能化为分数吗

🎯 活动目标

✔了解无限循环小数都可以化为分数

✔能区分纯循环小数与混循环小数的转化规律
✔体会数学中的化归思想和方程思想

📌 我们知道：分数都可以化成小数，要么化成有限小数，要么化成无限循环小数。

📌 反过来，有限小数都可以写成分数。那么，无限循环小数是否也能化为分数呢？

🔍 探索活动一：纯循环小数

\(0.47 = 0.474747\ldots\) 是一个以47为循环节的无限循环小数

问题：怎样将这个无限循环小数化为分数？

📐 第1步：扩大倍数

将 \(0.47\) 扩大到100倍，把第一个循环节移到小数点之前：

\(100 \times 0.47 = 47.4747\ldots\)

📐 第2步：观察发现

发现 \(47.4747\ldots\) 的小数部分依然是循环节为47的无限循环小数，即仍是原数！

\(100 \times 0.47 = 47 + 0.47\)

📐 第3步：列方程求解

设 \(x = 0.47\)，则：

\(100x = 47 + x\)

\(100x - x = 47\)

\(99x = 47\)

\(x = \dfrac{47}{99}\)

🎉 所以：\(0.47 = \dfrac{47}{99}\)

✏️ 试一试：这种方法是否适用于其他纯循环小数？

\(0.74\)

(循环节 74)

\(0.8\)

(循环节 8)

\(0.325\)

(循环节 325)

\(0.74\)：设 \(x=0.74\)，则 \(100x=74+x\)，\(99x=74\)，\(x=\dfrac{74}{99}\)
\(0.8\)：设 \(x=0.8\)，则 \(10x=8+x\)，\(9x=8\)，\(x=\dfrac{8}{9}\)
\(0.325\)：设 \(x=0.325\)，则 \(1000x=325+x\)，\(999x=325\)，\(x=\dfrac{325}{999}\)
📌 纯循环小数化分数规律

一个纯循环小数化成分数，分子是一个循环节的数字组成的数，分母是由若干个9组成的数，9的个数等于循环节的位数。[citation:1]

例如：\(0.\dot{4}\dot{7} = \dfrac{47}{99}\)，\(0.\dot{8} = \dfrac{8}{9}\)，\(0.\dot{3} = \dfrac{3}{9} = \dfrac{1}{3}\)

🔍 探索活动二：混循环小数

问题：对于不是从小数点后第一位开始循环的小数（混循环小数），例如 \(0.32\dot{5}\)，能化为分数吗？

📐 方法：先转化为纯循环小数

以 \(0.32\dot{5} = 0.325555\ldots\) 为例：

设 \(x = 0.325555\ldots\)

先乘以 \(10^2\) 把不循环部分移到整数部分：\(100x = 32.5555\ldots\)

而 \(0.5555\ldots = \dfrac{5}{9}\)

所以 \(100x = 32 + \dfrac{5}{9} = \dfrac{288}{9} + \dfrac{5}{9} = \dfrac{293}{9}\)

因此 \(x = \dfrac{293}{900}\)

📐 另一种解法（方程法）

设 \(x = 0.32\dot{5}\)

先乘以100：\(100x = 32.\dot{5}\)

再乘以10：\(1000x = 325.\dot{5}\)

两式相减：\(1000x - 100x = 325.\dot{5} - 32.\dot{5} = 293\)

所以 \(900x = 293\)，\(x = \dfrac{293}{900}\)

📌 混循环小数化分数规律

一个混循环小数化成分数，分子是第二个循环节以前的小数部分组成的数与小数部分中不循环部分组成的数的差；分母的头几位是9，末几位是0，9的个数与循环节位数相同，0的个数与不循环部分位数相同。[citation:1]

例如：\(0.32\dot{5} = \dfrac{325-32}{900} = \dfrac{293}{900}\)

✏️ 巩固练习

1. 将下列纯循环小数化为分数：
(1) \(0.\dot{6}\) (2) \(0.\dot{1}\dot{2}\) (3) \(0.\dot{2}\dot{3}\dot{4}\)
(1) \(0.\dot{6} = \dfrac{6}{9} = \dfrac{2}{3}\)
(2) \(0.\dot{1}\dot{2} = \dfrac{12}{99} = \dfrac{4}{33}\)
(3) \(0.\dot{2}\dot{3}\dot{4} = \dfrac{234}{999} = \dfrac{78}{333} = \dfrac{26}{111}\)
2. 将下列混循环小数化为分数：
(1) \(0.1\dot{6}\) (2) \(0.23\dot{4}\) (3) \(0.12\dot{3}\dot{4}\)
(1) \(0.1\dot{6} = \dfrac{16-1}{90} = \dfrac{15}{90} = \dfrac{1}{6}\)
(2) \(0.23\dot{4} = \dfrac{234-23}{900} = \dfrac{211}{900}\)
(3) \(0.12\dot{3}\dot{4} = \dfrac{1234-12}{9900} = \dfrac{1222}{9900} = \dfrac{611}{4950}\)
3. 将 \(2.1\dot{8}\) 化为分数。
\(2.1\dot{8} = 2 + 0.1\dot{8}\)
\(0.1\dot{8} = \dfrac{18-1}{90} = \dfrac{17}{90}\)
所以 \(2.1\dot{8} = 2 + \dfrac{17}{90} = \dfrac{180}{90} + \dfrac{17}{90} = \dfrac{197}{90}\)

🤔 思考：\(0.\dot{9} = 1\) 吗？

用我们今天的方法：

设 \(x = 0.999\ldots\)，则 \(10x = 9.999\ldots\)

两式相减：\(10x - x = 9.999\ldots - 0.999\ldots\)

\(9x = 9\)，所以 \(x = 1\)

结论：\(0.\dot{9} = 1\)，这不是近似，而是严格相等！[citation:5]

📜 数学文化 · 庄子一尺之棰

《庄子·天下篇》：“一尺之棰，日取其半，万世不竭。”

用数学式子表示：\(\dfrac{1}{2} + \dfrac{1}{4} + \dfrac{1}{8} + \dfrac{1}{16} + \cdots = 1\)

你能用今天学的方法证明吗？

设 \(S = \dfrac{1}{2} + \dfrac{1}{4} + \dfrac{1}{8} + \dfrac{1}{16} + \cdots\)
则 \(2S = 1 + \dfrac{1}{2} + \dfrac{1}{4} + \dfrac{1}{8} + \cdots = 1 + S\)
所以 \(2S - S = 1\)，即 \(S = 1\)
