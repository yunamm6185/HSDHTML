1.7 有理数的减法

🎯 本课时学习目标

✔理解有理数减法法则
✔会将减法转化为加法进行计算
✔能熟练进行有理数减法运算
✔体会化归思想在数学中的应用

🏔️ 珠穆朗玛峰海拔8848.86m，吐鲁番盆地海拔-154.31m。珠峰比吐鲁番盆地高多少米？

列式：\(8848.86 - (-154.31)\)。怎样计算呢？

🔍 探究：有理数的减法法则

计算：\((-8) - (-3)\)。根据减法的意义，就是求一个数“？”，使 \((?) + (-3) = -8\)。

由加法运算，\((-5) + (-3) = -8\)，所以 \((-8) - (-3) = -5\)。

同时，\((-8) + (+3) = -5\)。因此 \((-8) - (-3) = (-8) + (+3)\)。

一般地，对于任意有理数 \(a, b\)，由减法的意义，\(a - b\) 就是要求一个数“？”，使

\((?) + b = a\)。

因为

\([a + (-b)] + b = a + [(-b) + b]\) （加法结合律）
\(= a + 0\) （加法法则3：互为相反数的两个数相加得0）
\(= a\) （加法法则4：一个数与0相加，仍得这个数），

所以 \(a - b = a + (-b)\)。

有理数的减法法则： \(a - b = a + (-b)\)

减去一个数，等于加上这个数的相反数。

✏️ 例题

例 计算：

(1) \((-32) - (+5)\)； (2) \(7.3 - (-6.8)\)； (3) \((-2) - (-25)\)； (4) \(12 - 21\)。

(1) \((-32) - (+5) = (-32) + (-5) = -37\)
(2) \(7.3 - (-6.8) = 7.3 + 6.8 = 14.1\)
(3) \((-2) - (-25) = (-2) + 25 = 23\)
(4) \(12 - 21 = 12 + (-21) = -9\)

练习

1. 在下列括号内填上适当的数：
(1) \((-2) - (-3) = (-2) + ( \quad )\)；
(2) \(0 - (-4) = 0 + ( \quad )\)；
(3) \((-6) - 3 = (-6) + ( \quad )\)；
(4) \(1 - (+39) = 1 + ( \quad )\)。
(1) \(+3\)； (2) \(+4\)； (3) \(-3\)； (4) \(-39\)。
2. 计算：
(1) \((+3) - (-2)\)； (2) \((-1) - (+2)\)； (3) \(0 - (-3)\)；
(4) \(1 - 5\)； (5) \((-23.6) - (-12.4)\)； (6) \(\dfrac{2}{3} - \left(-\dfrac{1}{2}\right)\)。
(1) \((+3) - (-2) = 3 + 2 = 5\)
(2) \((-1) - (+2) = -1 + (-2) = -3\)
(3) \(0 - (-3) = 0 + 3 = 3\)
(4) \(1 - 5 = 1 + (-5) = -4\)
(5) \((-23.6) - (-12.4) = -23.6 + 12.4 = -11.2\)
(6) \(\frac{2}{3} - \left(-\frac{1}{2}\right) = \frac{2}{3} + \frac{1}{2} = \frac{4}{6} + \frac{3}{6} = \frac{7}{6}\)
3. 填空：
(1) 温度 \(3^{\circ}C\) 比 \(-8^{\circ}C\) 高 \(\_\_\_\) \(^{\circ}C\)；
(2) 温度 \(-9^{\circ}C\) 比 \(-1^{\circ}C\) 低 \(\_\_\_\) \(^{\circ}C\)；
(3) 海拔 \(-20\mathrm{m}\) 比 \(-30\mathrm{m}\) 高 \(\_\_\_\) m；
(4) 从海拔 \(22\mathrm{m}\) 到 \(-10\mathrm{m}\)，下降了 \(\_\_\_\) m。
(1) \(3 - (-8) = 3 + 8 = 11\)
(2) 低 \(-9 - (-1) = -9 + 1 = -8\)，即低8℃
(3) \((-20) - (-30) = -20 + 30 = 10\)
(4) 下降 \(22 - (-10) = 22 + 10 = 32\)

📖 读一读 · 化归思想——将新知识变成老朋友

▶

在数学中，我们经常把新遇到的、不会的问题，想办法变成我们已经会解决的问题。这种思想叫做“化归”，简单说就是“将未知转化为已知”。

小学时，我们就接触过化归：例如，计算 \(3 + (?) = 5\)，我们想 (?) 是多少，就用减法 \(5 - 3 = 2\)。反过来，减法 \(5 - 3\) 也可以看作是求“5 加上 3 的相反数”。今天学习的减法法则 \(a - b = a + (-b)\)，正是把减法转化成了加法——而加法我们已经很熟悉了！

在本章中，其实已经多次用到化归思想：

  • 比较负数的大小：我们不会直接比较两个负数，但会先求它们的绝对值（变成正数），再比较绝对值的大小，最后得出结论“绝对值大的反而小”。
  • 简化符号：像 \(-(-3)\) 这样的式子，我们利用相反数的概念，把它转化为 \(+3\)，这也是化归。
  • 绝对值运算：求一个负数的绝对值，我们转化为求它的相反数（正数）。

以后我们还会遇到很多这样的例子：比如把除法转化为乘法，把乘方转化为乘法，把复杂的算式逐步化简……化归思想是学习数学最重要的法宝之一。掌握了它，你就能不断用旧知识解决新问题，越学越轻松！
