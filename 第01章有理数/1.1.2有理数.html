我们以前学过正整数、0、分数（小数），比如：\(1, 2, 3, 0, \frac{1}{2}, 0.75\)。现在又认识了负数，数的家族更庞大了。它们之间有什么关系呢？

📚 有理数的分类

🔹 按定义分类

▼
  • 整数
    • 正整数：1, 2, 3, …
    • 0
    • 负整数：-1, -2, -3, …
  • 分数
    • 正分数：\(\frac{1}{2}, \frac{22}{7}, 4.5\)
    • 负分数：\(-\frac{1}{3}, -2.5, -\frac{5}{6}\)

整数和分数统称为有理数。

🔸 按性质分类

▼
  • 正有理数
    • 正整数：1, 2, 3, …
    • 正分数：\(\frac{1}{2}, \frac{3}{4}, \frac{22}{7}, …\)
  • 0（既不是正数也不是负数）
  • 负有理数
    • 负整数：-1, -2, -3, …
    • 负分数：\(-\frac{1}{2}, -\frac{2}{3}, -0.6, …\)

📖 读一读 · “有理数”名称的由来

▶

“有理数”的英文名 rational number 中的 rational 应看成 ratio（比、比率）的形容词形式，因此 rational number 应该理解为“比率数”，即可以表示为两个整数之商（比率）的数。在学习了有理数的除法之后，这样的解释准确地描述了有理数的本质。

练习

1. 请说出两个正整数、两个负整数、两个正分数、两个负分数。它们都是有理数吗？
正整数：3, 10；负整数：-5, -21；正分数：\(\frac{1}{2}, \frac{3}{4}\)；负分数：\(-\frac{2}{3}, -\frac{5}{7}\)。它们都是有理数。
2. 有理数集中有没有这样的数，它既不是正数，也不是负数？若有，请说出这样的数。
有，0 既不是正数也不是负数。
3. 把下列各数填入相应的集合：\(-7, 0.5, -\frac{2}{3}, 0, 2024, -3.14, \frac{22}{7}\)
正数集：\(\{0.5, 2024, \frac{22}{7}, \ldots\}\)；负数集：\(\{-7, -\frac{2}{3}, -3.14, \ldots\}\)；整数集：\(\{-7, 0, 2024, \ldots\}\)；分数集：\(\{0.5, -\frac{2}{3}, -3.14, \frac{22}{7}, \ldots\}\)。

📌 课堂小结

▶
整数（正整数、0、负整数）
分数（正分数、负分数）
有理数 = 整数 + 分数
