1.6 有理数的加法

第二课时 有理数加法的运算律

🎯 本课时学习目标

✔理解加法交换律和结合律
✔能运用运算律简化有理数加法
✔会用简便方法解决多个数相加问题
✔能应用加法运算律解决实际问题

📌 我们学过有理数的加法法则，但对于多个有理数相加，怎样算更简便呢？例如：\((-7)+(+10)+(-11)+(-2)\)，直接从左到右算可以，但有没有更快的办法？

🔍 探究：加法运算律

任意选择两个有理数（至少有一个负数）填入□和○，比较：

\(\square + \bigcirc\) 与 \(\bigcirc + \square\)

任意选择三个有理数填入□、○、◇，比较：

\((\square + \bigcirc) + \Diamond\) 与 \(\square + (\bigcirc + \Diamond)\)

通过计算可以发现，有理数的加法仍满足交换律和结合律：

加法交换律： \(a + b = b + a\)

加法结合律： \((a + b) + c = a + (b + c)\)

这样，多个有理数相加，可以任意交换加数的位置，也可以先把其中的几个数相加，使计算简化。

✏️ 例题

例2 计算：

(1) \((+26)+(-18)+5+(-16)\)；

(2) \((-1.75)+1.5+(+7.3)+(-2.25)+(-8.5)\)。

(1) \((+26)+(-18)+5+(-16)\)
\(= 26 + 5 + [(-18)+(-16)]\) （交换、结合）
\(= 31 + (-34)\)
\(= -3\)
(2) \((-1.75)+1.5+(+7.3)+(-2.25)+(-8.5)\)
\(= [(-1.75)+(-2.25)] + [1.5+(-8.5)] + 7.3\) （交换、结合）
\(= (-4) + (-7) + 7.3\)
\(= (-11) + 7.3\)
\(= -3.7\)

例3 10筐苹果，以每筐30kg为基准，超过的千克数记作正数，不足的千克数记作负数，记录如下：2，-4，2.5，3，-0.5，1.5，3，-1，0，-2.5。问：这10筐苹果总共重多少？

\(2 + (-4) + 2.5 + 3 + (-0.5) + 1.5 + 3 + (-1) + 0 + (-2.5)\)
\(= (2+3+3) + (-4) + [2.5+(-2.5)] + [(-0.5)+1.5+(-1)] + 0\) （交换、结合）
\(= 8 + (-4) + 0 + 0\)
\(= 4\)
总重量 \(= 30 \times 10 + 4 = 304\) (kg)
答：这10筐苹果总共重304kg。

练习

1. 计算：
(1) \((-7)+(+10)+(-11)+(-2)\)；
(2) \(2+(-3)+(+4)+(-5)+6\)；
(3) \((-9.6)+1.5+(-0.4)+(-0.3)+8.5\)。
(1) \((-7)+(+10)+(-11)+(-2)\)
\(= [(-7)+(-11)+(-2)] + 10\) （交换、结合）
\(= (-20) + 10\)
\(= -10\)

(2) \(2+(-3)+(+4)+(-5)+6\)
\(= (2+4+6) + [(-3)+(-5)]\) （交换、结合）
\(= 12 + (-8)\)
\(= 4\)

(3) \((-9.6)+1.5+(-0.4)+(-0.3)+8.5\)
\(= [(-9.6)+(-0.4)] + [1.5+8.5] + (-0.3)\) （交换、结合）
\(= (-10) + 10 + (-0.3)\)
\(= -0.3\)
2. 某天早晨的气温是 \(-3^{\circ}C\)，到中午升高了 \(5^{\circ}C\)，到晚上又降低了 \(3^{\circ}C\)，到午夜又降低了 \(4^{\circ}C\)。求午夜时的气温。（提示：降低了 \(3^{\circ}C\) 就是升高了 \(-3^{\circ}C\)）
列式：\((-3) + 5 + (-3) + (-4)\)
\(= [(-3)+(-3)+(-4)] + 5\) （交换、结合）
\(= (-10) + 5\)
\(= -5\)
答：午夜时的气温是 \(-5^{\circ}C\)。

📖 读一读 · 什么时候用运算律更简便？

▶

在有理数加法中，灵活运用运算律可以大大简化计算。以下几种情况特别适合用交换律和结合律：

  • ① 相反数结合法：先把互为相反数的数相加（和为0）。
    例：\((-2)+3+1+(-3)+2 = [(-2)+2] + [3+(-3)] + 1 = 0+0+1 = 1\)
  • ② 同号结合法：把正数与正数、负数与负数分别结合相加。
    例：\(16+(-25)+24+(-35) = (16+24) + [(-25)+(-35)] = 40 + (-60) = -20\)
  • ③ 凑整法：把相加能得到整数、整十、整百的数先结合。
    例：\((-1.75)+1.5+7.3+(-2.25)+(-8.5) = [(-1.75)+(-2.25)] + [1.5+(-8.5)] + 7.3\)
  • ④ 同分母结合法：先把分母相同的分数结合相加。
    例：\(\frac{2}{5}+(-\frac{3}{7})+\frac{3}{5}+(-\frac{4}{7}) = (\frac{2}{5}+\frac{3}{5}) + [(-\frac{3}{7})+(-\frac{4}{7})] = 1 + (-1) = 0\)
  • ⑤ 易于通分的数：对于异分母分数，先把易于通分的数结合。
    例：\(\frac{1}{2}+(-\frac{1}{3})+\frac{1}{6} = \frac{1}{2}+[\frac{1}{6}+(-\frac{1}{3})] = \frac{1}{2}+(-\frac{1}{6}) = \frac{1}{3}\)

总之，运算前先观察数的特征，选择合适的方法组合，可以让计算又快又准！

📌 课堂小结

▶
加法交换律：\(a+b=b+a\)
加法结合律：\((a+b)+c=a+(b+c)\)
多个数相加，可任意交换、结合，简化计算
实际问题中，常用基准量与偏差相加
