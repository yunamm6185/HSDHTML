第11章 整式的乘除

11.5.1 因式分解

🎯 学习目标

✔理解因式分解的意义
✔掌握提公因式法分解因式
✔能运用平方差公式、完全平方公式分解因式

问题引入

计算下列整式乘法：

$(1)\ m(a+b+c) = \underline{\qquad\qquad}$

$(2)\ (a+b)(a-b) = \underline{\qquad\qquad}$

$(3)\ (a+b)^2 = \underline{\qquad\qquad}$

$(1)\ ma+mb+mc$

$(2)\ a^2-b^2$

$(3)\ a^2+2ab+b^2$

反过来，把一个多项式化成几个整式的积的形式，叫做因式分解。

试一试

观察下面的等式，左边是多项式，右边是整式的乘积：

$(1)\ ma+mb+mc = m(a+b+c)$

$(2)\ a^2-b^2 = (a+b)(a-b)$

$(3)\ a^2+2ab+b^2 = (a+b)^2$

这就是因式分解。

概括
把一个多项式化为几个整式的积的形式，叫做多项式的因式分解（factorization）。
多项式 $ma+mb+mc$ 中的每一项都含有一个相同的因式 $m$，我们称之为公因式（common factor）。
提公因式法：$ma+mb+mc = m(a+b+c)$
公式法：$a^2-b^2 = (a+b)(a-b)$，$a^2+2ab+b^2 = (a+b)^2$，$a^2-2ab+b^2 = (a-b)^2$。

例1 把下列多项式分解因式

$(1)\ -5a^2+25a$； $(2)\ 3a^2-9ab$； $(3)\ 25x^2-16y^2$； $(4)\ x^2+4xy+4y^2$。

$(1)\ -5a^2+25a = -5a(a-5)$；

$(2)\ 3a^2-9ab = 3a(a-3b)$；

$(3)\ 25x^2-16y^2 = (5x)^2-(4y)^2 = (5x+4y)(5x-4y)$；

$(4)\ x^2+4xy+4y^2 = x^2+2\cdot x\cdot2y+(2y)^2 = (x+2y)^2$。

例2 把下列多项式分解因式

$(1)\ 4x^3y-4x^2y^2+xy^3$； $(2)\ 3x^3-12xy^2$。

$(1)\ 4x^3y-4x^2y^2+xy^3 = xy(4x^2-4xy+y^2) = xy(2x-y)^2$；

$(2)\ 3x^3-12xy^2 = 3x(x^2-4y^2) = 3x(x+2y)(x-2y)$。

✏️ 练习

1. 判断下列因式分解是否正确，如果不正确，请写出正确答案：

$(1)\ 4a^2-4a+1 = 4a(a-1)+1$； $(2)\ x^2-4y^2 = (x+4y)(x-4y)$。

(1) 不正确，应为 $(2a-1)^2$；

(2) 不正确，应为 $(x+2y)(x-2y)$。

2. 把下列多项式分解因式：

$(1)\ a^2+a$； $(2)\ 4ab-2a^2b$； $(3)\ 9m^2-n^2$； $(4)\ 2am^2-8a$； $(5)\ 2a^2+4ab+2b^2$。

(1) $a(a+1)$；

(2) $2ab(2-a)$；

(3) $(3m+n)(3m-n)$；

(4) $2a(m-2)(m+2)$；

(5) $2(a+b)^2$。

📖 读一读 · 因式分解的起源

▶

因式分解的相关研究源远流长。在代数理论的演进中，阿拉伯数学家及文艺复兴时期的欧洲学者对多项式运算的探索，为因式分解理论的初步形成奠定了基础。19世纪，抽象代数和多项式理论的兴起使因式分解的研究进入新的阶段，其理论体系不断完善，成为现代代数的重要组成部分。

🧠 数学思想·核心素养

逆向思维： 因式分解是整式乘法的逆变形。
整体思想： 提公因式时把公因式看作整体。
转化思想： 利用公式将多项式转化为乘积形式。

📌 课堂小结

▼
因式分解：把一个多项式化成几个整式的积的形式。
提公因式法：找各项的公因式，然后写成 $m(a+b+c)$ 的形式。
公式法：平方差公式 $a^2-b^2=(a+b)(a-b)$，完全平方公式 $a^2\pm2ab+b^2=(a\pm b)^2$。
分解因式必须分解到每个因式不能再分解为止。
