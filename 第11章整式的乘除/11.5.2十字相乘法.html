第11章 整式的乘除

11.5.2 十字相乘法

🎯 学习目标

✔理解十字相乘法的原理
✔能运用十字相乘法分解二次三项式
✔掌握 $x^2+(a+b)x+ab=(x+a)(x+b)$

问题引入

计算：$(x+2)(x+3) = \underline{\qquad\qquad}$

$(x+2)(x+3) = x^2+5x+6$

反过来，如何把 $x^2+5x+6$ 分解成两个一次因式的乘积？这就是十字相乘法要解决的问题。

试一试

观察 $x^2+5x+6$，常数项 $6 = 2\times3$，一次项系数 $5 = 2+3$，所以 $x^2+5x+6 = (x+2)(x+3)$。

一般地，如果 $x^2+px+q$ 可以分解为 $(x+a)(x+b)$，那么 $p=a+b$，$q=ab$。

十字相乘法 (Cross Multiplication Method)
对于二次三项式 $x^2+px+q$，若能找到两个数 $a$、$b$，使 $a+b=p$，$ab=q$，则 $$x^2+px+q = (x+a)(x+b).$$ 十字相乘法的步骤（以 $x^2+5x+6$ 为例）：
$x$$2$
$x$$3$
$x\cdot3 + x\cdot2 = 5x$
对于二次项系数不是1的情形，也可类似分解。

例1 分解因式

$(1)\ x^2+7x+12$； $(2)\ x^2-5x+6$； $(3)\ x^2-2x-8$。

$(1)\ 3+4=7,\ 3\times4=12$，所以 $x^2+7x+12 = (x+3)(x+4)$；

$(2)\ (-2)+(-3)=-5,\ (-2)\times(-3)=6$，所以 $x^2-5x+6 = (x-2)(x-3)$；

$(3)\ 2+(-4)=-2,\ 2\times(-4)=-8$，所以 $x^2-2x-8 = (x+2)(x-4)$。

例2 分解因式

$(1)\ 2x^2+7x+3$； $(2)\ 3x^2-10x+3$。

$(1)\ 2x^2+7x+3 = (2x+1)(x+3)$（因为 $2x\cdot3 + x\cdot1 = 6x+x=7x$）；

$(2)\ 3x^2-10x+3 = (3x-1)(x-3)$（因为 $3x\cdot(-3) + x\cdot(-1) = -9x - x = -10x$）。

📖 读一读 · 十字相乘法的历史

▶

十字相乘法是一种直观的因式分解方法，其名称来源于计算过程中画出的交叉线形状。它最早出现在19世纪英国数学家德·摩根（A. De Morgan）的著作中，后来被广泛用于分解二次三项式。我国中学数学教材在20世纪50年代引入此法，成为因式分解的重要内容。

🧠 数学思想·核心素养

逆向思维： 从整式乘法得到分解方法。
转化思想： 将二次三项式转化为两个一次式乘积。
数形结合： 十字交叉帮助理解系数关系。

📌 课堂小结

▼
十字相乘法：$x^2+px+q = (x+a)(x+b)$，其中 $a+b=p$，$ab=q$。
对于 $ax^2+bx+c$，需将 $a$ 分解为 $a_1a_2$，$c$ 分解为 $c_1c_2$，使 $a_1c_2+a_2c_1 = b$。
注意符号：常数项为正时，两数同号；常数项为负时，两数异号。
