5.3 实践与探索
第三课时 工程问题与报酬分配

🎯 学习目标

✔会列方程解决合作、分配问题
✔能根据工作量计算报酬

问题3 课外活动时李老师来教室布置作业，有一道题目只写了“某工厂需制作一块广告牌，请来两名工人。已知师傅单独完成需4天，徒弟单独完成需6天”就停住了。片刻后，同学们带着疑惑的目光……

小明抢先说：“让我试一试。”于是，上去添了：“两人合作需要几天完成？”

李老师选了两位同学的问题，综合后，在黑板上写出：现由徒弟先做1天，再两人合作，完成后共得到报酬900元。如果按各人完成的工作量计算报酬，那么该如何分配？

试解答这一问题，并和同学交流各自的做法。

解：设两人合作需要 $x$ 天完成。

徒弟先做1天，完成工作量为 $\frac{1}{6}$；

两人合作的工作效率为 $\frac{1}{4} + \frac{1}{6} = \frac{5}{12}$，合作 $x$ 天完成工作量为 $\frac{5}{12}x$。

根据总工作量为1，得方程：$\frac{1}{6} + \frac{5}{12}x = 1$。

解得 $\frac{5}{12}x = \frac{5}{6}$，$x = 2$（天）。

因此，师徒工作时间及工作量如下：

人员单独完成时间工作效率工作时间工作量
师傅 4天 $\frac{1}{4}$ 2天（合作） $2 \times \frac{1}{4} = \frac{1}{2}$
徒弟 6天 $\frac{1}{6}$ 1天（先做）+2天（合作）=3天 $1 \times \frac{1}{6} + 2 \times \frac{1}{6} = \frac{1}{2}$

两人完成工作量相等，所以每人应得 $900 \div 2 = 450$ 元。

答：徒弟得450元，师傅得450元。你还有其他做法吗？可以和同学交流。

练习1

小慧阅读一本科普图书，原来每天阅读20页，读完120页后，抽出一定的时间练毛笔字，每天的阅读量降为原来的一半，结果多花了5天才读完，求这本科普图书的总页数。

解：设总页数为 $x$ 页。前120页用了 $\frac{120}{20} = 6$ 天。剩余 $x-120$ 页，后来每天读10页，用了 $\frac{x-120}{10}$ 天。实际总天数为 $6 + \frac{x-120}{10}$，原计划天数为 $\frac{x}{20}$，多花5天，所以 $6 + \frac{x-120}{10} = \frac{x}{20} + 5$。

乘以20：$120 + 2(x-120) = x + 100$，$120 + 2x - 240 = x + 100$，$2x -120 = x + 100$，$x = 220$。

答：总页数为220页。

练习2

某航空公司规定：乘坐飞机经济舱的旅客每人最多可免费托运20kg行李，超过部分每千克按飞机票价的1.5%购买行李票。一位乘坐经济舱的旅客托运了35kg行李，机票连同行李费共付了1323元，求这位旅客的机票价。

解：设机票价为 $x$ 元。超重 $35-20=15$ kg，行李费 $15 \times 1.5\% x = 0.225x$ 元。总费用 $x + 0.225x = 1.225x = 1323$，解得 $x = \frac{1323}{1.225} = 1080$ 元。

答：机票价为1080元。

练习3

为庆祝学校运动会开幕，七年级(2)班同学接受了制作小旗的任务，原计划一半同学参加制作，每天制作40面。完成了三分之一以后，全班同学一起参加了制作，结果比原计划提前一天半完成任务。假设每人的制作效率相同，问：共制作小旗多少面？

解：设共制作小旗 $x$ 面。原计划一半同学每天40面，则全班同学每天80面。前 $\frac{1}{3}x$ 面用了 $\frac{x/3}{40} = \frac{x}{120}$ 天；后 $\frac{2}{3}x$ 面用了 $\frac{2x/3}{80} = \frac{x}{120}$ 天；实际总天数为 $\frac{x}{120} + \frac{x}{120} = \frac{x}{60}$ 天。原计划一半同学全程制作需 $\frac{x}{40}$ 天。提前1.5天，所以 $\frac{x}{40} - \frac{x}{60} = 1.5$。

解得 $\frac{3x - 2x}{120} = 1.5$，$\frac{x}{120} = 1.5$，$x = 180$。

答：共制作小旗180面。

📖 读一读 · 《九章算术》中的“共作”问题

▶

《九章算术》第七章“盈不足”中有一类“共作”问题，例如：“今有甲乙二人共作一器，甲日作八分，乙日作六分，今共作十日，成器。问各作几日？”这相当于已知工作效率和合作时间，求各自工作时间。古人用“盈不足术”求解，与现代设未知数列方程的方法一致。这类问题反映了古代劳动人民在工程合作方面的数学智慧。

📌 课堂小结

工程问题中，常把总工作量看作1，工作效率 = 1/工作时间
工作量按比例分配报酬，先计算每人完成的工作量
分析题意时，要分清工作阶段、人员变化

🧠 数学思想·核心素养

建模思想： 将工程问题抽象为方程模型。
数形结合： 画线段图分析工作量。
逻辑推理： 根据条件逐步推导未知量。
