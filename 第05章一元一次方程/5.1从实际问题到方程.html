第5章 一元一次方程

★ 从实际问题中抽象出方程，体会方程是刻画现实世界的有效模型。
5.1 从实际问题到方程

问题1 猜年龄

课外活动中，张老师组织同学们进行“猜年龄”游戏，她首先提出如下问题：

同学们今年的年龄是13岁，我今年的年龄是45岁，经过几年我的年龄正好是你们年龄的3倍？

解法1（尝试－检验） 小敏首先发现了答案：

  • 经过1年，同学们14岁，老师46岁，不是3倍；
  • 经过2年，同学们15岁，老师47岁，不是3倍；
  • 经过3年，同学们16岁，老师48岁，恰好是3倍。

解法2（分析－列算式） 年龄差不变：\(45-13=32\)（岁）。当老师年龄是同学3倍时，年龄差是同学年龄的2倍，此时同学年龄 \(32\div2=16\)（岁），所以经过 \(16-13=3\) 年。

探索

如果设经过 \(x\) 年，老师的年龄为 \(45+x\)，同学的年龄为 \(13+x\)，根据“老师的年龄 = 3 × 同学的年龄”，可得方程：

\(45 + x = 3(13 + x)\)

试一试

同学们今年的年龄是13岁，班主任李老师今年的年龄是55岁，经过几年李老师的年龄是同学们年龄的3倍？

设经过 \(x\) 年，则 \(55 + x = 3(13 + x)\)。

问题2 学校运动队晨跑

甲、乙两队员同时出发，跑完一圈乙比甲多用 \(1\min\)，已知甲、乙队员跑步的平均速度分别是 \(4\ \mathrm{m/s}\)、\(3.5\ \mathrm{m/s}\)，这一圈步道有多长？

探索：设步道一圈的长为 \(x\ \mathrm{m}\)，则乙所用时间为 \(\frac{x}{3.5}\ \mathrm{s}\)，甲所用时间为 \(\frac{x}{4}\ \mathrm{s}\)。由“乙用时 = 甲用时 + 60”得方程：

\(\displaystyle \frac{x}{3.5} = \frac{x}{4} + 60\)

📌 方程、方程的解、解方程

方程：含有未知数的等式叫做方程。
方程的解：能使方程左、右两边相等的未知数的值叫做方程的解。例如 \(x=3\) 是方程 \(45+x=3(13+x)\) 的解。
根：一元方程的解也叫根。
解方程：求方程解的过程叫做解方程。

📖 读一读 · 尝试检验法

▶

尝试检验法是一种通过代入可能的值来检验方程解的方法。例如解方程 \(45 + x = 3(13 + x)\)，可以列表如下：

\(x\)左边 \(45+x\)右边 \(3(13+x)\)是否相等
14642不相等
24745不相等
34848相等
44951不相等

从而发现 \(x=3\) 是方程的解。这种方法虽然有时较慢，但直观易懂。

✏️ 练习

根据题意列出方程（不必求解）：

  1. 某班原分成两个小组进行课外体育活动，第一组26人，第二组22人，现根据学校活动器材的数量，要将第一组的人数调整为第二组的一半，应从第一组调多少人到第二组去？
  2. 加工某种零件，师傅平均每小时做5个，徒弟平均每小时做4个，加工一盒零件，师傅比徒弟少用 \(2\ \mathrm{h}\)。问：一盒零件有多少个？

1. 设应从第一组调 \(x\) 人到第二组，则调整后第一组 \(26-x\) 人，第二组 \(22+x\) 人，根据题意得：\(26-x = \frac{1}{2}(22+x)\)。

2. 设一盒零件有 \(x\) 个，则师傅用时 \(\frac{x}{5}\) h，徒弟用时 \(\frac{x}{4}\) h，根据题意得：\(\frac{x}{4} - \frac{x}{5} = 2\)。

📚 习题5.1 A组

  1. 检验下列方程后面大括号内所列各数是否为相应方程的解：
    (1) \(\frac{5x+1}{8}=x-1,\quad \{-5,3\}\);
    (2) \(2(y-2)-9(1-y)=3(4y-1),\quad \{-10,10\}\).
  2. 小明去商店买练习本，回来后问同学：“店主告诉我，如果多买一些就给我打八折，于是，我就买了20本，结果便宜了4.80元，原来每本的价格是多少？”你能列出方程吗？
  3. 根据班级内男、女同学的人数编一道应用题，和同学交流一下。

(1) 当 \(x=-5\) 时，左边=\(\frac{5(-5)+1}{8}=\frac{-24}{8}=-3\)，右边=\(-5-1=-6\)，左边≠右边，不是解；当 \(x=3\) 时，左边=\(\frac{16}{8}=2\)，右边=\(2\)，左边=右边，是解。

(2) 当 \(y=-10\) 时，左边=\(2(-12)-9(11)=-24-99=-123\)，右边=\(3(-40-1)=3(-41)=-123\)，左边=右边，是解；当 \(y=10\) 时，左边=\(2(8)-9(-9)=16+81=97\)，右边=\(3(40-1)=117\)，左边≠右边，不是解。

设原来每本 \(x\) 元，则20本原价 \(20x\) 元，八折后总价 \(20x \times 0.8 = 16x\) 元，便宜了 \(20x - 16x = 4.8\)，即 \(4x = 4.8\)。

📚 习题5.1 B组

  1. 根据题意列出方程（不必求解）：
    (1) 某班到离校 \(30\ \mathrm{km}\) 的国家森林公园春游，先坐车，速度为 \(36\ \mathrm{km/h}\)，下车后以 \(6\ \mathrm{km/h}\) 的速度步行到达目的地，共花了 \(1\ \mathrm{h}\)。问：他们步行了多少时间？
    (2) 某车间接到一批小家电组装任务，原计划每天组装36台，预计若干天完成，在组装了任务的三分之一后，调整工序，改进操作技术，工效提高了1倍，结果提前2天完成任务，求这次组装小家电的总台数。

(1) 设步行了 \(x\ \mathrm{h}\)，则坐车时间为 \(1-x\) h，根据路程关系：\(36(1-x) + 6x = 30\)。

(2) 设总台数为 \(x\)，原计划天数为 \(\frac{x}{36}\) 天；前 \(\frac{1}{3}\) 任务用了 \(\frac{x}{3\times36}\) 天，后 \(\frac{2}{3}\) 任务工效提高一倍即每天72台，用了 \(\frac{2x}{3\times72}\) 天，实际总天数为 \(\frac{x}{108} + \frac{x}{108} = \frac{x}{54}\) 天。提前2天，即 \(\frac{x}{36} - \frac{x}{54} = 2\)。
