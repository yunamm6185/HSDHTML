8.1.2 三角形的内角和与外角和

三角形的内角和等于 $180^{\circ}$。

证明：三角形的内角和等于180°

如图，延长 $BC$ 至 $E$，过点 $C$ 作 $CD \parallel BA$，则 $\angle B = \angle DCE$，$\angle A = \angle ACD$。

图8.1.8
图8.1.9

已知：$\triangle ABC$，求证：$\angle A + \angle B + \angle C = 180°$。

证明：

\[ \begin{aligned} &\because CD \parallel BA \quad (\text{作图}), \\ &\therefore \angle B = \angle DCE \quad (\text{同位角相等}), \\ &\angle A = \angle ACD \quad (\text{内错角相等}). \\ &\because \angle ACB + \angle ACD + \angle DCE = 180^{\circ} \quad (\text{平角定义}), \\ &\therefore \angle A + \angle B + \angle ACB = 180^{\circ} \quad (\text{等量代换}). \end{aligned} \]

因此，三角形的内角和等于 $180^{\circ}$。

思考

如图8.1.10，在 Rt$\triangle ABC$ 中，$\angle C = 90°$，则 $\angle A$ 与 $\angle B$ 有什么关系？

图8.1.10

$\angle A + \angle B = 90°$，即直角三角形的两个锐角互余。

有两个角互余的三角形是直角三角形。

例1

如图8.1.11，$AD$ 是 $\triangle ABC$ 的边 $BC$ 上的高，$\angle 1 = 45°$，$\angle C = 65°$。求 $\angle BAC$ 的度数。

图8.1.11
\[ \begin{aligned} &\text{在 Rt}\triangle ABD \text{中，} \angle 1 + \angle B = 90^{\circ}, \\ &\therefore \angle B = 90^{\circ} - \angle 1 = 90^{\circ} - 45^{\circ} = 45^{\circ}. \\ &\text{在 }\triangle ABC \text{中，} \angle BAC = 180^{\circ} - \angle B - \angle C, \\ &\therefore \angle BAC = 180^{\circ} - 45^{\circ} - 65^{\circ} = 70^{\circ}. \end{aligned} \]

三角形外角的性质：

  • 三角形的一个外角等于与它不相邻的两个内角的和；
  • 三角形的一个外角大于任何一个与它不相邻的内角。

做一做

如图8.1.14，$\angle 1 + \angle 2 + \angle 3$ 是三角形的外角和。计算 $\angle 1 + \angle 2 + \angle 3$ 的值。

图8.1.14
\[ \begin{aligned} &\angle 1 + \angle ACB = 180^{\circ},\\ &\angle 2 + \angle BAC = 180^{\circ},\\ &\angle 3 + \angle ABC = 180^{\circ}.\\ &\text{三式相加：}\\ &(\angle 1+\angle2+\angle3) + (\angle ACB+\angle BAC+\angle ABC) = 540^{\circ}.\\ &\because \angle ACB+\angle BAC+\angle ABC = 180^{\circ},\\ &\therefore \angle 1+\angle2+\angle3 = 540^{\circ} - 180^{\circ} = 360^{\circ}. \end{aligned} \]

例2

如图8.1.15，$D$ 是 $\triangle ABC$ 的边 $BC$ 上一点，$\angle B = \angle BAD$，$\angle ADC = 80°$，$\angle BAC = 70°$。求 $\angle B$ 和 $\angle C$ 的度数。

图8.1.15
\[ \begin{aligned} &\because \angle ADC \text{ 是 } \triangle ABD \text{ 的外角},\\ &\therefore \angle B + \angle BAD = \angle ADC = 80^{\circ}.\\ &\because \angle B = \angle BAD,\\ &\therefore \angle B = \frac{80^{\circ}}{2} = 40^{\circ}.\\[1em] &\text{在 } \triangle ABC \text{ 中},\\ &\angle C = 180^{\circ} - \angle B - \angle BAC \\ &\quad = 180^{\circ} - 40^{\circ} - 70^{\circ} = 70^{\circ}. \end{aligned} \]

1. 如图，∠A = 40°，则 ∠1 + ∠2 + ∠3 + ∠4 = ______。

第1题

∠1 + ∠2 + ∠3 + ∠4 = 280°

2. 在△ABC中，∠A + ∠B = 80°，∠C = 2∠B。求∠A、∠B和∠C的度数。

\[ \begin{aligned} &\angle A + \angle B + \angle C = 180^\circ,\\ &\angle A + \angle B = 80^\circ,\ \angle C = 2\angle B,\\ &\therefore 80^\circ + 2\angle B = 180^\circ,\\ &2\angle B = 100^\circ,\ \angle B = 50^\circ,\\ &\angle A = 80^\circ - 50^\circ = 30^\circ,\\ &\angle C = 2 \times 50^\circ = 100^\circ. \end{aligned} \]

3. 在△ABC中，∠B = ∠A + 30°，∠C = ∠B + 30°。求△ABC的各内角的度数。

\[ \begin{aligned} &\text{设 } \angle A = x,\quad \angle B = x + 30^\circ,\quad \angle C = x + 60^\circ,\\ &x + (x + 30^\circ) + (x + 60^\circ) = 180^\circ,\\ &3x + 90^\circ = 180^\circ,\quad 3x = 90^\circ,\quad x = 30^\circ,\\ &\angle A = 30^\circ,\quad \angle B = 60^\circ,\quad \angle C = 90^\circ. \end{aligned} \]

4. 如图，在Rt△ABC中，∠C = 90°，D、E分别是边CB、AB延长线上的点，∠A = ∠D。试说明△BDE是直角三角形。

第4题
\[ \begin{aligned} &\because \angle C = 90^\circ,\quad \angle A + \angle ABC = 90^\circ,\\ &\angle D = \angle A,\quad \angle DBE = \angle ABC \ (\text{对顶角相等}),\\ &\therefore \angle D + \angle DBE = \angle A + \angle ABC = 90^\circ,\\ &\therefore \triangle BDE \text{ 中，}\angle BDE = 90^\circ,\ \text{即} \triangle BDE \text{ 是直角三角形}. \end{aligned} \]

5. （口答）一个三角形可以有两个内角都是直角吗？可以有两个内角都是钝角或都是锐角吗？为什么？

不能有两个直角或两个钝角，因为内角和会超过180°；可以有两个锐角。

解（答案）

(1) ∵ CD ⊥ AB（已知），

∴ ∠CDB = 90°。

∴ ∠EBC = ∠CDB + ∠BCD（三角形的一个外角等于与它不相邻的两个内角的和），

∠BCD = 35°（已知），

∴ ∠EBC = 90° + 35° = 125°（等量代换）。

(2) ∵ ∠EBC = ∠A + ∠ACB（三角形的一个外角等于与它不相邻的两个内角的和），

∴ ∠A = ∠EBC - ∠ACB（等式的性质）。

∵ ∠ACB = 90°（已知），

∴ ∠A = 125° - 90° = 35°（等量代换）。
