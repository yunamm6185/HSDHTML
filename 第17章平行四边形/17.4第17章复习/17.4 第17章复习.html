四、本章要点

📌 性质定理（从平行四边形出发）

∵ 四边形 $ABCD$ 是 平行四边形，
∴ 对边相等 | 对角相等 | 对角线互相平分 | 是中心对称图形

📌 判定定理（推导出平行四边形）

① 定义法（两组对边平行） ② 两组对边相等 ③ 一组对边平行且相等 ④ 对角线互相平分 ⑤ 两组对角相等

📌 三角形中位线定理

$D$、$E$ 分别是 $AB$、$AC$ 的中点 ⇒ $DE \parallel BC$，且 $DE = \dfrac{1}{2}BC$

🔑
核心关系：性质与判定互为逆命题——从性质逆向思考，可得到判定定理。这就是几何研究中"探索—猜想—证明"的基本范式。
五、方法归纳

判定方法选择策略 & 常用辅助线

🔍 条件分析法

条件在边上 → 用"对边相等"或"平行且相等"

条件在角上 → 用"对角相等"或"邻角互补→平行"

条件在对角线上 → 用"对角线互相平分"

✏️ 常用辅助线

① 连结对角线

② 倍长中位线（构造平行四边形）

③ 过点作平行线

④ 延长线段

六、典例精讲

典例 1（性质应用）

如图，在 $\parallelogram ABCD$ 中，对角线 $AC$ 与 $BD$ 相交于点 $O$，点 $E$ 是 $AD$ 的中点。若 $OE = 3$，求 $AB$ 的长。

分析：在 ▱$ABCD$ 中，$O$ 是 $AC$ 的中点（对角线互相平分）。$E$ 是 $AD$ 的中点。所以 $OE$ 是 $\triangle ACD$ 的中位线 → $OE \parallel CD$ 且 $OE = \dfrac{1}{2}CD$。而 $CD = AB$。

解： ∵ 四边形 $ABCD$ 是平行四边形，

∴ $OA = OC$（平行四边形对角线互相平分），即 $O$ 是 $AC$ 的中点。

又 ∵ $E$ 是 $AD$ 的中点，

∴ $OE$ 是 $\triangle ACD$ 的中位线。

∴ $OE = \dfrac{1}{2}CD = 3$，得 $CD = 6$。

又 ∵ $AB = CD$（平行四边形对边相等），

∴ $AB = 6$。

典例 2（判定应用）

如图，在 $\triangle ABC$ 中，$D$、$E$、$F$ 分别是 $AB$、$BC$、$CA$ 的中点。
求证：四边形 $DECF$ 是平行四边形。

证明： ∵ $D$、$E$ 分别是 $AB$、$BC$ 的中点，

∴ $DE \parallel AC$（三角形中位线定理），即 $DE \parallel CF$。

同理，∵ $E$、$F$ 分别是 $BC$、$CA$ 的中点，

∴ $EF \parallel AB$，即 $EF \parallel DC$。

∴ 四边形 $DECF$ 是平行四边形（两组对边分别平行的四边形是平行四边形）。

方法反思：本题还可从不同角度证明：

• 方法二：$DE = CF$ 且 $DE \parallel CF$ → 一组对边平行且相等。

• 方法三：证 $DE = CF$ 且 $DF = CE$ → 两组对边分别相等。

这体现了"一题多解"，选择最简捷的方法需要根据"已知条件就近"原则。

七、分层练习

基础篇 A组 双基巩固

1. 在 ▱$ABCD$ 中，$\angle A = 65^\circ$，则 $\angle B =$ ______，$\angle C =$ ______，$\angle D =$ ______。

$\angle B = 115^\circ$，$\angle C = 65^\circ$，$\angle D = 115^\circ$。（邻角互补，对角相等）

2. ▱$ABCD$ 的周长为 $30$，$AB = 8$，则 $BC =$ ______，$CD =$ ______。

$BC = 7$，$CD = 8$。（对边相等：$AB + BC = 15$）

3. 如图，在 ▱$ABCD$ 中，对角线 $AC$、$BD$ 相交于点 $O$，下列结论不一定成立的是（ ）

A. $AB = CD$   B. $OA = OC$   C. $AC \perp BD$   D. $\angle ABC = \angle CDA$

C。对角线互相平分但不一定垂直，只有菱形/正方形的对角线才垂直。

4. 补充一个条件，使四边形 $ABCD$ 成为平行四边形：已知 $AB \parallel CD$，再添加条件 ______。

$AB = CD$（一组对边平行且相等）；或 $AD \parallel BC$（定义法）。本题为开放性题目，答案不唯一。

提高篇 B组 综合运用

5. 在 $\parallelogram ABCD$ 中，点 $E$、$F$ 分别是 $AD$、$BC$ 的中点。
求证：四边形 $BFDE$ 是平行四边形。（试用两种不同方法证明）

方法一：$DE = \dfrac{1}{2}AD$，$BF = \dfrac{1}{2}BC$，由 $AD = BC$ 得 $DE = BF$。又 $AD \parallel BC$ 即 $DE \parallel BF$。∴ ▱$BFDE$。

方法二：证 $\triangle ABE \cong \triangle CDF$（SAS），得 $BE = DF$。结合 $DE = BF$，两组对边相等。

6. 顺次连接四边形 $ABCD$ 各边中点 $E$、$F$、$G$、$H$。

求证：(1) 四边形 $EFGH$ 是平行四边形；

(2) 四边形 $EFGH$ 的周长等于 $AC + BD$。

(1) 连接 $AC$。$E$、$F$ 分别是 $AB$、$BC$ 中点 → $EF \parallel AC$ 且 $EF = \dfrac{1}{2}AC$。同理 $GH \parallel AC$ 且 $GH = \dfrac{1}{2}AC$。∴ $EF \parallel GH$ 且 $EF = GH$ → ▱$EFGH$。

(2) 同理 $FG = \dfrac{1}{2}BD$，$EH = \dfrac{1}{2}BD$。周长 $= EF + FG + GH + EH = AC + BD$。

挑战篇 C组 拓展思考

7. 如图，在 $\triangle ABC$ 中，$D$、$E$、$F$ 分别是各边中点，$AH$ 是 $BC$ 边上的高。求证：$\angle DHF = \angle DEF$。

分析：在 Rt$\triangle ABH$ 中，$D$ 是斜边 $AB$ 中点 → $DH = \dfrac{1}{2}AB$。

又 $EF$ 是 $\triangle ABC$ 中位线 → $EF = \dfrac{1}{2}AB$。∴ $DH = EF$。

同理，$FH = \dfrac{1}{2}AC = DE$。

又 $DF$ 为公共边 → $\triangle DHF \cong \triangle EFD$（SSS）。

∴ $\angle DHF = \angle DEF$。

八、课堂小结
