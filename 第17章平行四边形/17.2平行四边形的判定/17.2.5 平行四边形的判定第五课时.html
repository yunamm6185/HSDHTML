例 7

如图 17.2.18，已知 $\parallelogram ABCD$，延长边 $AD$ 至点 $F$，使 $DF = DA$。连结 $BF$，交边 $DC$ 于点 $E$。
求证：$EF = EB$。

证明：∵ 四边形 $ABCD$ 是平行四边形，

∴ $DA \parallel CB$（平行四边形的对边平行且相等）。

∴ $\angle FDE = \angle BCE$，$\angle DFE = \angle CBE$。

又 ∵ $DA = DF$，

∴ $DF = CB$。

在 $\triangle DFE$ 与 $\triangle CBE$ 中，

∵ $\angle FDE = \angle BCE$，$DF = CB$，$\angle DFE = \angle CBE$，

∴ $\triangle DFE \cong \triangle CBE$。

∴ $EF = EB$。

情景导入

在图 17.2.18 中，点 $D$、$E$ 分别是 $\triangle AFB$ 的两边 $AF$、$BF$ 的中点，即 $DE$ 是连结 $\triangle AFB$ 的两边中点的线段。连结三角形两边中点的线段，叫做三角形的中位线（median line）。

新知探索

三角形的中位线

定义：连结三角形两边中点的线段，叫做三角形的中位线（median line）。

如图 17.2.27，在 $\triangle ABC$ 中，$D$、$E$ 分别是 $AB$、$AC$ 的中点，则线段 $DE$ 是 $\triangle ABC$ 的一条中位线。一个三角形有 3 条中位线。

三角形中位线定理

三角形中位线定理

三角形的中位线平行于第三边，并且等于第三边的一半。

例 8 如图 17.2.19，在 $\triangle ABC$ 中，点 $D$、$E$ 分别是边 $AB$ 和 $AC$ 的中点。求证：$DE \parallel BC$，$DE = \dfrac{1}{2}BC$。

分析： 如图 17.2.20，过点 $C$ 作 $CF \parallel AB$，且与 $DE$ 的延长线交于点 $F$。由平行线性质和已知条件，可以证明 $\triangle ADE \cong \triangle CFE$，从而推出四边形 $BCFD$ 是平行四边形，可得 $DE \parallel BC$，$DE = EF = \dfrac{1}{2}BC$。

证明： 如图 17.2.20，延长 $DE$ 到点 $F$，使 $EF = DE$，连接 $CF$。

在 $\triangle ADE$ 和 $\triangle CFE$ 中，

$\because AE = CE$（$E$ 是 $AC$ 的中点），

$\angle AED = \angle CEF$（对顶角相等），

$DE = FE$（所作），

$\therefore \triangle ADE \cong \triangle CFE$（SAS）。

$\therefore AD = CF$，$\angle ADE = \angle CFE$。

$\therefore AB \parallel CF$（内错角相等，两直线平行）。

又 $\because AD = BD$（$D$ 是 $AB$ 的中点），

$\therefore BD = CF$。

$\therefore$ 四边形 $BCFD$ 是平行四边形。

$\therefore DF \parallel BC$，$DF = BC$。

又 $\because DE = \dfrac{1}{2}DF$，

$\therefore DE \parallel BC$，且 $DE = \dfrac{1}{2}BC$。

三角形中位线定理几何语言

在 $\triangle ABC$ 中，若 $D$、$E$ 分别是 $AB$、$AC$ 的中点，
则 $DE \parallel BC$，且 $DE = \dfrac{1}{2}BC$。

例 9

证明：三角形的一条中位线与第三边上的中线互相平分。

已知：如图 17.2.30，在 $\triangle ABC$ 中，$AD = DB$，$BF = FC$，$AE = EC$。

求证：$AF$ 与 $DE$ 互相平分。

证明： 如图 17.2.31，连结 $DF$、$EF$。

$\because AD = DB$，$BF = FC$，

$\therefore DF \parallel AC$（三角形中位线定理）。

同理可得 $EF \parallel AB$。

$\therefore$ 四边形 $ADFE$ 是平行四边形。

$\therefore AF$ 与 $DE$ 互相平分。

巩固练习

1. 三角形的周长为 $56\text{ cm}$，求它的三条中位线组成的三角形的周长。

解： 设原三角形三边为 $a$、$b$、$c$，则 $a + b + c = 56\text{ cm}$。

三条中位线分别长 $\dfrac{1}{2}a$、$\dfrac{1}{2}b$、$\dfrac{1}{2}c$。

$\therefore$ 中位线组成的三角形周长为 $\dfrac{1}{2}(a + b + c) = \dfrac{1}{2} \times 56 = 28\text{ cm}$。

2. 如图，在 $\triangle ABC$ 中，$D$、$E$、$F$ 分别是 $AB$、$BC$、$CA$ 的中点。求四边形 $ADEF$ 的周长（用 $\triangle ABC$ 的边长表示）。

解： $\because D$、$E$、$F$ 分别是各边中点，

$\therefore DE \parallel AC$，$DE = \dfrac{1}{2}AC$；$EF \parallel AB$，$EF = \dfrac{1}{2}AB$。

又 $AD = \dfrac{1}{2}AB$，$AF = \dfrac{1}{2}AC$。

$\therefore$ 四边形 $ADEF$ 的周长 $= AD + DE + EF + AF$

$= \dfrac{1}{2}AB + \dfrac{1}{2}AC + \dfrac{1}{2}AB + \dfrac{1}{2}AC = AB + AC$。

顺次连结四边形各边中点所得图形是什么？

答：顺次连结任意四边形各边中点，所得四边形是平行四边形。

（证明：连结一条对角线，利用三角形中位线定理，可得对边平行且相等。）
