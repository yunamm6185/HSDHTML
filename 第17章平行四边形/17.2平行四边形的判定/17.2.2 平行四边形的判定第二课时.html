17.2 平行四边形的判定

课题导入

上节课我们学习了平行四边形的两个判定定理。今天，我们将从平行四边形的另一条重要性质——“对角线互相平分”出发，逆向思考，探索新的判定方法。

思考

逆向思考

由平行四边形的性质“平行四边形的两条对角线互相平分”，逆向思考，互换条件与结论，试写出它的逆命题。你认为它是一个真命题吗？

条件 结论
平行四边形的两条对角线互相平分 四边形是平行四边形 两条对角线互相平分
逆命题
逆命题 两条对角线互相平分 四边形是平行四边形

猜想： 这个逆命题是真命题。下面我们通过作图和证明来验证。

新知探索

试一试

如图 17.2.8，作一个两条对角线互相平分的四边形。

作法：

(1) 任意作两条相交直线 $m$、$n$，记交点为 $O$；

(2) 以点 $O$ 为中心，分别在直线 $m$、$n$ 上截取 $OB$ 与 $OD$、$OA$ 与 $OC$，使 $OB = OD$，$OA = OC$，顺次连接所得的四个点。

四边形 $ABCD$ 即为所要求作的四边形。

它是平行四边形吗？

判定定理 3

判定定理 3

对角线互相平分的四边形是平行四边形。

已知： 如图 17.2.9，在四边形 $ABCD$ 中，对角线 $AC$ 与 $BD$ 相交于点 $O$，$OA = OC$，$OB = OD$。

求证： 四边形 $ABCD$ 是平行四边形。

证明： 如图 17.2.9，在 $\triangle AOB$ 和 $\triangle COD$ 中，

$\because OA = OC$，$\angle 1 = \angle 2$（对顶角相等），$OB = OD$，

$\therefore \triangle AOB \cong \triangle COD$（SAS）。

$\therefore \angle OAB = \angle OCD$。

$\therefore AB \parallel CD$（内错角相等，两直线平行）。

同理可证 $AD \parallel BC$。

$\therefore$ 四边形 $ABCD$ 是平行四边形（两组对边分别平行的四边形是平行四边形）。

另证： 由 $\triangle AOB \cong \triangle COD$ 可得 $AB = CD$，同理 $AD = BC$，由“两组对边分别相等的四边形是平行四边形”也可得证。

例题解析

例 2

如图 17.2.10，在 $▱ABCD$ 中，点 $E$、$F$ 是对角线 $AC$ 上的两点，且 $AE = CF$。求证：四边形 $BFDE$ 是平行四边形。

分析： 连接 $BD$，交 $AC$ 于点 $O$。由四边形 $ABCD$ 是平行四边形，可得 $OB = OD$。如果能证明 $OE = OF$，就可以根据“对角线互相平分的四边形是平行四边形”得到四边形 $BFDE$ 是平行四边形。（在复杂图形中，主动添加对角线 $BD$ 是常用的辅助线思路。）

证明： 如图 17.2.11，连接 $BD$，交 $AC$ 于点 $O$。

$\because$ 四边形 $ABCD$ 是平行四边形，

$\therefore OB = OD$，$OA = OC$（平行四边形的对角线互相平分）。

又 $\because AE = CF$，

$\therefore OA - AE = OC - CF$，

即 $OE = OF$。

$\therefore$ 四边形 $BFDE$ 是平行四边形（对角线互相平分的四边形是平行四边形）。

思考与总结

方法梳理

结合上节课所学的两个判定定理，我们总共学习了多少种判定平行四边形的方法（包括定义）？这些判定方法与平行四边形的性质之间，又有怎样的关系呢？

平行四边形的判定方法汇总：

1. 定义法： 两组对边分别平行的四边形是平行四边形。

2. 两组对边法： 两组对边分别相等的四边形是平行四边形。

3. 一组对边法： 一组对边平行且相等的四边形是平行四边形。

4. 对角线法： 对角线互相平分的四边形是平行四边形。

这四种方法从边和对角线两个角度，完整地构建了平行四边形的判定体系，它们与性质定理互为逆定理，体现了数学中的对称与统一。

读一读

由平行四边形的性质，联想平行四边形的判定方法，通过合情推理，提出猜想。这是一个由原命题到逆命题的逆向思维过程，今后在探索和研究其他几何问题时还会继续运用。

巩固练习

1. 如图，延长 $\triangle ABC$ 的中线 $AD$ 至点 $E$，使 $DE = AD$，那么四边形 $ABEC$ 是平行四边形吗？为什么？

解： 四边形 $ABEC$ 是平行四边形。

$\because AD$ 是 $\triangle ABC$ 的中线，$\therefore BD = CD$。

又 $\because DE = AD$，

$\therefore$ 四边形 $ABEC$ 的对角线 $AE$ 与 $BC$ 互相平分于点 $D$。

$\therefore$ 四边形 $ABEC$ 是平行四边形（对角线互相平分的四边形是平行四边形）。

2. 如图，在 $▱ABCD$ 中，两条对角线 $AC$ 与 $BD$ 相交于点 $O$，$E$、$F$、$G$、$H$ 分别是 $AO$、$BO$、$CO$、$DO$ 的中点。以图中标明字母的点为顶点，尽可能多地画出平行四边形。

分析： 利用对角线互相平分的判定定理，可以找出以下平行四边形：

  • $▱ABCD$（已知）；
  • $▱EFGH$（$OE=OG$，$OF=OH$）；
  • $▱BEDG$（$OB=OD$，$OE=OG$）；
  • $▱AFCH$（$OA=OC$，$OF=OH$）。

此外，由中位线等知识还可发现 $▱AEFD$、$▱EBCF$ 等。你能继续找出其他的吗？利用对边平行且相等的方法试一试。

3. 如图，在四边形 $ABCD$ 中，$AB \parallel CD$，对角线 $AC$、$BD$ 相交于点 $O$，$EF$ 过点 $O$ 交 $AB$ 于点 $E$，交 $CD$ 于点 $F$，且 $OE = OF$。求证：四边形 $ABCD$ 是平行四边形。

证明： $\because AB \parallel CD$，

$\therefore \angle OAE = \angle OCF$（两直线平行，内错角相等），

$\angle OEA = \angle OFC$（两直线平行，内错角相等）。

又 $\because OE = OF$，

$\therefore \triangle OAE \cong \triangle OCF$（AAS）。

$\therefore OA = OC$。

同理可证 $\triangle OBE \cong \triangle ODF$，得 $OB = OD$。

$\therefore$ 四边形 $ABCD$ 的对角线互相平分。

$\therefore$ 四边形 $ABCD$ 是平行四边形（对角线互相平分的四边形是平行四边形）。
