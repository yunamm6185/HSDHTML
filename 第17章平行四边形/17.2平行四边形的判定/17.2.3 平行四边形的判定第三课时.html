17.2 平行四边形的判定

第 3 课时：综合运用判定方法
学习目标

目标

  • 综合运用多种判定方法进行证明。
  • 根据条件灵活选择判定方法。
  • 体会"构造中间平行四边形"的转化策略。
课题导入

我们已经学习了四种判定平行四边形的方法（包括定义）。当题目条件不同时，如何选择最合适的判定方法？本节课通过例题体会判定方法的灵活运用。

知识铺垫

1. 定义法：两组对边分别平行
2. 两组对边法：两组对边分别相等
3. 一组对边法：一组对边平行且相等
4. 对角线法：对角线互相平分

例 3

例 3

如图1，在 $\parallelogram ABCD$ 中，点 $F$、$H$ 分别在边 $AB$、$CD$ 上，且 $BF = DH$。求证：$AC$ 与 $HF$ 互相平分。

分析：$AC$ 和 $HF$ 是四边形 $AFCH$ 的对角线，要证它们互相平分，只需证四边形 $AFCH$ 是平行四边形。

证明：如图2，分别连结 $AH$、$CF$。

∵ 四边形 $ABCD$ 是平行四边形，

∴ $AB \parallel CD$，$AB = CD$。

又 ∵ $BF = DH$，∴ $AB - BF = CD - DH$，即 $AF = CH$。

∴ 四边形 $AFCH$ 是▱（一组对边平行且相等）。

∴ $AC$ 与 $HF$ 互相平分。

例 4

例 4

如图3，在四边形 $ABCD$ 中，$\angle A = \angle C$，$\angle B = \angle D$。求证：四边形 $ABCD$ 是平行四边形。

证明：∵ $\angle A + \angle B + \angle C + \angle D = 360^\circ$，$\angle A = \angle C$，$\angle B = \angle D$，

∴ $2(\angle A + \angle B) = 360^\circ$，即 $\angle A + \angle B = 180^\circ$。∴ $AD \parallel BC$。

同理 $AB \parallel CD$。∴ 四边形 $ABCD$ 是平行四边形。

两组对角分别相等的四边形是平行四边形。（PS：这个结论可以直接用吗？）
方法归纳

判定方法选择策略

1. 条件涉及边 → 优先"两组对边相等"或"一组对边平行且相等"。
2. 条件涉及角 → 优先"对角相等"或"邻角互补→平行"。
3. 条件涉及对角线 → 优先"对角线互相平分"。
4. 结论需"互相平分" → 先证四边形是▱，再运用性质。

巩固练习

1. 如图，在 $\parallelogram ABCD$ 中，点 $E$、$F$ 分别在 $BC$、$AD$ 上，且 $AE \parallel CF$。求证：$AE = CF$。

证明：∵ $AD \parallel BC$ 即 $AF \parallel EC$，又 $AE \parallel CF$，∴ 四边形$AECF$是平行四边形。∴ $AE = CF$。

2. 如图，在 $\parallelogram ABCD$ 中，$E$、$F$、$G$、$H$ 是各边中点。求证：四边形$EFGH$ 是平行四边形。

证明：由中点条件证 $\triangle AEH \cong \triangle CGF$（SAS）→ $EH = GF$；同理 $EF = GH$。
∴ 四边形$EFGH$ 是平行四边形。

3. 如图，$\parallelogram ABCD$ 中，$AF = CH$，$DE = BG$。求证：$EG$ 与 $HF$ 互相平分。

证明：连结 $EF$、$FG$、$GH$、$HE$（如图虚线）。证 $\triangle AEF \cong \triangle CGH$ → $EF = GH$；同理 $EH = FG$。
∴ 四边形$EFGH$ 是平行四边形 → $EG$ 与 $HF$ 互相平分。

课堂小结
  • 综合运用四种判定方法解决证明问题。
  • 根据条件灵活选择最简捷的判定路径。
  • "构造中间平行四边形"是转化问题的有效策略。
