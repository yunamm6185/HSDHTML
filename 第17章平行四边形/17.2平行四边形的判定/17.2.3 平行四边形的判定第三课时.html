一、判定方法5：两组对角分别相等

探究活动

猜想：如果四边形的两组对角分别相等，它是平行四边形吗？

已知：在四边形ABCD中，∠A=∠C，∠B=∠D

求证：四边形ABCD是平行四边形

证明：

∵ 四边形内角和为360° → ∠A+∠B+∠C+∠D = 360°

又 ∠A=∠C，∠B=∠D → 2∠A + 2∠B = 360°

→ ∠A + ∠B = 180°

→ AD ∥ BC（同旁内角互补）……①

同理：∠B + ∠C = 180° → AB ∥ CD ……②

由①②得：两组对边分别平行 → $\square$ABCD ✅

判定定理5：两组对角分别相等的四边形是平行四边形。

∵ ∠A=∠C 且 ∠B=∠D ∴ 四边形ABCD是平行四边形

二、例题讲解

例1 综合运用多种判定方法

如图，E、F分别是▱ABCD的对角线AC上的两点，且AE=CF。求证：四边形BFDE是平行四边形。

证法一（对角线法）：

连接BD、EF交于O。∵ ▱ABCD → 对角线互相平分

→ O是AC的中点。又 AE=CF → E、F关于O对称

→ OE=OF 且 OB=OD → 四边形BFDE是$\square$ ✅

证法二（一边法）：证明DE∥BF且DE=BF

证法三（SSS法）：证明BE=DF且DE=BF

例2 角度条件的转化

在四边形ABCD中，∠A : ∠B : ∠C : ∠D = 3 : 2 : 3 : 2，求证：四边形ABCD是平行四边形。

解：设∠A=3x，则∠B=2x，∠C=3x，∠D=2x

内角和：3x+2x+3x+2x=360° → 10x=360° → x=36°

∴ ∠A=∠C=108°，∠B=∠D=72°

即两组对角分别相等 → 四边形ABCD是平行四边形 ✅

例3 实际应用——木工问题

木匠师傅要制作一个平行四边形的桌面，他测量了四个角，发现相对的两个角都是85°和95°。这个桌面合格吗？为什么？

答：不合格！因为四个角的度数应该是：
• 两组对角相等：两个85°，两个95°
• 但如果测量结果是"相邻两角分别为85°和95°"，那说明两组对角相等（85°=85°，95°=95°），应该是合格的。

⚠️ 关键要看相对的两个角是否相等！

三、课堂练习

练习1

下列哪个条件不能单独判定平行四边形？
A) 两组对角分别相等    B) 一组对角相等且一组邻角互补
C) 邻角都互补    D) 一个角的两边被对角线平分

B。"一组对角相等+一组邻角互补"不能保证另一组对角也相等。C选项等价于两组对边平行。

练习2

▱ABCD中，E是BC的中点，F是CD上一点，满足DF:FC=1:2。若EF交BD于G，求EG:GF。

提示：作辅助线或利用相似三角形。过F作FK∥CD交BG于K，或延长FE交AB延长线于H。
⚠️ 易错提醒
  • "两组对角相等"可以判定，但"一组对角相等"不行
  • "邻角互补"（所有邻角都补）等价于"两组对角分别相等"
  • 角度条件和边/对角线条件要配合使用才能高效解题

方法条件核心思路
