17.2 平行四边形的判定

第 4 课时：综合例题与判定提升
学习目标
  • 提高综合运用多种判定方法的能力。
  • 学会通过添加辅助线构造平行四边形。
  • 体会"性质"与"判定"的辩证统一。
知识铺垫

回顾：

定义法 | 两组对边相等法 | 一组对边平行且相等法 | 对角线互相平分法 | 两组对角分别相等法

例 5

例 5

如图1，四边形 $AEFD$ 和 $EBCF$ 都是平行四边形。
求证：四边形 $ABCD$ 是平行四边形。

证明：∵ 四边形 $AEFD$ 是平行四边形，

∴ $AD \parallel EF$，$AD = EF$。

又 ∵ 四边形 $EBCF$ 是平行四边形，

∴ $BC \parallel EF$，$BC = EF$。

∴ $AD \parallel BC$，$AD = BC$。

∴ 四边形 $ABCD$ 是平行四边形。（一组对边平行且相等的四边形是平行四边形）

例 6

例 6

如图2，$E$、$F$ 是 $\parallelogram ABCD$ 对角线 $AC$ 上的两点，且 $AE = CF$。$G$、$H$ 分别是 $AB$、$CD$ 的中点。
求证：四边形$EGFH$ 是平行四边形。

证明：如图3，连结 $GH$，交 $AC$ 于点 $O$。

∵ 四边形 $ABCD$ 是平行四边形，

∴ $AB = CD$，$OA = OC$。

又 ∵ $G$、$H$ 分别是 $AB$、$CD$ 的中点，

∴ $AG = CH$。

又 ∵ $AB \parallel CD$，

∴ $\angle GAO = \angle HCO$。

在 $\triangle GAO$ 和 $\triangle HCO$ 中，

$AG = CH$，$\angle GAO = \angle HCO$，$AO = CO$，

∴ $\triangle GAO \cong \triangle HCO$（SAS）。

∴ $GO = HO$。

又 ∵ $AE = CF$，

∴ $OE = OA - AE = OC - CF = OF$。

∴ 四边形 $EGFH$ 是平行四边形。（对角线互相平分的四边形是平行四边形）

巩固练习

1. 在四边形 $ABCD$ 中，$AB \parallel CD$，$\angle B = \angle D$。
求证：四边形$ABCD$ 是平行四边形。

证明：∵ $AB \parallel CD$（已知），

∴ $\angle A + \angle D = 180^\circ$（两直线平行，同旁内角互补）。

又 ∵ $\angle B = \angle D$（已知），

∴ $\angle A + \angle B = 180^\circ$（等量代换）。

∴ $AD \parallel BC$（同旁内角互补，两直线平行）。

∴ 四边形 $ABCD$ 是平行四边形。（两组对边分别平行的四边形是平行四边形）

2. 如图，四边形 $ABCD$ 是平行四边形，$AE$、$CF$ 分别与直线 $DB$ 相交于点 $E$ 和点 $F$，且 $AE \parallel CF$。分别连结点 $C$、$E$ 和点 $A$、$F$。
求证：四边形 $AFCE$ 是平行四边形。

证明：∵ 四边形 $ABCD$ 是平行四边形，

∴ $AD \parallel BC$，$AD = BC$（平行四边形的对边平行且相等）。

∴ $\angle ADB = \angle CBD$（两直线平行，内错角相等），

即 $\angle ADE = \angle CBF$。

又 ∵ $AE \parallel CF$，

∴ $\angle AED = \angle CFB$（两直线平行，内错角相等）。

在 $\triangle ADE$ 和 $\triangle CBF$ 中，

∵ $\angle ADE = \angle CBF$
    $\angle AED = \angle CFB$
    $AD = CB$，

∴ $\triangle ADE \cong \triangle CBF$（AAS）。

∴ $AE = CF$。

又 ∵ $AE \parallel CF$，

∴ 四边形 $AFCE$ 是平行四边形。（一组对边平行且相等的四边形是平行四边形）

3. 如图，$\parallelogram ABCD$ 的对角线 $AC$ 与 $BD$ 相交于点 $O$，直线 $EF$ 过点 $O$，且与 $AB$、$DC$ 分别相交于点 $E$ 和点 $F$，直线 $GH$ 过点 $O$ 且与 $AD$、$BC$ 分别相交于点 $G$ 和点 $H$。
求证：四边形 $GEHF$ 是平行四边形。

证明：∵ 四边形 $ABCD$ 是平行四边形，

∴ $OA = OC$，$OB = OD$（平行四边形的对角线互相平分）。

在 $\triangle AOE$ 和 $\triangle COF$ 中，

∵ $\angle EAO = \angle FCO$（$AB \parallel CD$，内错角相等），

　$AO = CO$，

　$\angle AOE = \angle COF$（对顶角相等），

∴ $\triangle AOE \cong \triangle COF$（ASA）。

∴ $OE = OF$。

同理，在 $\triangle AOG$ 和 $\triangle COH$ 中，

∵ $\angle GAO = \angle HCO$（$AD \parallel BC$，内错角相等），

　$AO = CO$，

　$\angle AOG = \angle COH$（对顶角相等），

∴ $\triangle AOG \cong \triangle COH$（ASA）。

∴ $OG = OH$。

∴ 四边形 $GEHF$ 的对角线互相平分。

∴ 四边形 $GEHF$ 是平行四边形。（对角线互相平分的四边形是平行四边形）

课堂小结
  • 综合运用多种判定方法解决复杂证明。
  • 构造辅助线（对角线、中点连线）是常用技巧。
  • "对角线交点"是证明"互相平分"的关键要素。
方法归纳

常用辅助线

1. 连结对角线 → 用"对角线互相平分"判定  
2. 构造中位线 → 借三角形关联  
3. 延长或过点作平行线 → 构造新的平行四边形
