一、平行四边形的对角线

观察与测量

如图，在 $\square ABCD$ 中，画出两条对角线AC和BD，它们相交于点O。

测量结果：OA = OC，OB = OD

猜想：平行四边形的对角线互相平分！

证明：为什么对角线互相平分？

已知：$\square ABCD$的对角线AC、BD交于点O

求证：OA = OC，OB = OD

证明：

∵ 四边形ABCD是平行四边形

∴ AB = CD（平行四边形对边相等），AB ∥ CD

∴ $\angle OAB = \angle OCD$，$\angle OBA = \angle ODC$（内错角相等）

又 $\angle AOB = \angle COD$（对顶角相等）

∴ $\triangle AOB \cong \triangle COD$（ASA）

∴ OA = OC，OB = OD（全等三角形对应边相等）

✅ 得证！

定理：平行四边形的对角线互相平分。

∵ $\square ABCD$的对角线AC、BD交于O ∴ OA = OC，OB = OD

📌 推论：平行四边形是中心对称图形

平行四边形的两条对角线的交点是它的对称中心。绕此点旋转180°后，图形与自身重合。

平行四边形是中心对称图形，对称中心为对角线交点。

💡 过对称中心的任意直线被平行四边形所截得的线段互相平分。

二、例题讲解

例1 求三角形周长（利用对角线性质）

已知 $\square ABCD$ 的两条对角线相交于点O，若 $AC + BD = 36\,\text{cm}$，$AD = 12\,\text{cm}$，求 $\triangle OCD$ 的周长。

解：∵ $\square ABCD$ 的对角线互相平分

∴ $OC = \frac{1}{2}AC$，$OD = \frac{1}{2}BD$

∴ $OC + OD = \frac{1}{2}(AC + BD) = \frac{1}{2} \times 36 = 18\,\text{cm}$

又 ∵ CD = AD = 12 cm（平行四边形对边相等）

∴ $\triangle OCD$ 的周长 = $OC + OD + CD = 18 + 12 =$ $30\,\text{cm}$

💡 关键思路：将两对角线之和转化为两个半对角线之和，再利用对边相等的性质求周长。

例2 对称性应用（过中心的直线）

如图，过 $\square ABCD$ 的对称中心O作直线l，分别交AD于E，交BC于F。若 $OE = 3\,\text{cm}$，$EF = 4\,\text{cm}$，求BF的长。

解：

∵ O 是 $\square ABCD$ 的对称中心（对角线交点），直线l过O分别交AD、BC于E、F两点

∴ 点E和点F关于O中心对称

∴ OE = OF = 3 cm（中心对称点的连线经过且被中心平分）

又 EF = OE + OF = 3 + 3 = 6 cm ✅（验证：已知EF=4有矛盾，实际应为EF=6cm）

由中心对称性可知：△AOE ≅ △COF（AAS或ASA可证）

∴ AE = CF，DE = BF

若设AE = x，则 DE = AD − x，∴ BF = DE = AD − x

💡 核心结论：过对称中心O的直线截平行四边形所得的两条线段互相平分，即 OE = OF。

⚠️ 注意：本题中若只知OE和EF，要求BF的具体值还需知道AD的长度。典型完整题型为：已知AD=10cm，OE=3cm，则EF=6cm，若E在AD上距A点4cm处，则BF=DE=6cm。

例3 综合题（对角线互相垂直的特殊情况）

在 $\square ABCD$ 中，对角线 $AC \perp BD$，垂足为O。若 $AC = 10\,\text{cm}$，$BD = 6\,\text{cm}$，求平行四边形的面积和周长。

解：∵ AC ⊥ BD，O是对角线交点

∴ $OA = OC = \frac{1}{2}AC = 5\,\text{cm}$，$OB = OD = \frac{1}{2}BD = 3\,\text{cm}$

∠AOB = 90°

在 Rt△AOB 中，由勾股定理：

$AB = \sqrt{OA^2 + OB^2} = \sqrt{5^2 + 3^2} = \sqrt{25 + 9} = \sqrt{34}\,\text{cm}$

面积计算：

$S_{\square ABCD} = 4 \times S_{\triangle AOB} = 4 \times \frac{1}{2} \times OA \times OB = 4 \times \frac{1}{2} \times 5 \times 3 =$ $30\,\text{cm}^2$

周长计算：

周长 $= 4 \times AB = 4\sqrt{34} \approx$ $23.32\,\text{cm}$

📌 拓展：对角线互相垂直的平行四边形实际上是菱形！菱形的面积公式为 $S = \frac{1}{2}d_1 d_2$（其中 $d_1,d_2$ 为对角线长），即 $S = \frac{1}{2} \times 10 \times 6 = 30\,\text{cm}^2$ ✓

三、课堂练习

练习1 ⭐⭐

在 $\square ABCD$ 中，对角线AC、BD交于O。若周长为40cm，两对角线分别为16cm和12cm，判断能否确定各边的具体长度？说明理由。

答：不能唯一确定各边长的具体数值。

分析过程：

① 设相邻两边为 a 和 b

∵ 周长 = 40 → $2(a+b) = 40$ → $a + b = 20$ ……①

② 对角线的一半：$OA = OC = 8$ cm，$OB = OD = 6$ cm

③ 在△AOB中，根据三角形不等式：

   $|OA-OB| < AB < OA+OB$

   $|8-6| < a < 8+6$ → $2 < a < 14$ ……②

④ 同理：$2 < b < 14$ ……③

⑤ 由①得：$b = 20 - a$，代入③得 $2 < 20-a < 14$ → $6 < a < 18$

结合②③：$\boldsymbol{6 < a < 14}$，$\boldsymbol{6 < b < 14}$ 且 $a+b=20$

结论：a可以是(6, 14)范围内的任意值（如a=10,b=10 或 a=11,b=9等），只要满足条件即可。因此各边长不能唯一确定，只能给出范围约束。

练习2 ⭐

平行四边形的两条对角线把平行四边形分成4个三角形，这4个三角形的面积有什么关系？

四个三角形的面积都相等！

证明思路如下：

① 对角线互相平分 → OA=OC，OB=OD

② ∠AOB = ∠COD（对顶角相等）→ △AOB ≅ △COD（SAS）

同理：△AOD ≅ △COB（SAS）

③ 又 S△AOB = S△AOD（同底OA，等高：B、D到AC距离相等，因AB∥CD）

∴ 四个三角形面积都相等，每个面积为 $S_{\square}/4$

练习3 ⭐⭐⭐

在 $\square ABCD$ 中，对角线AC、BD交于O，OE ⊥ AC交CD于E。若 $AD = 10\,\text{cm}$，$OE = 6\,\text{cm}$，求 $S_{\square ABCD}$。

解法一（利用三角形面积关系）：

连接CE。∵ OE ⊥ AC

S△OCE = $\frac{1}{2}$ × OC × OE

注意到 S△ACD = $\frac{1}{2}$ × S□ABCD（对角线将平行四边形分成两个面积相等的三角形）

S△OCE 与 S△ACD 的关系取决于E的位置...

更优解法：

考虑以 AC 为底，平行四边形的高为 h，则 S = AC × h

OE 是点E到AC的距离，而 E 在CD上

实际上，S□ABCD = 2 × AC × OE'（其中OE'是从D到AC的高）

由于题目条件不足以直接求解（需补充AC的长度或其他信息），这是一个需要更多条件的开放型问题。

💡 提示：如果补充条件 "AC = 8cm"，则 S□ABCD = 2 × AC × (D到AC的距离)，可通过相似等方法求解。建议此类题目明确所有已知量。

⚠️ 易错点提醒
  • "对角线互相平分" ≠ "对角线相等" —— 只有矩形才对角线相等
  • "对角线互相平分" ≠ "对角线垂直" —— 只有菱形才对角线垂直
  • 平行四边形的对角线将其分成4个面积相等的三角形
  • 求面积时常用：$S = \frac{1}{2} d_1 d_2$ 仅适用于菱形！普通平行四边形用底×高
