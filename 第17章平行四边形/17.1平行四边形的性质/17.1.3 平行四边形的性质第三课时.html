17.1 平行四边形的性质

第 3 课时：对角线互相平分的性质

学习目标

1掌握平行四边形对角线互相平分的性质
2能利用对角线性质进行简单的计算与证明
3体会"转化"思想在几何证明中的应用

课题导入

前面我们学习了平行四边形的对边相等、对角相等。如果画出平行四边形的两条对角线，它们相交于一点，这个交点把对角线分成的四条线段之间有什么关系呢？

复习回顾

我们已经知道平行四边形的以下性质：

性质类别具体内容
对边关系对边平行且相等：$AB = CD$，$AD = BC$
对角关系对角相等，邻角互补：$\angle A = \angle C$，$\angle B = \angle D$
对角线？
一、平行四边形的对角线

探索

在第79页对于图 17.1.3 的探索过程中，你观察到 $OA$ 与 $OC$、$OB$ 与 $OD$ 各有什么关系？

任意画几个平行四边形，量量看，是否都是这样。

我们已经发现，$\parallelogram ABCD$ 是一个中心对称图形，对角线的交点 $O$ 就是对称中心，有

$$OA = OC,\quad OB = OD.$$

由此可得：

📌 性质定理 3

平行四边形的对角线互相平分。

我们可以用演绎推理证明这个结论。

二、证明

已知：如图 17.1.11，$\parallelogram ABCD$ 的对角线 $AC$ 与 $BD$ 相交于点 $O$。

求证：$OA = OC$，$OB = OD$。

观察图形，$OA$ 与 $OC$ 以及 $OB$ 与 $OD$ 分别属于哪两个三角形？

分析

要证明的相等的 $OA$ 与 $OC$ 以及 $OB$ 与 $OD$ 分别属于 $\triangle AOB$ 与 $\triangle COD$，只需证明这两个三角形全等。

你能写出证明的完整过程吗？

证明

$\because$ 四边形 $ABCD$ 是平行四边形，

$\therefore AB = CD$（平行四边形的对边相等），

$\quad AB \parallel CD$。

$\therefore \angle OAB = \angle OCD$，$\angle OBA = \angle ODC$（两直线平行，内错角相等）。

又 $\because \angle AOB = \angle COD$（对顶角相等），

$\therefore \triangle AOB \cong \triangle COD$（A.S.A.）。

$\therefore OA = OC$，$OB = OD$。

三、例题

▶ 例 5

如图 17.1.12，$\parallelogram ABCD$ 的对角线 $AC$ 与 $BD$ 相交于点 $O$，$\triangle AOB$ 的周长为 15，$AB = 6$，那么对角线 $AC$ 与 $BD$ 的和是多少？

解　在 $\parallelogram ABCD$ 中，

$\because AB = 6$，$AO + BO + AB = 15$，

$\therefore AO + BO = 15 - 6 = 9$。

又 $\because AO = OC$ 且 $BO = OD$（平行四边形的对角线互相平分），

$\therefore AC + BD = 2AO + 2BO = 2(AO + BO) = 2 \times 9 = \mathbf{18}$。

关键：利用"对角线互相平分"，将 $AC + BD$ 转化为 $2(AO+BO)$。

▶ 例 6

如图 17.1.13，$\parallelogram ABCD$ 的对角线 $AC$ 与 $BD$ 相交于点 $O$，$EF$ 过点 $O$ 且与边 $AB$、$CD$ 分别相交于点 $E$ 和点 $F$。求证：$OE = OF$。

观察图形，$OE$ 与 $OF$ 分别属于哪两个三角形？

证明　$\because$ 四边形 $ABCD$ 是平行四边形，

$\therefore OB = OD$（平行四边形的对角线互相平分）。

又 $\because AB \parallel DC$，

$\therefore \angle EBO = \angle FDO$（两直线平行，内错角相等）。

又 $\because \angle BOE = \angle DOF$（对顶角相等），

$\therefore \triangle BEO \cong \triangle DFO$（A.S.A.）。

$\therefore OE = OF$。

核心方法：证明线段相等 $\rightarrow$ 证三角形全等。利用对角线平分得 $OB=OD$，再利用平行得内错角相等。

  • 求和转化：计算对角线和时，常用AC + BD = 2(AO + BO)

  • 第二步：证明这两个三角形全等
    • S.A.S. A.S.A. A.A.S. S.S.S.
🔑 核心方法
03
平行四边形中的"桥梁"
  • ∥ 对边平行 → 内错角相等
  • ◇ 对角线互相平分 → OA=OC，OB=OD
  • ＝ 对边相等 → AB=CD，AD=BC
    • 🌉 转化桥梁
    四、练习

    1.

    如图，$\parallelogram ABCD$ 的对角线 $AC$ 与 $BD$ 相交于点 $O$，指出图中各对相等的线段。

    相等的线段有：

    $AB = CD$，$AD = BC$（平行四边形对边相等）

    $OA = OC$，$OB = OD$（平行四边形对角线互相平分）

    2.

    如图，在 $\parallelogram ABCD$ 中，$O$ 是对角线 $AC$ 与 $BD$ 的交点，$BE \perp AC$，$DF \perp AC$，垂足分别为点 $E$ 和点 $F$。求证：$OE = OF$。

    证明　$\because$ 四边形 $ABCD$ 是平行四边形，

    $\therefore OB = OD$（平行四边形的对角线互相平分）。

    $\because BE \perp AC$，$DF \perp AC$，

    $\therefore \angle BEO = \angle DFO = 90°$。

    又 $\because \angle BOE = \angle DOF$（对顶角相等），

    $\therefore \triangle BEO \cong \triangle DFO$（A.A.S.）。

    $\therefore OE = OF$。

    3.

    如图，在 $\parallelogram ABCD$ 中，$EF$ 过对角线的交点 $O$，且与边 $AB$、$CD$ 分别相交于点 $E$ 和点 $F$。$AB = 4$，$AD = 3$，$OF = 1.3$。求四边形 $BCFE$ 的周长。

    解　$\because$ 四边形 $ABCD$ 是平行四边形，

    $\therefore AB = CD = 4$，$AD = BC = 3$，$OB = OD$（平行四边形的对角线互相平分）。

    又 $\because AB \parallel CD$，

    $\therefore \angle EBO = \angle FDO$（内错角相等）。

    又 $\because \angle BOE = \angle DOF$（对顶角相等），

    $\therefore \triangle BEO \cong \triangle DFO$（A.S.A.）。

    $\therefore OE = OF = 1.3$，$BE = DF$。

    $\therefore EF = OE + OF = 1.3 + 1.3 = 2.6$。

    四边形 $BCFE$ 的周长 $= BC + CF + FE + EB$

    $\quad = BC + (CD - DF) + FE + BE$

    $\quad = BC + CD + FE$（$\because BE = DF$，抵消）

    $\quad = 3 + 4 + 2.6 = \mathbf{9.6}$。

    技巧：利用 $\triangle BEO \cong \triangle DFO$ 得到 $BE = DF$，将四边形 $BCFE$ 的周长转化为 $BC + CD + FE$。

    易错警示
    • "对角线互相平分"是指对角线交点将每条对角线分成相等的两段，不是指两条对角线长度相等。
    • 只有矩形（后面会学）的对角线才相等，一般平行四边形的对角线不相等。
    • 证明线段相等的常用方法：证明包含这两条线段的三角形全等。

    课堂小结

    本课核心：平行四边形的对角线互相平分。

    性质结论
    对边平行且相等
    对角相等（邻角互补）
    对角线互相平分（$OA = OC$，$OB = OD$）
    对称性中心对称图形（对称中心为对角线交点）

    证明策略：要证线段相等 $\rightarrow$ 找包含这两条线段的三角形 $\rightarrow$ 证三角形全等。

    数学核心素养

    直观想象

    通过观察图形，发现对角线交点平分对角线的规律。

    逻辑推理

    利用三角形全等，演绎证明对角线互相平分。

    数学抽象

    将"对角线互相平分"抽象为 $OA=OC$、$OB=OD$ 的符号表达。
