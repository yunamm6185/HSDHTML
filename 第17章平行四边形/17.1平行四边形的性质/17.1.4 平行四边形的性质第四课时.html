17.1 平行四边形的性质

第 4 课时：性质的应用与面积问题

学习目标

1 综合运用平行四边形的性质解决线段、周长计算问题。
2 掌握平行四边形面积的计算方法及其在几何计算中的应用。
3 体会转化思想和等积法在解决几何问题中的作用。
知识铺垫

1. 平行四边形的边、角、对角线分别有哪些性质？

2. 平行四边形的周长公式：$C = 2(a + b)$（其中 $a, b$ 为邻边长）。

3. 平行四边形的面积公式：$S = \text{底} \times \text{高}$。

例题精讲

例 7

如图 17.1.14，$\parallelogram ABCD$ 的对角线 $AC$ 与 $BD$ 相交于点 $O$，其周长为 16，且 $\triangle AOB$ 的周长比 $\triangle BOC$ 的周长小 2。求边 $AB$ 和 $BC$ 的长。

分析：

1. 由平行四边形对角线互相平分可知 $OA = OC$。

2. $\triangle AOB$ 周长 $= AB + OA + OB$，$\triangle BOC$ 周长 $= BC + OB + OC$。

3. 两周长之差实际上就是邻边 $AB$ 与 $BC$ 之差。

4. 结合周长公式列方程组求解。

解： $\because$ 四边形 $ABCD$ 是平行四边形，

$\therefore OA = OC$（平行四边形的对角线互相平分）。

$\because \triangle AOB$ 的周长 $+ 2 = \triangle BOC$ 的周长，

$\therefore AB + OA + OB + 2 = BC + OB + OC$，

即 $AB + 2 = BC$。

又 $\because \parallelogram ABCD$ 的周长等于 16，

$\therefore 2(AB + BC) = 16$，

即 $4AB + 4 = 16$。

$\therefore AB = 3$，$BC = 5$。

想一想

在例 7 中，若已知的是 $\triangle AOB$ 与 $\triangle BOC$ 的周长之和，你能求出什么？

周长之和 $= (AB+OA+OB) + (BC+OB+OC) = (AB+BC) + 2OB + (OA+OC) = \frac{1}{2}C + BD + AC$。可以求出对角线之和与周长的一半之和。

例 8

如图 17.1.15，在 $\parallelogram ABCD$ 中，对角线 $AC = 21\text{ cm}$，$BE \perp AC$，垂足为点 $E$，且 $BE = 5\text{ cm}$，$AD = 7\text{ cm}$。求 $AD$ 与 $BC$ 之间的距离。

解： 设 $AD$ 与 $BC$ 之间的距离为 $x$，则 $\parallelogram ABCD$ 的面积等于 $AD \cdot x$。

$\because S_{\parallelogram ABCD} = 2S_{\triangle ABC} = AC \cdot BE$，

$\therefore AD \cdot x = AC \cdot BE$，

即 $7x = 21 \times 5$。

$\therefore x = 15\text{ (cm)}$。

即 $AD$ 与 $BC$ 之间的距离为 $15\text{ cm}$。

你知道其中的理由吗？

为什么 $S_{\parallelogram ABCD} = AC \cdot BE$？

$\triangle ABC$ 的面积为 $\frac{1}{2} \times AC \times BE$。而对角线 $AC$ 将平行四边形分成两个面积相等的三角形，所以平行四边形面积是 $\triangle ABC$ 面积的 2 倍。

方法归纳

1. 周长问题： 利用对角线互相平分，将三角形周长之差转化为邻边之差。

2. 面积问题： 常用“等积法”。同一个平行四边形的面积可以用不同的底乘以对应的高来表示，即 $S = a \cdot h_a = b \cdot h_b$。

3. 辅助线： 遇到面积问题，常连接对角线转化为三角形面积。

巩固练习

练习 1. $\parallelogram ABCD$ 的两条对角线 $AC$ 与 $BD$ 相交于点 $O$，$AB = 8\text{ cm}$，$BC = 6\text{ cm}$，$\triangle AOB$ 的周长是 $18\text{ cm}$。求 $\triangle AOD$ 的周长。

解： $\because$ 四边形 $ABCD$ 是平行四边形，$\therefore AD = BC = 6\text{ cm}$，$OA = OC$，$OB = OD$。

$\because \triangle AOB$ 的周长 $= AB + OA + OB = 18\text{ cm}$，$\therefore OA + OB = 18 - 8 = 10\text{ (cm)}$。

$\therefore \triangle AOD$ 的周长 $= OA + OD + AD = (OA + OB) + AD = 10 + 6 = 16\text{ (cm)}$。

练习 2. 如图，如果 $\triangle AOB$ 与 $\triangle AOD$ 的周长之差为 8，而 $AB : AD = 3 : 2$，那么 $\parallelogram ABCD$ 的周长为多少？

解： $\because \triangle AOB$ 周长 $- \triangle AOD$ 周长 $= (AB+OA+OB) - (AD+OA+OD) = AB - AD = 8$。

$\because AB : AD = 3 : 2$，设 $AB = 3k, AD = 2k$，则 $3k - 2k = 8 \Rightarrow k = 8$。

$\therefore AB = 24, AD = 16$。$\parallelogram ABCD$ 的周长 $= 2(AB + AD) = 2(24 + 16) = 80$。

练习 3. 在 $\parallelogram ABCD$ 中，两条对角线 $AC$ 与 $BD$ 相交于点 $O$，$BC = 5$，$AC = 6$，$BD = 8$。求 $\triangle AOB$ 的周长。

解题思路：

1. 找线段： $\triangle AOB$ 的周长 $= OA + OB + AB$。我们需要分别求出这三条线段的长度。

2. 用性质： 利用“平行四边形的对角线互相平分”，可以求出 $OA$ 和 $OB$。

3. 算边长： 在 $\triangle BOC$ 中，发现 $OC=3, OB=4, BC=5$。由于 $3^2 + 4^2 = 5^2$，根据勾股定理逆定理，可知 $\angle BOC = 90^\circ$。这意味着 $\triangle AOB$ 也是直角三角形，从而求出 $AB$。

规范解答：

$\because$ 四边形 $ABCD$ 是平行四边形，

$\therefore OA = \frac{1}{2}AC = 3$，$OB = \frac{1}{2}BD = 4$。

在 $\triangle BOC$ 中，$\because OC^2 + OB^2 = 3^2 + 4^2 = 25 = BC^2$，

$\therefore \angle BOC = 90^\circ$（勾股定理的逆定理）。

$\therefore \angle AOB = 90^\circ$。在 $\text{Rt}\triangle AOB$ 中，$AB = \sqrt{OA^2 + OB^2} = 5$。

$\therefore \triangle AOB$ 的周长 $= OA + OB + AB = 3 + 4 + 5 = 12$。

课堂小结

  • 掌握了利用对角线互相平分性质解决三角形周长差的问题。
  • 学会了利用面积法（等积法）求平行四边形的高或底。
  • 强化了方程思想在几何计算中的应用。
读一读

数学史话： 在古代，人们在测量土地面积时，最早发现矩形的面积可以用长乘宽来计算。对于平行四边形，古人通过“割补法”将其转化为矩形，从而推导出面积公式。这种“转化”的思想是几何学中极其重要的研究方法。
