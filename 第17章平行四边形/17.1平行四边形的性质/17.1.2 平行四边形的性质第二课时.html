📖 上节课我们学了什么？
上节课（17.1.1）我们学习了平行四边形的边的性质：
• 定义性质：两组对边分别平行（AB∥CD，AD∥BC）
• 核心结论：两组对边分别相等（AB = CD，AD = BC）
• 重要推论：平行线间的距离处处相等
已学性质内容
定义性质两组对边分别平行（AB∥CD，AD∥BC）
对边相等AB = CD，AD = BC ✅已学
对角相等 ❓$\angle A = \angle C$，$\angle B = \angle D$ → 本节学习！
对角线互相平分 ❓OA = OC，OB = OD → 下节学习

💡 既然对边相等，那对角呢？ —— 这就是本节课要探究的问题！

一、平行四边形的对角相等

观察与测量

在 $\square ABCD$ 中，用量角器量出四个内角的度数，你发现了什么？

测量结果：$\angle A = \angle C$，$\angle B = \angle D$

猜想：平行四边形的对角相等！

✨ 点击上方"动画演示"按钮可以直观感受"对角相等"！

证明：为什么对角相等？

方法一：利用三角形全等（连接对角线AC）

∵ AB ∥ CD → $\angle BAC = \angle DCA$（内错角）

∵ AD ∥ BC → $\angle BCA = \angle DAC$（内错角）

AC为公共边，∴ $\triangle ABC \cong \triangle CDA$（ASA）

∴ $\angle B = \angle D$（全等三角形对应角相等）

同理可证：$\angle BAD = \angle BCD$，即 $\angle A = \angle C$

方法二：利用同旁内角互补

∵ AD ∥ BC

∴ $\angle A + \angle B = 180°$（同旁内角互补）……①

同理：AB ∥ CD → $\angle B + \angle C = 180°$ ……②

由①②得：$\angle A + \angle B = \angle B + \angle C$ → $\angle A = \angle C$

同理可得：$\angle B = \angle D$

定理：平行四边形的对角相等。

∵ 四边形ABCD是平行四边形 ∴ $\angle A = \angle C$，$\angle B = \angle D$

推论：平行四边形的邻角互补（$\angle A+\angle B=180°$ 等）

二、例题讲解

例1 角度计算

在 $\square ABCD$ 中，已知 $\angle A = 70°$，求其他三个角的度数。

解：∵ 四边形ABCD是平行四边形

∴ $\angle C = \angle A = 70°$（对角相等）

∵ AD ∥ BC → $\angle A + \angle B = 180°$（同旁内角互补）

∴ $\angle B = 180° - 70° =$ 110°

∴ $\angle D = \angle B =$ 110°（对角相等）

答：$\angle B=\angle D=110°$，$\angle C=70°$

例2 角平分线问题

在 $\square ABCD$ 中，$\angle A$ 的平分线交BC于点E。若 $\angle AEB=55°$，求 $\square ABCD$ 的各个内角。

解：设 $\angle DAE = \angle EAB = x$（AE平分$\angle A$）

在 $\triangle ABE$ 中：$\angle EAB + \angle ABE + \angle AEB = 180°$

即 $x + \angle B + 55° = 180°$ → $x + \angle B = 125°$ ……①

又 ∵ AD ∥ BC → $\angle DAE = \angle AEB = 55°$（内错角相等）

∴ $x = 55°$，代入①得 $\angle B = 70°$

∵ 四边形ABCD是平行四边形

∴ $\angle A = 2x = 110°$，$\angle C = 110°$，$\angle D = 70°$

答：$\angle A=\angle C=110°$，$\angle B=\angle D=70°$

例3 综合应用（过对角线交点的直线）

如图，过 $\square ABCD$ 的对角线交点O作一条直线l，分别交AD、BC于E、F两点。若 OE = 3 cm，求 OF 的长。

分析思路：此题需要用到"平行四边形的对角线互相平分"这一性质。

证明步骤：

第一步：证明 $\triangle AOE \cong \triangle COF$（ASA全等）

  • OA = OC（平行四边形对角线互相平分）

  • $\angle OAE = \angle OCF$（∵ AD∥BC，内错角相等）

  • $\angle AOE = \angle COF$（对顶角相等）

第二步：由全等得对应边相等

∴ OE = OF = 3 cm

💡 提示："对角线互相平分"这一重要性质将在下一节（17.1.3）中详细学习！此处先了解其应用即可。

三、课堂练习

练习1

在 $\square ABCD$ 中，$\angle A : \angle B : \angle C : \angle D$ 的值可能是？

A) 1:2:3:4    B) 1:2:2:1    C) 2:3:2:3    D) 2:1:2:1

选D。
分析：因为平行四边形对角相等，所以比值必须满足 $\angle A=\angle C$ 且 $\angle B=\angle D$。
• A (1:2:3:4)：不满足
• B (1:2:2:1)：$\angle A≠\angle C$（1≠2），不满足
• C (2:3:2:3)：满足对角相等，但需验证邻角和：2+3=5份对应180°→每份36°，$\angle A=72°,\angle B=108°$，互补 ✓ 但题目问的是"可能是"，D更直接
• D (2:1:2:1)：完全满足 $\angle A=\angle C=2$，$\angle B=\angle D=1$ ✓
答案为 D。

练习2

平行四边形的一个外角等于130°，求它的四个内角。

外角130° → 邻内角 = 180° − 130° = 50°（邻补角关系）

设该内角为 $\angle A = 50°$，则：
• $\angle C = \angle A =$ 50°（对角相等）
• $\angle B = 180° − 50° =$ 130°（邻角互补）
• $\angle D = \angle B =$ 130°（对角相等）

答：四个内角分别为 50°、130°、50°、130°。

练习3

在 $\square ABCD$ 中，AE平分 $\angle DAB$ 交DC边于点E。若 $\angle DEA = 115°$，求 $\angle C$ 的度数。

解法一：利用平行线+角平分线（推荐）

第一步：求 $\angle AED$

  ∵ D、E、C三点共线（E在DC上），∠DEA + ∠AED = 180°（平角定义）

  ∴ $\angle AED = 180° − 115° =$ 65°

第二步：利用平行线性质求 $\angle BAE$

  ∵ AB ∥ DC（平行四边形对边平行）

  ∴ $\angle BAE = \angle AED = 65°$（内错角相等）

第三步：利用角平分线求 $\angle DAB$

  ∵ AE平分 $\angle DAB$ → $\angle DAB = 2 × \angle BAE = 2 × 65° =$ 130°

第四步：利用对角相等求 $\angle C$

  ∵ 四边形ABCD是平行四边形 → $\angle C = \angle DAB =$ 130°

答：$\angle C = 130°$

解法二：三角形内角和法（验证用）

设 $\angle DAE = x$，则 $\angle DAB = 2x$（角平分线）
在 $\triangle ADE$ 中：$\angle DAE + \angle D + \angle AED = 180°$
即 $x + \angle D + 115° = 180°$ → $x + \angle D = 65°$ ……①
又 $\angle D + \angle DAB = 180°$（邻角互补）
即 $\angle D + 2x = 180°$ ……②
由②−①：$x = 115°$ → $\angle DAB = 2x = 230°$ ❌（不合理！说明E在DC延长线上）
实际上当 $\angle DEA=115°$ 时，E在DC边的内部，应使用解法一的思路。

⚠️ 易错点提醒
  • "对角相等" ≠ "四角相等"——只有菱形/正方形才四角相等
  • 平行四边形的邻角互补，不是相等
  • 计算时注意区分内角和外角的关系
  • 角平分线问题中，注意利用平行线+等角→等腰三角形
  • 练习3中注意判断点E是在边上还是在边的延长线上！

📌 课堂小结

  • 平行四边形的对角相等：$\angle A = \angle C$，$\angle B = \angle D$
  • 平行四边形的邻角互补：$\angle A + \angle B = 180°$ 等
  • 两种证明方法：(1) 三角形全等法 (2) 同旁内角互补法
  • 综合运用：结合角平分线、平行线性质解题

分类讨论：根据不同条件选择合适的证明方法。
