生活中的平行四边形

一、平行四边形的概念

观察与思考

下面这些图形有什么共同特点？它们与一般的四边形有什么不同？

类型特征描述
一般四边形四条边围成的封闭图形（没有特殊条件）
平行四边形两组对边分别平行的四边形

🔍 观察发现：平行四边形的两组对边是互相平行的。这是它区别于其他四边形的本质特征。

你可以拿两把直尺交叉摆放，就能看到一组平行线被另一组平行线所截形成的图形！

📌 平行四边形的定义

定义：两组对边分别平行的四边形叫做平行四边形（parallelogram）。

如图：在四边形ABCD中，如果 $AB \parallel CD$ 且 $AD \parallel BC$，那么四边形ABCD就是平行四边形。

记法：平行四边形ABCD 记作 "$\square ABCD$" 或 "平行四边形ABCD"。

⚠️ 注意：平行四边形的定义既是判定方法（满足此条件→是平行四边形），也是基本性质（是平行四边形→具有此特征）——平行四边形的两组对边分别平行。

二、平行四边形的性质——对边相等

🔬 动手探究：对边的数量关系

问题：平行四边形的对边除了平行之外，还有什么数量关系？

操作：在 $\square ABCD$ 中，连接对角线AC。观察 $\triangle ABC$ 和 $\triangle CDA$ 的关系。

✅ 完整证明过程：

∵ 四边形ABCD是平行四边形

∴ AB ∥ CD（平行四边形对边平行）

∴ $\angle BAC = \angle DCA$（两直线平行，内错角相等）……①

同理：AD ∥ BC → $\angle BCA = \angle DAC$ ……②

又 AC = CA（公共边）……③

由①②③得：$\triangle ABC \cong \triangle CDA$（ASA）

∴ AB = CD，AD = BC（全等三角形对应边相等）

💡 关键思路：通过连接对角线，将四边形问题转化为三角形全等问题来解决！这就是转化思想的应用。

⭐ 定理：平行四边形的对边相等。

几何语言表达：

∵ 四边形ABCD是平行四边形
∴ AB = CD，AD = BC

📌 重要推论：平行线间的距离

定义：两条平行线中，一条直线上任意一点到另一条直线的距离，叫做这两条平行线间的距离。

结论：平行线间的距离处处相等。

（因为平行四边形的对边之间的垂线段都相等）

三、例题讲解

例1 求周长（基础计算）

如图，在 $\square ABCD$ 中，已知 $AB = 5\,\text{cm}$，$BC = 3\,\text{cm}$，求 $\square ABCD$ 的周长。

解：

∵ 四边形ABCD是平行四边形

∴ AB = CD，AD = BC（平行四边形的对边相等）

∵ $AB = 5\,\text{cm}$，$BC = 3\,\text{cm}$

∴ 周长 = AB + BC + CD + AD = 5 + 3 + 5 + 3 = $16\,\text{cm}$

💡 快捷公式：$C_{\square} = 2(AB + BC)$

例2 角度与比例（综合应用）

如图，在 $\square ABCD$ 中，$\angle A : \angle B = 2 : 1$，求各内角的度数。

解：设 $\angle B = x$，则 $\angle A = 2x$

∵ AD ∥ BC（平行四边形对边平行）

∴ $\angle A + \angle B = 180°$（同旁内角互补）

∴ $2x + x = 180°$ → $3x = 180°$ → $x = 60°$

∴ $\angle A = 2x =$ $120°$，$\angle B =$ $60°$

∵ AB ∥ CD，AD ∥ BC

∴ $\angle C = \angle A =$ $120°$，$\angle D = \angle B =$ $60°$

答：$\angle A=\angle C=120°$，$\angle B=\angle D=60°$

例3 实际应用（木条围框架）

情境：用两根长分别为27cm和45cm的木条作为一组对边，另两根同样长的木条作为另一组对边，能否围成一个平行四边形？如果能，求它的周长。

解：能围成一个平行四边形。

设一组对边为27cm，则另一组对边为45cm

根据平行四边形的性质：两组对边分别相等 ✓

∴ 周长 = $2 \times (27 + 45) = 2 \times 72 =$ $144\,\text{cm}$

⚠️ 思考延伸：任意四根木条一定能围成平行四边形吗？
❌ 不一定！必须满足"两组对边分别相等"的条件才能围成平行四边形。

四、课堂练习

练习1 ⭐

在 $\square ABCD$ 中，$AB + BC = 12\,\text{cm}$，求周长。

周长 $= 2(AB+BC) = 2 \times 12 =$ $24\,\text{cm}$
（利用对边相等性质，周长等于相邻两边之和的两倍）

练习2 ⭐⭐

在 $\square ABCD$ 中，$AB - BC = 4\,\text{cm}$，周长为28cm，求AB和BC的长。

设 $BC = x$ cm，则 $AB = (x+4)$ cm
周长方程：$2(x+4+x) = 28$
→ $4x + 8 = 28$ → $4x = 20$ → $x = 5$
∴ $BC = 5\,\text{cm}$，$AB = 9\,\text{cm}$

练习3 ⭐⭐

平行四边形的周长为56cm，相邻两边之差为8cm，求各边的长度。

设较短边为 $x$ cm，则较长边为 $(x+8)$ cm
方程：$2[x+(x+8)] = 56$ → $2(2x+8)=56$ → $4x+16=56$
→ $4x = 40$ → $x = 10$
∴ 各边长：$10\,\text{cm},\; 18\,\text{cm},\; 10\,\text{cm},\; 18\,\text{cm}$
⚠️ 易错点提醒
  • 不要混淆"对边"和"邻边" —— 对边是相对的两条边（AB与CD），邻边是相邻的两条边（AB与BC）
  • 平行四边形的周长公式：$C = 2(a+b)$，其中 a、b 是相邻两边的长度
  • 证明时必须写全条件：要明确写出"平行四边形的对边平行"，不能直接使用未证明的结论
  • 注意符号规范：使用 $\square ABCD$ 表示平行四边形，不要写成其他形式

📌 课堂小结

  • 平行四边形定义：两组对边分别平行的四边形，记作 $\square ABCD$
  • 本课核心性质（重点！）：对边相等 —— AB = CD，AD = BC
  • 重要推论：平行线间的距离处处相等
  • 证明方法：连接对角线 → 利用ASA证三角形全等 → 得对应边相等

🧠 数学思想·核心素养

转化思想：将四边形问题转化为三角形问题（连接对角线是最常用的辅助线方法）。
数形结合：通过图形直观理解对边相等的性质。
逻辑推理：利用全等三角形证明边相等，培养严谨的几何推理能力。
