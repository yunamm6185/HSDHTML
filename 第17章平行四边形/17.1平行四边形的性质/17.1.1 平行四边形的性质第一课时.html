一、平行四边形的概念

回忆 平行四边形的定义

我们知道，有两组对边分别平行的四边形叫做平行四边形（parallelogram）。

平行四边形 $ABCD$ 可以记作▱$ABCD$。

你能从下图所示的图形中找出平行四边形吗？

图 17.1.1

答案： 显然，图②与图⑤是平行四边形。而图①的一组对边平行，另一组对边不平行，这个四边形称为梯形。

根据定义，平行四边形的一个主要性质是两组对边分别平行。由此，可知平行四边形的相邻两个内角互补。除此之外，它还有什么性质呢？

试一试 作一个平行四边形

作法：

  1. 任意作一条直线 $m$；
  2. 在直线 $m$ 上任取点 $A$，在直线 $m$ 外任取点 $B$，连结 $AB$；
  3. 过点 $B$ 作直线 $m$ 的平行线 $n$，在直线 $n$ 上任取点 $C$；
  4. 过点 $C$ 作直线 $AB$ 的平行线，交直线 $m$ 于点 $D$，四边形 $ABCD$ 即为平行四边形。
二、平行四边形的对称性

探索 平行四边形的中心对称性

在平行四边形 $ABCD$ 中，连结对角线 $AC$ 和 $BD$，它们相交于点 $O$。让我们通过互动操作，探索点 $O$ 与四个顶点之间的位置关系。

图 17.1.3

💡 提示：点击上方按钮，按步骤探索平行四边形的对称性。

我们发现：

点 $A$ 绕点 $O$ 旋转 $180^\circ$ 后与点 $C$ 重合，即 $OA = OC$；

点 $B$ 绕点 $O$ 旋转 $180^\circ$ 后与点 $D$ 重合，即 $OB = OD$；

整个平行四边形绕点 $O$ 旋转 $180^\circ$ 后与自身重合。

因此，平行四边形是中心对称图形，对称中心是两条对角线的交点 $O$。

$AB = CD,\quad AD = CB,$
$\angle A = \angle C,\quad \angle B = \angle D.$

证明 演绎推理

我们可以用演绎推理证明上述探索得到的结论。

已知：如图 17.1.4，▱$ABCD$。

求证：$AB = CD$，$AD = CB$，$\angle A = \angle C$，$\angle ABC = \angle CDA$。

图 17.1.4 / 图 17.1.5

证明（如图 17.1.5，连结 $BD$）：

$\because$ 四边形 $ABCD$ 是平行四边形，

$\therefore AB \parallel DC$ 且 $AD \parallel BC$（平行四边形的两组对边分别平行）。

$\therefore \angle ABD = \angle CDB$，$\angle ADB = \angle CBD$。

又 $\because BD = DB$，

$\therefore \triangle ABD \cong \triangle CDB$（ASA）。

$\therefore AB = CD$，$AD = CB$，$\angle A = \angle C$。

由 $\angle ABD = \angle CDB$ 和 $\angle ADB = \angle CBD$，得

$\angle ABD + \angle CBD = \angle CDB + \angle ADB$，

即 $\angle ABC = \angle CDA$。

分析：我们已经知道，证明边相等或角相等的一种重要方法是找出它们分别所属的三角形，然后证明这两个三角形全等。从上面旋转纸片的探索过程中，可以发现一条对角线恰好将平行四边形分成两个全等的三角形。

三、平行四边形的性质

定理1：平行四边形的对边相等

$\because$ 四边形 $ABCD$ 是平行四边形，$\therefore AB = CD$，$AD = BC$

定理2：平行四边形的对角相等

$\because$ 四边形 $ABCD$ 是平行四边形，$\therefore \angle A = \angle C$，$\angle B = \angle D$

四、例题讲解

▶ 例 1

如图 17.1.6，在▱$ABCD$ 中，$\angle A = 40^\circ$。求其他各内角的度数。

图 17.1.6

解 在▱$ABCD$ 中，

$\angle A = \angle C$ 且 $\angle B = \angle D$（平行四边形的对角相等）。

$\because \angle A = 40^\circ$，

$\therefore \angle C = 40^\circ$。

又 $\because AD \parallel BC$，

$\therefore \angle A + \angle B = 180^\circ$。

$\therefore \angle B = 180^\circ - \angle A = 180^\circ - 40^\circ = 140^\circ$。

$\therefore \angle D = \angle B = 140^\circ$。

💡 平行四边形的邻角互补。

▶ 例 2

如图 17.1.7，在▱$ABCD$ 中，$AB = 8$，周长等于 $24$。求其余三条边的长。

图 17.1.7

解 在▱$ABCD$ 中，有

$AB = DC$ 且 $AD = BC$（平行四边形的对边相等）。

$\because AB = 8$，

$\therefore DC = 8$。

又 $\because AB + BC + DC + AD = 24$，

$\therefore AD = BC = \frac{1}{2}(24 - 2AB) = \frac{1}{2}(24-16) = 4$。

五、平行线之间的距离

试一试 平行线之间的距离

如图 17.1.8，在方格图上画两条互相平行的直线，在其中一条直线上任取若干点，过这些点作另一条直线的垂线，用刻度尺量出这两条平行线之间这些垂线段的长度。

图 17.1.8

🤔 想一想：你能发现什么结论？试用平行四边形的性质定理加以说明。

经过度量，我们发现这些垂线段的长度都相等。由此我们得到平行线的又一个性质：

平行线之间的距离处处相等

定义：两条直线平行，其中一条直线上的任一点到另一条直线的距离，叫做这两条平行线之间的距离。

六、练习

练习 1

在 ▱$ABCD$ 中，$\angle A = 120^\circ$，求其余各内角的度数。

$\angle C = 120^\circ$，$\angle B = \angle D = 60^\circ$。

练习 2 · 等面积三角形

如图，如果直线 $l_1 \parallel l_2$，那么 $\triangle ABC$ 和 $\triangle DBC$ 的面积相等吗？你能说出理由吗？还能画其他等面积的三角形吗？

因为 $l_1 \parallel l_2$，平行线间距离处处相等，$\triangle ABC$ 与 $\triangle DBC$ 同底 $BC$ 且高相等，故面积相等。在 $l_1$ 上任取一点 $P$（不与 $A,D$ 重合），$\triangle PBC$ 也与它们面积相等。

练习 3

用一根长度为36cm的铁丝围成一个平行四边形，各边的长度恰好都是3的整数倍，试找出所有满足条件的平行四边形，并分别求出各边的长。

设相邻两边分别为 $3a,3b$，$2(3a+3b)=36$ ⇒ $a+b=6$，正整数解：

(1,5)→3cm,15cm；(2,4)→6cm,12cm；(3,3)→9cm,9cm（菱形）。

📖 读一读

数学史 中国古代数学中的"出入相补"

我国魏晋时期的数学家刘徽在注《九章算术》时，创造性地提出了"出入相补原理"——即一个平面图形从一处移置他处，面积不变；又若把图形分割成若干块，那么各部分面积之和等于原来图形的面积。

利用这一原理，刘徽巧妙地证明了平行四边形可以"割补"成长方形：沿平行四边形的高剪下一个直角三角形，平移到另一侧，恰好拼成一个长方形。因为长方形面积等于"底×高"，所以平行四边形的面积也等于"底×高"。

这种"以盈补虚"的思想，体现了中国古代数学注重直观、算法化的独特风格，比西方同类方法早了一千多年！
