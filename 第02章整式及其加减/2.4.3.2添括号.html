2.4 整式的加减

第4课时 添括号

🎯 学习目标

✔理解添括号法则的推导过程
✔掌握添括号法则，能正确添加括号
✔能运用添括号法则简化计算
✔体会逆向思维在数学中的应用

📌 我们学习了去括号法则：

\(a + (b + c) = a + b + c\)  \(a - (b + c) = a - b - c\)

反过来，将等式左右两边对调，就得到添括号的规律：

\(a + b + c = a + (b + c)\)  \(a - b - c = a - (b + c)\)

括号前是“+”号
\(a + b + c\)
\(a + (b + c)\)
括到括号里的各项都不改变正负号
括号前是“-”号
\(a - b - c\)
\(a - (b + c)\)
括到括号里的各项都改变正负号
📌 添括号法则

所添括号前面是“+”号，括到括号里的各项都不改变正负号。

所添括号前面是“-”号，括到括号里的各项都改变正负号。

例如：\(a + b - c = a + (b - c)\)； \(a - b + c = a - (b - c)\)。

📝 做一做

在括号内填入适当的项：

1. \(x^2 - x + 1 = x^2 - ( \quad )\)；
2. \(2x^2 - 3x - 1 = 2x^2 + ( \quad )\)；
3. \((a - b) - (c - d) = a - ( \quad )\)。
1. \(x - 1\)（因为 \(x^2 - x + 1 = x^2 - (x - 1)\)）
2. \(-3x - 1\)（\(2x^2 - 3x - 1 = 2x^2 + (-3x - 1)\)）
3. \(b + c - d\)（\((a - b) - (c - d) = a - b - c + d = a - (b + c - d)\)）

✏️ 例题

例8 计算：

(1) \(214a + 47a + 53a\)； (2) \(214a - 39a - 61a\)。

(1) \(214a + 47a + 53a = 214a + (47a + 53a) = 214a + 100a = 314a\)
(2) \(214a - 39a - 61a = 214a - (39a + 61a) = 214a - 100a = 114a\)

📌 注意：添括号与去括号的过程正好相反，添括号是否正确，不妨通过去括号检验一下。

练习

1. 计算：
(1) \(117x + 138x - 38x\)；
(2) \(125x - 64x - 36x\)；
(3) \(136x - 87x + 57x\)。
(1) \(117x + (138x - 38x) = 117x + 100x = 217x\)
(2) \(125x - (64x + 36x) = 125x - 100x = 25x\)
(3) \(136x - 87x + 57x = 136x - (87x - 57x) = 136x - 30x = 106x\)
2. 在下列各式的括号内填入适当的项：
(1) \(3x^2 - 2xy^2 + 2y^2 = 3x^2 - (\)__________\()\)；
(2) \(3x^2y^2 - 2x^3 + y^3 = 3x^2y^2 - (\)__________\()\)；
(3) \(-a^3 + 2a^2 - a + 1 = - (\)__________\()\)。
(1) \(2xy^2 - 2y^2\) （因为括号前是“-”，括号内各项变号：\(-2xy^2\) 变 \(2xy^2\)，\(+2y^2\) 变 \(-2y^2\)）
(2) \(2x^3 - y^3\) （\(-2x^3\) 变 \(2x^3\)，\(+y^3\) 变 \(-y^3\)）
(3) \(a^3 - 2a^2 + a - 1\) （每一项都变号）

📖 读一读 · 逆向思维——巧添括号，计算更轻松

▶

同学们，我们已经学会了去括号，现在来玩一个“逆向游戏”——添括号。想一想，如果给你一个没有括号的算式，比如 \(a + b + c\)，你想把它变成 \(a + (b + c)\)，这相当于给 \(b + c\) 穿上了“括号外套”。

关键问题是：这件外套前面应该放什么符号？如果放“+”，里面的 \(b\) 和 \(c\) 不需要变化；如果放“-”，里面的每一项都要变号！比如 \(a - b - c\) 要变成 \(a - (b + c)\)，就是因为括号前是“-”，所以 \(-b\) 和 \(-c\) 放进括号后要变成 \(+b\) 和 \(+c\)，括号里是 \(b + c\)。

我们总结一下：

  • 添“+”括号：直接加括号，里面的项什么都不变，就像穿透明外套。
  • 添“-”括号：加括号后，里面每一项都要变号（正变负，负变正），就像穿了变色外套。

看例8，\(214a + 47a + 53a\) 如果先算后两项，添上括号变成 \(214a + (47a + 53a)\)，一下子就得到 \(214a + 100a\)，多方便！添括号能帮我们“打包”计算，又快又准。你们学会了吗？

核心素养链接： 逆向思维 · 运算能力 · 转化思想

📌 课堂小结

▼
添括号法则：所添括号前是“+”号，括到括号里的各项不变号；所添括号前是“-”号，括到括号里的各项都变号。
添括号是去括号的逆过程，可以用去括号检验添括号是否正确。
适当添括号可以使计算简便（如合并同类项时先结合后计算）。
添括号体现了逆向思维和转化思想。
