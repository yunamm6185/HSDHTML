2.4 整式的加减

第1课时 同类项

🎯 学习目标

✔理解同类项的概念，能识别同类项
✔知道常数项都是同类项
✔能根据同类项的定义确定字母指数的取值
✔体会分类思想

📌 我们常常把具有相同特征的事物归为一类。在多项式的各个项中，也可以把具有某些相同特征的项归为一类。

例如，多项式 \(3x^2y - 4xy^2 - 3 + 5x^2y + 2xy^2 + 5\) 有6项，分别是：

\(3x^2y,\; -4xy^2,\; -3,\; 5x^2y,\; 2xy^2,\; 5\)。

在这6项中，可以把 \(3x^2y\) 与 \(5x^2y\) 归为一类， \(-4xy^2\) 与 \(2xy^2\) 归为一类， \(-3\) 与 \(5\) 归为一类。

这些被归为同一类的项有什么相同特征？

📌 同类项

同类项：所含字母相同，并且相同字母的指数都相等的项叫做同类项。

所有的常数项都是同类项。例如，上述多项式中的 \(-3\) 与 \(5\) 也是同类项。

例如：\(3x^2y\) 与 \(5x^2y\) 是同类项（字母都是 \(x,y\)，且 \(x\) 的指数都是2，\(y\) 的指数都是1）； \(-4xy^2\) 与 \(2xy^2\) 也是同类项。

✏️ 例题

例1 指出下列多项式中的同类项：

(1) \(3x - 2y + 1 + 3y - 2x - 5\)；

(2) \(3x^2y - 2xy^2 + \frac{1}{3}xy^2 - \frac{3}{2}x^2y\)。

(1) \(3x\) 与 \(-2x\) 是同类项， \(-2y\) 与 \(3y\) 是同类项， \(1\) 与 \(-5\) 是同类项。
(2) \(3x^2y\) 与 \(-\frac{3}{2}x^2y\) 是同类项， \(-2xy^2\) 与 \(\frac{1}{3}xy^2\) 是同类项。

例2 \(k\) 取何值时， \(3x^ky\) 与 \(-x^2y\) 是同类项？

要使 \(3x^ky\) 与 \(-x^2y\) 是同类项，那么这两项中 \(x\) 的指数必须相等，即 \(k = 2\)。
所以当 \(k = 2\) 时，它们是同类项。

练习

1. 将下列两组中的同类项用线连起来。
\(3x^2y\) \(ba\)
\(-2\) \(-6xy^2\)
\(4m\) \(3\)
\(5xy^2\) \(-4x^2y\)
\(-ab\) \(m\)
（第1题）
左边 \(3x^2y\) 与右边 \(-4x^2y\) 是同类项；
左边 \(-2\) 与右边 \(3\) 是同类项；
左边 \(4m\) 与右边 \(m\) 是同类项；
左边 \(5xy^2\) 与右边 \(-6xy^2\) 是同类项；
左边 \(-ab\) 与右边 \(ba\) 是同类项（\(ab=ba\)）。
2. 写出 \(3ab^2c^3\) 的一个同类项。你能写出多少个？
例如：\(-5ab^2c^3\)、\(\frac{1}{2}ab^2c^3\)、\(ab^2c^3\) 等都是它的同类项。
只要所含字母相同且相同字母的指数也相同，系数可以是任意不为0的有理数，所以可以写出无数个。
3. \(k\) 取何值时， \(-3x^2y^k\) 与 \(4x^2y^6\) 是同类项？
要使它们是同类项，相同字母的指数必须相等，即 \(y\) 的指数 \(k = 6\)。
所以当 \(k = 6\) 时，它们是同类项。

📖 读一读 · 分类——数学中的基本思想

▶

同类项的概念体现了数学中的分类思想。将具有相同特征的项归为一类，使复杂的多项式变得有条理，为后续的合并运算打下基础。这种思想贯穿于整个数学学习，例如数的分类、图形的分类等。

通过识别同类项，我们进一步理解了“字母表示数”的抽象性——字母相同且指数相同的项，尽管系数不同，但本质上是同一类事物。这有助于培养我们的抽象概括能力。
