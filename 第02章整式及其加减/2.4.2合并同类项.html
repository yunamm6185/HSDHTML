2.4 整式的加减

第2课时 合并同类项

🎯 学习目标

✔理解合并同类项的法则
✔能熟练合并多项式中的同类项
✔能通过先化简再求值简化计算
✔体会化归思想在整式运算中的应用

📌 如果一个多项式中含有同类项，那么我们可以把同类项合并起来，使结果得以简化。

例如，根据分配律，有 \(3x^2y + 5x^2y = (3+5)x^2y = 8x^2y\)。

📌 合并同类项的法则

合并同类项：把同类项的系数相加，所得的结果作为和的系数，字母和字母的指数保持不变。

例如，对多项式 \(3x^2y - 4xy^2 - 3 + 5x^2y + 2xy^2 + 5\)，我们可以先运用加法的交换律和结合律将同类项组合在一起，再根据分配律合并：

\(3x^2y\) \(-4xy^2\) \(-3\) \(+5x^2y\) \(+2xy^2\) \(+5\)

\(= (3+5)x^2y + (-4+2)xy^2 + (-3+5)\)
\(= 8x^2y - 2xy^2 + 2\)。

（相同底色表示同类项）

✏️ 例题

例3 合并下列多项式中的同类项：

(1) \(2a^2b - 3a^2b + \dfrac{1}{2}a^2b\)；

(2) \(a^3 - a^2b + ab^2 + a^2b - ab^2 + b^3\)。

(1) \(2a^2b - 3a^2b + \dfrac{1}{2}a^2b\)
\(= \left(2 - 3 + \dfrac{1}{2}\right)a^2b\)
\(= \left(-1 + \dfrac{1}{2}\right)a^2b = -\dfrac{1}{2}a^2b\)。
(2) \(a^3 - a^2b + ab^2 + a^2b - ab^2 + b^3\)
\(= a^3 + b^3 + (-a^2b + a^2b) + (ab^2 - ab^2)\)
\(= a^3 + b^3\)。

例4 求多项式 \(3x^2 + 4x - 2x^2 - x + x^2 - 3x - 1\) 的值，其中 \(x = -3\)。

\(3x^2 + 4x - 2x^2 - x + x^2 - 3x - 1\)
\(= (3 - 2 + 1)x^2 + (4 - 1 - 3)x - 1\)
\(= 2x^2 - 1\)。
当 \(x = -3\) 时，原式 \(= 2 \times (-3)^2 - 1 = 2 \times 9 - 1 = 17\)。

例5 如图所示的窗框，上部分为半圆，下部分为6个大小一样的长方形，长方形的长与宽的比为3:2。如果长方形的长分别为 \(0.4\,\text{m}\)、\(0.5\,\text{m}\)、\(0.6\,\text{m}\) 等，那么窗框所需材料的长度分别是多少？如果长方形的长为 \(a\,\text{m}\) 呢？

如果长方形的长为 \(a\,\text{m}\)，那么它的宽为 \(\dfrac{2}{3}a\,\text{m}\)。
窗框所需材料的长度为：
\(9a + 9 \times \dfrac{2}{3}a + \pi a\)
\(= (9 + 6 + \pi)a = (15 + \pi)a\,\text{m}\)。
当 \(a = 0.4\) 时，\((15 + \pi) \times 0.4 \approx (15 + 3.14) \times 0.4 = 18.14 \times 0.4 = 7.256 \approx 7.3\,\text{m}\)。
当 \(a = 0.5\) 时，\((15 + \pi) \times 0.5 \approx 18.14 \times 0.5 = 9.07\,\text{m}\)。
当 \(a = 0.6\) 时，\((15 + \pi) \times 0.6 \approx 18.14 \times 0.6 = 10.884 \approx 10.9\,\text{m}\)。

练习

1. 如果两个同类项的系数互为相反数，那么合并同类项后，结果是________。
结果是0。
2. 先标出下列各多项式中的同类项，再合并同类项：
(1) \(3x - 2x^2 + 5 + 3x^2 - 2x - 5\)；
(2) \(a^3 + a^2b + ab^2 - a^2b - ab^2 - b^3\)；
(3) \(6a^2 - 5b^2 + 2ab + 5b^2 - 6a^2\)。
(1) 同类项：\(3x\) 与 \(-2x\)， \(-2x^2\) 与 \(3x^2\)， \(5\) 与 \(-5\)。
\(3x - 2x^2 + 5 + 3x^2 - 2x - 5 = (3x-2x) + (-2x^2+3x^2) + (5-5) = x + x^2\)。
(2) 同类项：\(a^2b\) 与 \(-a^2b\)， \(ab^2\) 与 \(-ab^2\)， \(a^3\) 与 \(-b^3\) 不是同类项。
\(a^3 + a^2b + ab^2 - a^2b - ab^2 - b^3 = a^3 - b^3\)。
(3) 同类项：\(6a^2\) 与 \(-6a^2\)， \(-5b^2\) 与 \(5b^2\)， \(2ab\) 单独。
\(6a^2 - 5b^2 + 2ab + 5b^2 - 6a^2 = 2ab\)。
3. 求下列多项式的值：
(1) \(7x^2 - 3x^2 - 2x - 2x^2 + 5 + 6x\)，其中 \(x = -2\)；
(2) \(5a - 2b + 3b - 4a - 1\)，其中 \(a = -1,\; b = 2\)；
(3) \(2x^2 - 3xy + y^2 - 2xy - 2x^2 + 5xy - 2y + 1\)，其中 \(x = \dfrac{22}{7},\; y = -1\)。
(1) 先化简：\(7x^2 - 3x^2 - 2x - 2x^2 + 5 + 6x = (7-3-2)x^2 + (-2+6)x + 5 = 2x^2 + 4x + 5\)。
当 \(x = -2\) 时，原式 \(= 2 \times 4 - 8 + 5 = 8 - 8 + 5 = 5\)。
(2) 先化简：\(5a - 2b + 3b - 4a - 1 = (5a-4a) + (-2b+3b) - 1 = a + b - 1\)。
当 \(a = -1,\; b = 2\) 时，原式 \(= -1 + 2 - 1 = 0\)。
(3) 先化简：\(2x^2 - 3xy + y^2 - 2xy - 2x^2 + 5xy - 2y + 1\)
\(= (2x^2-2x^2) + (-3xy-2xy+5xy) + y^2 - 2y + 1\)
\(= 0 + 0 + y^2 - 2y + 1 = y^2 - 2y + 1\)。
当 \(y = -1\) 时，原式 \(= (-1)^2 - 2 \times (-1) + 1 = 1 + 2 + 1 = 4\)。
（注：\(x\) 的值不影响结果）

📖 读一读 · 化归——将复杂化为简单

▶

合并同类项的过程，就是将一个复杂多项式转化为简单形式的过程，这体现了数学中的化归思想。所谓化归，就是把待解决的问题通过某种转化，归结到另一个已解决或较易解决的问题中去。

在整式运算中，我们常通过合并同类项、去括号等操作，将整式化简，从而更容易求值或进行其他运算。这种化繁为简的思想，是数学学习中最重要的思想之一。

核心素养链接： 数学抽象 · 化归思想 · 运算能力

📌 课堂小结

▼
合并同类项法则：系数相加，字母和字母的指数不变。
步骤：①找出同类项；②利用加法交换律、结合律将同类项放在一起；③合并系数。
求多项式的值时，先化简再代入求值可以简化计算。
合并同类项是整式加减的基础，体现了化归思想。
