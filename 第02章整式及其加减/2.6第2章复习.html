🔍 典型例题

例1 用代数式表示：

(1) \(a\) 与 \(b\) 的平方和；

(2) \(x\) 的相反数与 \(y\) 的倒数的和；

(3) 一个两位数，十位数字是 \(m\)，个位数字是 \(n\)，这个两位数可表示为______。

(1) \(a^2 + b^2\)
(2) \(-x + \dfrac{1}{y}\)
(3) \(10m + n\)

例2 当 \(a = -2, b = 3\) 时，求代数式 \(2a^2 - 3ab + b^2\) 的值。

当 \(a = -2, b = 3\) 时，
\(2a^2 - 3ab + b^2 = 2\times 4 - 3\times(-2)\times3 + 9\)
\(= 8 - (-18) + 9 = 8 + 18 + 9 = 35\)。

例3 指出下列单项式的系数和次数，多项式的项和次数：

(1) \(-\dfrac{3}{5}x^2y\)； (2) \(3x^3 - 2x^2 + 5x - 1\)。

(1) 系数 \(-\dfrac{3}{5}\)，次数 \(3\)。
(2) 项：\(3x^3\)、\(-2x^2\)、\(5x\)、\(-1\)；次数 \(3\)，是三次四项式。

例4 将多项式 \(3x^2y - 4xy^2 + x^3 - 2y^3\) 按 \(x\) 的降幂排列。

按 \(x\) 的降幂排列：\(x^3 + 3x^2y - 4xy^2 - 2y^3\)。

例5 计算：\((4a^2 - 3a + 1) - 2(a^2 - 2a + 3)\)。

原式 \(= 4a^2 - 3a + 1 - 2a^2 + 4a - 6\)
\(= (4a^2 - 2a^2) + (-3a + 4a) + (1 - 6)\)
\(= 2a^2 + a - 5\)。

例6 在括号内填入适当的项：\(x^2 - 2x + 3 = x^2 - (\)______\()\)。

括号前是“-”，括号内各项变号：\(2x - 3\)，即 \(x^2 - 2x + 3 = x^2 - (2x - 3)\)。

例7 先化简，再求值：\(3x^2y - [2xy^2 - 2(xy - \dfrac{3}{2}x^2y)] + xy\)，其中 \(x = 3, y = -\dfrac{1}{3}\)。

原式 \(= 3x^2y - [2xy^2 - 2xy + 3x^2y] + xy\)
\(= 3x^2y - 2xy^2 + 2xy - 3x^2y + xy\)
\(= -2xy^2 + 3xy\)。
当 \(x = 3, y = -\dfrac{1}{3}\) 时，
原式 \(= -2 \times 3 \times \dfrac{1}{9} + 3 \times 3 \times \left(-\dfrac{1}{3}\right) = -\dfrac{2}{3} - 3 = -\dfrac{11}{3}\)。

⚠️ 易错点提醒

  • 单项式的系数包括符号，次数是所有字母的指数之和（系数中的指数不算）。
  • 多项式的项要连同它前面的符号，常数项也是项。
  • 去括号时，括号前是“-”，括号内每一项都要变号；括号前有数字因数，要利用分配律乘到括号内每一项。
  • 添括号时，所添括号前是“-”，括到括号里的各项都要变号。
  • 合并同类项时，系数相加减，字母部分不变；不是同类项不能合并。

综合练习

A组 基础巩固
1. 填空：
(1) 如果 \(a\) 表示一个有理数，那么它的相反数是______；
(2) 一个正方形的边长是 \(a\,\text{cm}\)，把这个正方形的边长增加 \(1\,\text{cm}\)后所得到的正方形的面积是______；
(3) 如果一个两位数的十位数字是 \(a\)，个位数字是 \(b\)，那么这个两位数可表示为______。
(1) \(-a\)
(2) \((a+1)^2\,\text{cm}^2\)
(3) \(10a + b\)
2. 填表：
单项式 \(-x\) \(\dfrac{2}{3}ab\) \(-\dfrac{x^2y^3}{3}\) \(1.2\times10^3 x^2\)
系数
次数
系数依次：\(-1\)，\(\dfrac{2}{3}\)，\(-\dfrac{1}{3}\)，\(1.2\times10^3\)；
次数依次：\(1\)，\(2\)，\(5\)，\(2\)。
3. 下列多项式各由哪些项组成？是几次几项式？
(1) \(3x^2 - 2x + 1\)；
(2) \(x^3 - 2x^2y^2 + 3y^2\)。
(1) 项：\(3x^2\)、\(-2x\)、\(1\)；二次三项式。
(2) 项：\(x^3\)、\(-2x^2y^2\)、\(3y^2\)；四次三项式（\(-2x^2y^2\)次数4）。
4. 计算：\((3x^2 + 2x - 5) - (x^2 - 3x + 1)\)。
原式 \(= 3x^2 + 2x - 5 - x^2 + 3x - 1 = 2x^2 + 5x - 6\)。
B组 能力提升
1. 已知 \(A = 2x^2 + 3xy - 2x - 1\)，\(B = -x^2 + xy - 1\)，求 \(A - 2B\) 的值。
\(A - 2B = (2x^2 + 3xy - 2x - 1) - 2(-x^2 + xy - 1)\)
\(= 2x^2 + 3xy - 2x - 1 + 2x^2 - 2xy + 2\)
\(= 4x^2 + xy - 2x + 1\)。
2. 先化简，再求值：\(5(3a^2b - ab^2) - (ab^2 + 3a^2b)\)，其中 \(a = \dfrac{1}{2}, b = -2\)。
化简：\(15a^2b - 5ab^2 - ab^2 - 3a^2b = 12a^2b - 6ab^2\)。
当 \(a = \dfrac{1}{2}, b = -2\) 时，
原式 \(= 12 \times \dfrac{1}{4} \times (-2) - 6 \times \dfrac{1}{2} \times 4 = 12 \times \dfrac{1}{4} \times (-2) - 6 \times \dfrac{1}{2} \times 4\)
\(= 3 \times (-2) - 3 \times 4 = -6 - 12 = -18\)。
3. 一个多项式加上 \(2x^2 - x + 5\) 得 \(3x^2 - 2x + 1\)，求这个多项式。
设这个多项式为 \(P\)，则 \(P + (2x^2 - x + 5) = 3x^2 - 2x + 1\)，
所以 \(P = (3x^2 - 2x + 1) - (2x^2 - x + 5) = 3x^2 - 2x + 1 - 2x^2 + x - 5 = x^2 - x - 4\)。
C组 拓展探究
1. 设 \(x\) 表示一个两位数，\(y\) 表示一个三位数，若把 \(x\) 放在 \(y\) 的右边组成一个五位数，则这个五位数可以表示为______。
\(y \times 100 + x\)，即 \(100y + x\)。
2. 已知 \(a - b = 3\)，\(ab = -2\)，求代数式 \((2a + 3b - 2ab) - (a + 4b + ab) - (3ab + 2b - 2a)\) 的值。
原式化简：先去括号：
\(= 2a + 3b - 2ab - a - 4b - ab - 3ab - 2b + 2a\)
\(= (2a - a + 2a) + (3b - 4b - 2b) + (-2ab - ab - 3ab)\)
\(= 3a - 3b - 6ab = 3(a - b) - 6ab\)。
代入 \(a - b = 3\)，\(ab = -2\) 得：原式 \(= 3 \times 3 - 6 \times (-2) = 9 + 12 = 21\)。

📖 数学思想 · 数式通性与化归

▼

本章从用字母表示数开始，将具体的数抽象为代数式，体现了数式通性——字母可以像数一样进行运算，并且运算律同样适用。

在整式加减中，我们常将减法转化为加法，将除法转化为乘法（倒数），将整式加减转化为合并同类项，这体现了化归思想——把新问题转化为已经解决的问题。

同类项的概念体现了分类思想；添括号与去括号的互逆过程体现了逆向思维。这些数学思想是后续学习的重要基础。

核心素养链接： 数学抽象 · 符号意识 · 运算能力 · 分类讨论 · 化归思想

📌 复习要点

▼
代数式：用字母表示数，列代数式，求代数式的值。
单项式：系数、次数（所有字母的指数和）。
多项式：项、常数项、次数（次数最高项的次数）。
升幂排列与降幂排列。
同类项：字母相同且相同字母指数相同。
合并同类项：系数相加，字母部分不变。
去括号法则：括号前是“+”不变号，是“-”全变号。
添括号法则：添“+”不变号，添“-”全变号。
整式加减步骤：先去括号，再合并同类项。
