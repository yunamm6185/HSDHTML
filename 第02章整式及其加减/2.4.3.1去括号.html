2.4 整式的加减

第3课时 去括号

🎯 学习目标

✔理解去括号法则的推导过程
✔掌握去括号法则，能正确去掉括号
✔能运用去括号法则化简整式
✔体会符号意识与运算的严谨性

📌 周三下午，校图书馆内起初有 \(a\) 位同学。后来又有一些同学前来阅读，第一批来了 \(b\) 位同学，第二批又来了 \(c\) 位同学，则图书馆内共有 \(a + b + c\) 位同学。我们还可以这样理解：后来两批一共来了 \((b + c)\) 位同学，因而图书馆内共有 \(a + (b + c)\) 位同学。由于 \(a + b + c\) 和 \(a + (b + c)\) 均表示同一个量，于是我们便得到等式：

\(a + (b + c) = a + b + c\)。

📝 做一做

图书馆内原有 \(a\) 位同学。后来有些同学因上课要离开，第一批走了 \(b\) 位同学，第二批又走了 \(c\) 位同学。试用两种方式写出图书馆内还剩下的同学数，你能从中发现什么关系？

方法一：直接减：\(a - b - c\)；
方法二：先计算一共走了 \((b + c)\) 人，则剩余 \(a - (b + c)\)。
由结果相同可得：\(a - (b + c) = a - b - c\)。

观察①②两个等式中括号和各项正负号的变化，你能发现什么规律？

括号没了，正负号没变
\(a + (b + c)\)
\(a + b + c\)
去括号后，括号内各项正负号不变
括号没了，正负号却变了
\(a - (b + c)\)
\(a - b - c\)
去括号后，括号内各项正负号都改变
📌 去括号法则

括号前面是“+”号，把括号和它前面的“+”号去掉，括号里各项都不改变正负号。

括号前面是“-”号，把括号和它前面的“-”号去掉，括号里各项都改变正负号。

例如：\(a + (b - c) = a + b - c\)； \(a - (b - c) = a - b + c\)。

✏️ 例题

例6 去括号：

(1) \(a + (b - c)\)； (2) \(a - (b - c)\)； (3) \(a + (-b + c)\)； (4) \(a - (-b - c)\)。

(1) \(a + (b - c) = a + b - c\)
(2) \(a - (b - c) = a - b + c\)
(3) \(a + (-b + c) = a - b + c\)
(4) \(a - (-b - c) = a + b + c\)

例7 先去括号，再合并同类项：

(1) \((x + y - z) + (x - y + z) - (x - y - z)\)；

(2) \((a^2 + 2ab + b^2) - (a^2 - 2ab + b^2)\)；

(3) \(3(2x^2 - y^2) - 2(3y^2 - 2x^2)\)。

(1) \((x + y - z) + (x - y + z) - (x - y - z)\)
\(= x + y - z + x - y + z - x + y + z\)
\(= (x + x - x) + (y - y + y) + (-z + z + z)\)
\(= x + y + z\)
(2) \((a^2 + 2ab + b^2) - (a^2 - 2ab + b^2)\)
\(= a^2 + 2ab + b^2 - a^2 + 2ab - b^2\)
\(= (a^2 - a^2) + (2ab + 2ab) + (b^2 - b^2)\)
\(= 4ab\)
(3) \(3(2x^2 - y^2) - 2(3y^2 - 2x^2)\)
\(= 6x^2 - 3y^2 - 6y^2 + 4x^2\)
\(= (6x^2 + 4x^2) + (-3y^2 - 6y^2)\)
\(= 10x^2 - 9y^2\)

练习

1. 去括号：
(1) \((a - b) + (-c - d)\)；
(2) \((a - b) - (-c - d)\)；
(3) \(-(a - b) + (-c - d)\)；
(4) \(-(a - b) - (-c - d)\)。
(1) \(a - b - c - d\)
(2) \(a - b + c + d\)
(3) \(-a + b - c - d\)
(4) \(-a + b + c + d\)
2. 判断下列去括号是否正确，如果不正确，请说明错在哪里，并加以改正：
(1) \(a - (b - c) = a - b - c\)；
(2) \(-(a - b + c) = -a + b - c\)；
(3) \(c + 2(a - b) = c + 2a - b\)。
(1) 不正确。括号前是“-”，去括号后各项应变号，正确应为：\(a - b + c\)。
(2) 正确。
(3) 不正确。括号前的系数2应乘以括号内每一项，正确应为：\(c + 2a - 2b\)。
3. 化简：
(1) \(a^2 - 2(ab - b^2) - b^2\)；
(2) \((x^2 - y^2) - 3(2x^2 - 3y^2)\)；
(3) \(7a^2b - (-4a^2b + 5ab^2) - 2(2a^2b - 3ab^2)\)。
(1) \(a^2 - 2ab + 2b^2 - b^2 = a^2 - 2ab + b^2\)
(2) \(x^2 - y^2 - 6x^2 + 9y^2 = -5x^2 + 8y^2\)
(3) \(7a^2b + 4a^2b - 5ab^2 - 4a^2b + 6ab^2\)
\(= (7a^2b + 4a^2b - 4a^2b) + (-5ab^2 + 6ab^2) = 7a^2b + ab^2\)

📖 读一读 · 符号意识——运算中的规则

▶

去括号法则本质上源于分配律和符号运算。例如，\(a - (b + c) = a - b - c\) 可以理解为减去两个数的和等于分别减去这两个数，这是加法逆运算的体现。当括号前是“-”时，去掉括号相当于乘以 \(-1\)，因此各项符号都要改变。

正确运用去括号法则，是培养符号意识的关键一步。在后续的整式加减、解方程等学习中，去括号都是基础技能。

核心素养链接： 符号意识 · 运算能力 · 推理能力

📌 课堂小结

▼
去括号法则：括号前是“+”号，去括号后各项不变号；括号前是“-”号，去括号后各项都变号。
当括号前有数字因数时，要利用分配律将数字乘到括号内每一项。
去括号后再合并同类项可化简整式。
去括号是整式加减的基础，需细心避免符号错误。
