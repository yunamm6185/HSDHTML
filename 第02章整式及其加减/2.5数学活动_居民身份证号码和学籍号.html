数学活动：居民身份证号码和学籍号

🎯 活动目标

✔了解学籍号的构成，体会数字编码在实际生活中的应用
✔能根据规则解读和分析身份证号码、学籍号中的信息
✔尝试设计简洁有效的校内学号，体会数学与生活的联系

📌 为了把众多的对象区分开来，人们常常设计使用一些编码方法。

我们国家为了区分每一个人，1987年实行了身份证制度，每一个公民都有一个身份证号码。从1999年10月1日起，身份证号码由原15位升至18位。2004年《中华人民共和国居民身份证法》正式施行。

教育部为了区分每一个学生，2015年1月21日通过全国中小学生学籍信息管理系统为全国中小学生发放了正式学籍号。学生学籍号以学生居民身份证号码为基础，从入幼儿园或入小学之初采集学籍信息后开始使用，一人一号，籍随人走，终身不变。

🔍 探究一：居民身份证号码

📋 18位身份证号码的构成
位数 1-6 7-14 15-17 18
名称 地址码 出生日期码 顺序码 校验码
含义 表示居民常住户口所在县（市、旗、区）的行政区划代码 表示出生年、月、日，格式为YYYYMMDD 在同一地址码区域内，对同年同月同日出生的人编定的顺序号
第17位奇数表示男性，偶数表示女性		 根据前17位数字按统一公式计算出来的校验码
校验码为10时用X表示

📝 示例分析： 身份证号码 11010119900307663X

前6位 110101：北京市东城区

第7-14位 19900307：出生于1990年3月7日

第15-17位 663：顺序码，第17位是3（奇数），表示男性

第18位 X：校验码，表示10

例2：320105198512124321 → 江苏省南京市浦口区，1985年12月12日出生，第17位是2（偶数），女性
例3：440304200108152319 → 广东省深圳市福田区，2001年8月15日出生，第17位是1（奇数），男性

🔍 探究二：学籍号

📋 全国正式学籍号的编码规则

1. 有身份证号的学生： G + 身份证号（19位）

2. 无身份证号的学生： J + 地址码(6位) + 出生日期(8位) + 顺序码(3位) + 校验码(1位)（19位）

3. 全国临时学籍号： L + 地址码(6位) + 出生日期(8位) + 随机码(4位)（19位）

📝 示例分析： 学籍号 G11010820100101453X

首位 G：表示有身份证的学生

后续18位 11010820100101453X：对应的身份证号码（北京市海淀区，2010年1月1日出生，顺序码453，校验码X）

学籍号 J440304200809150012：
J表示无身份证学生，440304是地址码（广东省深圳市福田区），20080915是出生日期，001是顺序码，2是校验码。

💬 思考与讨论

1. 数一数你自己的身份证号码，看看共有多少位？你知道身份证号码的编码规则吗？
18位。前6位地址码，第7-14位出生日期码，第15-17位顺序码（第17位奇数男偶数女），第18位校验码（X表示10）。
2. 你知道自己的学籍号吗？你知道学籍号的编码规则吗？
全国正式学籍号有19位，G开头表示有身份证的学生（后跟身份证号），J开头表示无身份证的学生，L开头表示临时学籍号。

✏️ 活动：设计校内学号

除了学籍号外，我们在学校内往往还有一个自己的学号，能反映出入学年份、班级、性别等信息。你知道自己的校内学号吗？你了解学号的编码规则吗？

任务： 请你对校内学号的编码规则提出一些好的建议，让学号更加简洁有效。

要求：学号应包含入学年份、班级、性别、个人序号等信息，总长度不宜过长，便于记忆和管理。

示例方案一（8位）： 入学年份后两位(2) + 年级(1) + 班级(2) + 性别(1) + 序号(2)
例如：230710115 表示2023年入学，七年级10班，1号，男生（1表示男，2表示女）
示例方案二（9位）： 入学年份(4) + 班级(2) + 性别(1) + 序号(2)
例如：2023070115 表示2023年入学，7班，1号，男生

📖 读一读 · 数字编码中的数学思想

▼

身份证号码和学籍号都是典型的数字编码，它们蕴含着丰富的数学思想：

  • 分类思想：将不同类型的信息用不同的数位表示（地址、出生日期、顺序码等）。
  • 有序性：通过数字的顺序体现逻辑关系（如顺序码按出生先后编排）。
  • 奇偶性：用第17位的奇偶性表示性别，是数学性质的巧妙应用。
  • 算法思想：校验码的计算（加权求和取模）是数学算法在数据校验中的典型应用。
  • 简洁性：用18位数字就能唯一标识14亿人口，体现了数学符号的高度抽象性。

数学不仅存在于课本中，更渗透在生活的方方面面。当我们用字母表示数、用整式描述规律时，其实就是在用数学语言“编码”这个世界。

核心素养链接： 数学抽象 · 符号意识 · 模型思想
