2.3 整式

第3课时 升幂排列与降幂排列

🎯 学习目标

✔理解多项式按某一字母升幂或降幂排列的意义
✔能熟练地将多项式按指定字母进行升幂或降幂排列
✔掌握重新排列时各项符号的处理方法
✔体会有序思考与分类讨论的数学思想

📌 运用加法交换律，可以任意交换多项式各项的位置。将多项式 \(x^2 + x + 1\) 中各项的位置交换，可以得到哪些不同的排列方式？在众多的排列方式中，你认为哪几种比较整齐？

试一试： 交换多项式 \(x^2 + x + 1\) 各项的位置，写出几种不同的排列方式。

例如：\(x^2 + x + 1\)，\(x^2 + 1 + x\)，\(x + x^2 + 1\)，\(x + 1 + x^2\)，\(1 + x^2 + x\)，\(1 + x + x^2\)。
其中 \(x^2 + x + 1\)（按 \(x\) 的指数从大到小）和 \(1 + x + x^2\)（按 \(x\) 的指数从小到大）比较整齐。
📌 升幂排列与降幂排列

降幂排列：把一个多项式的各项按某一个字母的指数从大到小的顺序排列。

升幂排列：把一个多项式的各项按某一个字母的指数从小到大的顺序排列。

例如，多项式 \(x^2 + x + 1\) 是按 \(x\) 的降幂排列，\(1 + x + x^2\) 是按 \(x\) 的升幂排列。

又如，多项式 \(5x^2 + 3x - 2x^3 - 1\)，按 \(x\) 的降幂排列是 \(-2x^3 + 5x^2 + 3x - 1\)；按 \(x\) 的升幂排列是 \(-1 + 3x + 5x^2 - 2x^3\)。

✏️ 例题

例4 把多项式 \(2r - 1 + \dfrac{4}{3}r^3 - r^2\) 按 \(r\) 的升幂排列。

按 \(r\) 的升幂排列为：
\(-1 + 2r - r^2 + \dfrac{4}{3}r^3\)。

例5 把多项式 \(a^3 + b^2 - 3a^2b - 3ab^3\) 重新排列：

(1) 按 \(a\) 的升幂排列； (2) 按 \(a\) 的降幂排列。

(1) 按 \(a\) 的升幂排列为：\(b^2 - 3ab^3 - 3a^2b + a^3\)。
(2) 按 \(a\) 的降幂排列为：\(a^3 - 3a^2b - 3ab^3 + b^2\)。
试试看，将这个多项式按 \(b\) 的升幂（或降幂）排列：
原多项式各项中，\(b\) 的指数分别为：0（\(a^3\)）、1（\(-3a^2b\)）、2（\(b^2\)）、3（\(-3ab^3\)）。
按 \(b\) 的升幂排列（指数从小到大）应为：\(a^3 - 3a^2b + b^2 - 3ab^3\)。
按 \(b\) 的降幂排列（指数从大到小）应为：\(-3ab^3 + b^2 - 3a^2b + a^3\)。

注意：

(1) 重新排列多项式时，每一项一定要连同它的正负号一起移动。

(2) 含有两个或两个以上字母的多项式，通常按其中某一个字母的升幂或降幂排列。

练习

1. 把多项式 \(2x^2 + \dfrac{2}{5}x^3 + x - 5x^4 - \dfrac{1}{3}\) 重新排列：
(1) 按 \(x\) 的升幂排列；
(2) 按 \(x\) 的降幂排列。
(1) 升幂排列：\(-\dfrac{1}{3} + x + 2x^2 + \dfrac{2}{5}x^3 - 5x^4\)。
(2) 降幂排列：\(-5x^4 + \dfrac{2}{5}x^3 + 2x^2 + x - \dfrac{1}{3}\)。
2. 把多项式 \(x^4 - y^4 + 3x^3y - 2xy^2 - 5x^2y^3\) 重新排列：
(1) 按 \(x\) 的降幂排列；
(2) 按 \(y\) 的降幂排列。
(1) 按 \(x\) 的降幂排列：\(x^4 + 3x^3y - 5x^2y^3 - 2xy^2 - y^4\)。
(2) 按 \(y\) 的降幂排列：\(-y^4 - 5x^2y^3 - 2xy^2 + 3x^3y + x^4\)。
（按 \(y\) 的指数从大到小：\(y^4\)、\(y^3\)、\(y^2\)、\(y^1\)、常数项）

📖 读一读 · 有序排列——数学中的秩序美

▶

将多项式按某个字母的指数从小到大或从大到小排列，就像整理书架上的书籍一样，使表达变得清晰、美观，便于后续运算（如加减、乘除、因式分解）。这种有序化的过程，体现了数学中的秩序感和分类思想。

升幂排列和降幂排列不仅是一种书写规范，更是数学抽象与符号表达的自然要求。通过排列，我们可以快速看出多项式的次数、主要项等信息，为后续学习整式的运算打下基础。

核心素养链接： 数学抽象 · 分类讨论 · 有序思维

📌 课堂小结

▼
降幂排列：按某字母指数从大到小排列；升幂排列：按指数从小到大排列。
重新排列时，每一项必须连同它的正负号一起移动。
含有多个字母的多项式，通常按其中某一个字母的升幂或降幂排列。
有序排列使多项式表达更清晰，便于后续运算。
