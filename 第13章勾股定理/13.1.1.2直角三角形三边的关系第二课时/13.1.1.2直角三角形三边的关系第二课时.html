第13章 勾股定理

13.1.1 直角三角形三边的关系（第二课时）

🎯 学习目标

✔灵活运用勾股定理求直角三角形各边
✔会用勾股定理解决实际测量问题
✔体会勾股定理在生活中的广泛应用

上一课时，我们通过探索发现并证明了勾股定理：在直角三角形中，若两直角边为 $a$、$b$，斜边为 $c$，则 $a^2 + b^2 = c^2$。

本课时我们将深入探索勾股定理的应用，学习如何利用勾股定理求边长，并用它解决实际问题。

例2

如图13.1.6，在 $\triangle ABC$ 中，$\angle ACB = 90°$，斜边 $AB$ 比直角边 $AC$ 长 $2\ \text{cm}$，另一条直角边 $BC = 6\ \text{cm}$，求 $AB$ 的长。

图13.1.6

解： 设 $AC = x$（$\text{cm}$），则 $AB = x + 2$（$\text{cm}$）。

由勾股定理得：$$AC^2 + BC^2 = AB^2$$

$$x^2 + 6^2 = (x+2)^2$$

$$x^2 + 36 = x^2 + 4x + 4$$

$$4x = 32$$，即 $$x = 8$$

所以 $AB = x + 2 = 10\ \text{cm}$。

例3

如图13.1.6，为了求出位于湖两岸的点 $A$、$B$ 之间的距离，一名观测者在点 $C$ 处设桩，使 $\triangle ABC$ 恰好为直角三角形。通过测量，得到 $AC$ 的长为 $160\ \text{m}$，$BC$ 的长为 $128\ \text{m}$。问：从点 $A$ 穿过湖到点 $B$ 有多远？

图13.1.6 湖两岸测距

解：如图13.1.6，在 Rt$\triangle ABC$ 中，$\angle B = 90°$，

$$AC = 160\ \text{m},\quad BC = 128\ \text{m}$$

根据勾股定理，可得

$$AB^2 + BC^2 = AC^2$$

$$AB^2 + 128^2 = 160^2$$

$$AB^2 = 25600 - 16384 = 9216$$

$$AB = \sqrt{9216} = 96\ (\text{m})$$

答：从点 $A$ 穿过湖到点 $B$ 的距离为 $\mathbf{96\ \text{m}}$。

例4

如图13.1.7，一根旗杆在离地面 $9\ \text{m}$ 处断裂，旗杆顶端落在离旗杆底部 $12\ \text{m}$ 处，旗杆折断之前有多高？

图13.1.7 旗杆折断问题

解： 设旗杆折断部分长 $x$（$\text{m}$）。

由题意，旗杆折断后形成一个直角三角形：

  • 一直角边为旗杆剩余高度 $= 9\ \text{m}$
  • 另一直角边为旗杆顶端落地点到旗杆底部的距离 $= 12\ \text{m}$
  • 斜边为旗杆折断部分的长度 $= x\ \text{m}$

根据勾股定理：$$x^2 = 9^2 + 12^2 = 81 + 144 = 225$$

$$x = \sqrt{225} = 15\ \text{m}$$

旗杆原来的高度 $= 9 + 15 = 24\ \text{m}$。

答：旗杆折断之前高 $\mathbf{24\ \text{m}}$。

✏️ 练习

1. 如图，小方格都是边长为 $1$ 的正方形，求四边形 $ABCD$ 的面积和周长（均精确到 $0.1$）。

练习1图（小方格边长为1，顶点都在格点上）

解：

（1）求面积：用割补法（皮克公式或外接矩形减去角上三角形）

四边形放入 $7\times5$ 的外接矩形中，矩形面积 $= 7 \times 5 = 35$（格）

四个角上的三角形面积（数格子或用底×高÷2）：

  • 左上角三角形（A点）：底$=1$格，高$=1$格，面积$=0.5$格
  • 右上角三角形（B点上方）：底$=2$格，高$=2$格，面积$=2$格
  • 右下角三角形（C点右方）：底$=1$格，高$=4$格，面积$=2$格
  • 左下角三角形（D点左方）：底$=2$格，高$=5$格，面积$=5$格

四边形面积 $= 35 - (0.5 + 2 + 2 + 5) = 35 - 9.5 = 25.5$（格）$= 25.5$（平方单位）

（2）求周长：用勾股定理计算各边长

  • $AB = \sqrt{6^2 + 1^2} = \sqrt{37} \approx 6.1$
  • $BC = \sqrt{1^2 + 4^2} = \sqrt{17} \approx 4.1$
  • $CD = \sqrt{4^2 + 1^2} = \sqrt{17} \approx 4.1$
  • $DA = \sqrt{1^2 + 4^2} = \sqrt{17} \approx 4.1$

周长 $\approx 6.1 + 4.1 + 4.1 + 4.1 = 18.4$

答：面积约为 $\mathbf{25.5}$（平方单位），周长约为 $\mathbf{18.4}$。

2. 假期中，王强和同学到某海岛上去探宝旅游。如图，按照探宝图，他们在点 $A$ 处登陆后先往东走 $8\ \text{km}$，又往北走 $2\ \text{km}$，遇到障碍后又往西走 $3\ \text{km}$，再折向北走到 $6\ \text{km}$ 处往东一拐，仅走 $1\ \text{km}$ 就找到了宝藏。问：登陆点 $A$ 到宝藏埋藏点 $B$ 的直线距离是多少千米？

练习2图（探宝路线示意图）

解：分析探宝路线中东西方向和南北方向的净移动量：

  • 先往东走 $8\ \text{km}$
  • 再往北走 $2\ \text{km}$
  • 然后往西走 $3\ \text{km}$（相当于抵消了部分向东的移动）
  • 接着往北走 $6\ \text{km}$
  • 最后往东走 $1\ \text{km}$

计算两个方向的净移动量：

• 东西方向：$8 - 3 + 1 = 6\ (\text{km})$（最终在点 $A$ 的东边 $6\ \text{km}$ 处）

• 南北方向：$2 + 6 = 8\ (\text{km})$（最终在点 $A$ 的北边 $8\ \text{km}$ 处）

从点 $A$ 到点 $B$ 的直线距离、东西方向的净位移、南北方向的净位移恰好构成一个直角三角形，其中两直角边分别为 $6\ \text{km}$ 和 $8\ \text{km}$。

由勾股定理得：

$$AB = \sqrt{6^2 + 8^2} = \sqrt{36 + 64} = \sqrt{100} = 10\ (\text{km})$$

答：登陆点 $A$ 到宝藏埋藏点 $B$ 的直线距离为 $\mathbf{10\ \text{km}}$。

📌 课堂小结

• 勾股定理应用：已知直角三角形两边可求第三边，$c = \sqrt{a^2+b^2}$（求斜边），$a = \sqrt{c^2-b^2}$（求直角边）。
• 实际应用：测量湖泊宽度（例3）、旗杆折断问题（例4）、探宝路线距离（练习2）等，都用到勾股定理。
• 坐标思想：对于复杂的折线路程问题，可以计算净位移量，再用勾股定理求直线距离。
• 遇到含直角的几何问题，常常需要构造直角三角形，再用勾股定理求解。

🧠 数学思想·核心素养

建模思想：将实际问题（湖宽、旗杆折断、探宝路线）转化为直角三角形模型。
坐标思想：利用净位移量（东西方向和南北方向的合成）简化折线路程问题。
化归思想：利用勾股定理，将求第三边的问题化归为方程求解。
