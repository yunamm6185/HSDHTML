📖 一、核心知识点回顾

1️⃣ 勾股定理（Pythagorean Theorem）

内容：直角三角形两直角边的平方和等于斜边的平方。

符号语言：在 Rt$\triangle ABC$ 中，$\angle C=90°$，则 $$\boxed{a^2 + b^2 = c^2}$$ 其中 $a$、$b$ 为两直角边，$c$ 为斜边。

适用条件：

  • 必须是直角三角形（或构造出直角三角形）
  • $c$ 必须是斜边（即直角所对的边）

2️⃣ 勾股定理的逆定理

内容：若 $\triangle ABC$ 的三边 $a,b,c$ 满足 $a^2+b^2=c^2$，则 $\triangle ABC$ 是直角三角形，且 $c$ 边所对的角为直角。

用途：判定一个三角形是否为直角三角形。

常见勾股数：$(3,4,5)$、$(5,12,13)$、$(6,8,10)$、$(7,24,25)$、$(8,15,17)$、$(9,40,41)$

3️⃣ 勾股定理的应用方法

问题类型解题思路关键步骤
求线段长度识别/构造直角三角形找已知两边 → 用公式求第三边
判定直角三角形检验最长边的平方是否等于其他两边平方和确定$c$ → 验证$a^2+b^2=c^2$
最短路径（立体展开）将立体图形表面展平成平面图形展开 → 构造直角三角形 → 计算
列方程求解设未知数，利用勾股定理建立方程设元 → 列式 → 解方程 → 检验
方格图法（数方格）利用网格纸，直接数格子算面积画方格 → 数勾方、股方格子数 → 开方求弦

4️⃣ 反证法

定义：先假设命题的反面成立，然后通过逻辑推理导出矛盾，从而证明原命题成立。
核心思想：正难则反，从结论的反面出发。

三个步骤：

① 反设
② 归谬
③ 结论

适用场景：

  • 证明"不存在"类命题
  • 证明"至少/至多"类命题
  • 证明"唯一性"命题
  • 直接证明困难的反面命题
【经典例题】用反证法证明：$\sqrt{2}$ 是无理数。

证明：假设 $\sqrt{2}$ 是有理数，则 $\sqrt{2} = \frac{p}{q}$（$p$、$q$ 为互质整数，$q\neq 0$）。
两边平方得 $2 = \frac{p^2}{q^2}$，即 $p^2 = 2q^2$。
所以 $p^2$ 是偶数，$p$ 也是偶数，设 $p=2k$。
代入得 $(2k)^2 = 2q^2$，即 $4k^2 = 2q^2$，$q^2 = 2k^2$。
所以 $q^2$ 也是偶数，$q$ 也是偶数。
这与 $p$、$q$ 互质矛盾！故假设不成立，$\sqrt{2}$ 是无理数。 ∎

⚠️ 5️⃣ 易错点提醒

  1. 混淆斜边和直角边：使用公式时必须确认 $c$ 是最大边（斜边）。
  2. 忘记分类讨论：已知两边求第三边时，需判断已知两边是"两直角边"还是"一直角边一斜边"。
  3. 单位不统一：计算前确保所有长度单位一致。
  4. 结果未化简：如 $\sqrt{50}$ 应化简为 $5\sqrt{2}$；$\sqrt{\frac{2}{3}}$ 应化为 $\frac{\sqrt{6}}{3}$。
  5. 忽略实际意义：长度必须为正数，负值要舍去。

6️⃣ 勾股定理的无字证明（数形结合的经典）

含义：不依赖文字语言，只通过图形面积关系证明勾股定理——"以形说数"的典范。

🌟 赵爽弦图（中国·公元3世纪）

$$c^2 = 4\times\frac{1}{2}ab + (b-a)^2$$

展开即得 $a^2+b^2=c^2$

🔷 青朱出入图（中国·三国时期）

$$(a+b)^2 = 4\times\frac{1}{2}ab + c^2$$

展开即得 $a^2+b^2=c^2$

核心思想：用面积相等证明勾股定理。大正方形面积 = 四个直角三角形 + 中间正方形，分别用两种方式表示，化简即得 $a^2+b^2=c^2$。

✏️ 二、综合练习

【基础巩固】

1. 在 Rt$\triangle ABC$ 中，$\angle C=90°$，$AC=5$，$BC=12$。求：(1) 斜边 $AB$；(2) 斜边上的高 $CD$。

解：(1) 由勾股定理：$$AB=\sqrt{AC^2+BC^2}=\sqrt{25+144}=13$$

(2) 利用面积法：$\frac{1}{2} \cdot AC \cdot BC = \frac{1}{2} \cdot AB \cdot CD$

所以 $CD = \frac{AC \cdot BC}{AB} = \frac{5\times 12}{13} = \frac{60}{13}$

2. 已知直角三角形的两直角边分别为下列各组数值，求斜边长：
(1) $6$ 和 $8$    (2) $5$ 和 $12$    (3) $8$ 和 $15$    (4) $9$ 和 $40$

(1) $c = \sqrt{6^2+8^2} = \sqrt{36+64} = \sqrt{100} = 10$

(2) $c = \sqrt{5^2+12^2} = \sqrt{25+144} = \sqrt{169} = 13$

(3) $c = \sqrt{8^2+15^2} = \sqrt{64+225} = \sqrt{289} = 17$

(4) $c = \sqrt{9^2+40^2} = \sqrt{81+1600} = \sqrt{1681} = 41$

💡 这些都是常见的勾股数：$(6,8,10)$、$(5,12,13)$、$(8,15,17)$、$(9,40,41)$

3. 已知三角形的三边分别为下列各组数值，判断哪些是直角三角形？
(1) $9$, $12$, $15$    (2) $\sqrt{3}$, $\sqrt{4}$, $\sqrt{7}$    (3) $1$, $1$, $\sqrt{2}$    (4) $2$, $3$, $4$

(1) $9^2+12^2=81+144=225=15^2$ ✓ 是直角三角形

(2) $(\sqrt{3})^2+(\sqrt{4})^2=3+4=7=(\sqrt{7})^2$ ✓ 是直角三角形

(3) $1^2+1^2=2=(\sqrt{2})^2$ ✓ 是等腰直角三角形

(4) $2^2+3^2=13\neq 16=4^2$ ✗ 不是直角三角形

4. 【方格图法】如图，每个小方格的边长为 1，试在方格纸上求线段 $AB$ 的长度。

解：在方格纸上，过点 $A$、$B$ 分别向 $x$ 轴、$y$ 轴作垂线，构造直角三角形：

$$AB = \sqrt{AC^2 + BC^2} = \sqrt{5^2 + 4^2} = \sqrt{25 + 16} = \sqrt{41}$$

方法总结：方格图中两点间距离 = 水平格子数的平方 + 垂直格子数的平方，然后开方。

答：$AB = \sqrt{41}$。

【能力提升】

5. 如图，在等腰 $\triangle ABC$ 中，$AB=AC=13$，$BC=10$。求底边上的高 $AD$ 和三角形的面积。

图：等腰三角形 ABC（AB=AC=13，BC=10）

解：因为 $\triangle ABC$ 是等腰三角形，$AB = AC$，$AD$ 是底边 $BC$ 上的高，所以 $D$ 是 $BC$ 的中点。

$BD = DC = \frac{BC}{2} = \frac{10}{2} = 5$

在 Rt$\triangle ABD$ 中，由勾股定理：

$$AD = \sqrt{AB^2 - BD^2} = \sqrt{13^2 - 5^2} = \sqrt{169 - 25} = \sqrt{144} = 12$$

三角形的面积：

$$S_{\triangle ABC} = \frac{1}{2} \times BC \times AD = \frac{1}{2} \times 10 \times 12 = 60$$

答：底边上的高 $AD = \mathbf{12}$，面积 $\mathbf{= 60}$。

【拓展挑战】

6. 蚂蚁爬长方体（拓展挑战）：如图，在长 $30\ \text{cm}$、宽 $50\ \text{cm}$、高 $40\ \text{cm}$ 的长方体上，有一只蚂蚁准备顺着长方体的表面从顶点 $A$ 处爬到相对的顶点 $G$ 处。试帮这只蚂蚁设计一条最佳路线，使其爬行的路程最短。

图：长方体（长30cm、宽50cm、高40cm，蚂蚁从A沿表面爬到G）

解：将长方体表面沿棱剪开展平，有三种展开方式。设长$a=30$, 宽$b=50$, 高$c=40$：

展开方式 矩形尺寸 最短距离
① 经前面+右面$(a+b)\times c = 80\times 40$$\sqrt{(30+50)^2+40^2}=\sqrt{6400+1600}=\sqrt{8000}\approx 89.44$
② 经上面+前面$(c+a)\times b = 70\times 50$$\sqrt{(40+30)^2+50^2}=\sqrt{4900+2500}=\sqrt{7400}\approx 86.02$
③ 经左面+上面$(c+b)\times a = 90\times 30$$\sqrt{(40+50)^2+30^2}=\sqrt{8100+900}=\sqrt{9000}\approx 94.87$

比较三种方式，最短路径约为 $\mathbf{86.02\ \text{cm}}$（取方式②：经上面+前面）。

💡 最佳路线：蚂蚁从A点出发，先沿前面（或后面）爬到上表面，再沿上表面爬到G点。

📐 长方体表面最短路径通解

设长方体的长、宽、高分别为 $a$、$b$、$c$（假设 $a \leq b \leq c$），从一个顶点爬到对角顶点，有三种展开方式：

展开面 矩形边长 路径平方
前面+右面 $(a+b) \times c$ $d_1^2 = (a+b)^2 + c^2$
上面+前面 $(c+a) \times b$ $d_2^2 = (c+a)^2 + b^2$
左面+上面 $(c+b) \times a$ $d_3^2 = (c+b)^2 + a^2$

✨ 通解规律（不需要每次都计算比较）：

最短路径对应的是"两个较小边之和"作为矩形的一边，"最大边"作为另一边。

即：最短距离 $= \sqrt{(\text{最小边}+\text{中间边})^2 + \text{最大边}^2}$

📝 本题：$a=30, b=50, c=40$，排序后最小30、中间40、最大50 → 最短 $= \sqrt{7400} \approx 86.02$ cm ✓

7. 【数学文化】折竹抵地（源自《九章算术》）：

"今有竹高一丈，末折抵地，去本三尺。问折者高几何？"

意即：一根竹子原高一丈（10尺），被风吹折后，竹梢恰好抵地，抵地处离竹子底部3尺远。求折断后竹子离地面的高度。

图：折竹抵地示意图（原高一丈=10尺，去本三尺）

解：设折断后竹子离地面的高度为 $x$ 尺，则：

  • 竹子直立部分（直角边）：$AB = x$ 尺
  • 竹子折断部分（斜边）：$BC = (10-x)$ 尺
  • 地面距离（另一直角边）：$AC = 3$ 尺

在 Rt$\triangle ABC$ 中，由勾股定理：

$$AB^2 + AC^2 = BC^2$$

$$x^2 + 3^2 = (10-x)^2$$

$$x^2 + 9 = 100 - 20x + x^2$$

$$9 = 100 - 20x$$

$$20x = 91$$

$$x = 4.55$$

答：折断后竹子离地面的高度为 $\mathbf{4.55}$ 尺（即 4尺5寸5分）。

📜 历史背景：此题出自中国古代数学名著《九章算术》勾股章，是我国最早记载的应用勾股定理解决实际问题的经典案例之一，距今已有两千多年历史。

📖 数学文化——勾股定理的全球之旅

中国（约公元前1100年）：《周髀算经》记载了"勾三股四弦五"，赵爽（公元3世纪）用弦图给出了优美的证明。

古巴比伦（约公元前1800年）：泥板文书 Plimpton 322 上记录了大量勾股数组，说明巴比伦人早已掌握了这一规律。

古希腊（约公元前550年）：毕达哥拉斯学派发现了这一定理，传说毕达哥拉斯发现此定理后宰牛庆祝！

印度（约公元前800年）：《绳法经》（Śulba Sūtras）中记录了勾股定理及其在祭坛建筑中的应用。

勾股定理是人类文明史上被独立发现次数最多的数学定理之一，它跨越时空、连接古今，是人类智慧的共同结晶。
