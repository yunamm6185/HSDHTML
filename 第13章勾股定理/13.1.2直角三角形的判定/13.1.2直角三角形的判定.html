第13章 勾股定理

13.1.2 直角三角形的判定

🎯 学习目标

✔理解并掌握勾股定理的逆定理
✔能用勾股定理逆定理判断三角形是否为直角三角形
✔了解勾股数的概念，能识别常见的勾股数
✔能运用勾股定理逆定理解决简单的实际问题

思考：在上一节课中，我们学习了勾股定理：如果 $\triangle ABC$ 是直角三角形，$\angle C = 90°$，那么 $AC^2 + BC^2 = AB^2$。

那么反过来，如果一个三角形的三边长 $a$、$b$、$c$ 满足 $a^2 + b^2 = c^2$，这个三角形一定是直角三角形吗？这就是我们今天要研究的问题——直角三角形的判定。

观察与探究

请分别计算下列三组数中的 $a^2 + b^2$ 和 $c^2$ 的值，看看有什么发现？

组别 $a$ $b$ $c$ $a^2+b^2$ $c^2$
第①组 3 4 5 25 25
第②组 5 12 13 169 169
第③组 6 8 11 100 121

结论：

  • 第①组和第②组满足 $a^2 + b^2 = c^2$
  • 第③组不满足 $a^2 + b^2 = c^2$

通过实际作图验证：当三边满足 $a^2 + b^2 = c^2$ 时，以这三边为边长的三角形是直角三角形！

勾股定理的逆定理

如果三角形的三边长 $a$、$b$、$c$ 满足 $a^2 + b^2 = c^2$，那么这个三角形是直角三角形，且$c$ 边所对的角为直角。

几何语言：在 $\triangle ABC$ 中，若 $AC^2 + BC^2 = AB^2$，则 $\angle C = 90°$。

📐 逆定理的证明（构造法）

已知：如图，在 $\triangle ABC$ 中，$AB = c$，$BC = a$，$AC = b$，且 $a^2 + b^2 = c^2$。

求证：$\angle C = 90°$。

证明：如图 ②，作 $\triangle A'B'C'$，使 $\angle C' = 90°$，$A'C' = b$，$B'C' = a$。

由勾股定理得：$A'B'^2 = a^2 + b^2 = c^2$，即 $A'B' = c$。

在 $\triangle ABC$ 和 $\triangle A'B'C'$ 中：

$\because\ BC = a = B'C'$，

$\quad AC = b = A'C'$，

$\quad AB = c = A'B'$，

$\therefore\ \triangle ABC \cong \triangle A'B'C'\ (\text{SSS})$。

$\therefore\ \angle C = \angle C' = 90°$。

图 13.1.8

💡 证明思路： 要证 $\angle C = 90°$，但 $\triangle ABC$ 不知道有没有直角。所以我们构造一个有直角的 $\triangle A'B'C'$， 让它的两条直角边分别等于 $a$ 和 $b$。由勾股定理算出斜边也是 $c$，这样两个三角形三边对应相等， 由 SSS 全等推出 $\angle C = \angle C' = 90°$。

📌 勾股数

能够成为直角三角形三条边长的正整数称为勾股数（或勾股数组）。

例如：$(3,4,5)$、$(5,12,13)$、$(6,8,10)$、$(7,24,25)$、$(8,15,17)$ 都是勾股数。

注意：如果 $(a,b,c)$ 是一组勾股数，那么 $(ka,kb,kc)$（其中 $k$ 为正整数）也是一组勾股数。

例

已知三角形的三边长分别为 $m^2-1$、$2m$、$m^2+1$（$m > 1$ 为整数），求证这个三角形是直角三角形。

证明：因为 $m > 1$ 且为整数，所以三边长均为正数。

比较三边的大小：由于 $m^2+1 - 2m = (m-1)^2 > 0$（$m \neq 1$），所以 $m^2+1 > 2m$；且显然 $m^2+1 > m^2-1$。因此 $m^2+1$ 是最长边。

计算两较短边的平方和与最长边的平方：

$$(m^2-1)^2 + (2m)^2 = (m^4 - 2m^2 + 1) + 4m^2 = m^4 + 2m^2 + 1$$

$$(m^2+1)^2 = m^4 + 2m^2 + 1$$

所以 $$(m^2-1)^2 + (2m)^2 = (m^2+1)^2$$

由勾股定理的逆定理可知，以这三边为边长的三角形是直角三角形。

补充例

判断以下各组数能否作为直角三角形的三边长：

(1) $9$, $12$, $15$；  (2) $7$, $24$, $25$；  (3) $1.5$, $2$, $2.5$

解：根据勾股定理的逆定理，判断最长边的平方是否等于其他两边的平方和。

(1) $9^2 + 12^2 = 81 + 144 = 225 = 15^2$ ✓

所以 $9$，$12$，$15$ 能构成直角三角形（是 $(3,4,5)$ 的3倍）。

(2) $7^2 + 24^2 = 49 + 576 = 625 = 25^2$ ✓

所以 $7$，$24$，$25$ 能构成直角三角形。

(3) $1.5^2 + 2^2 = 2.25 + 4 = 6.25 = 2.5^2$ ✓

所以 $1.5$，$2$，$2.5$ 能构成直角三角形（是 $(3,4,5)$ 的0.5倍）。
注：勾股定理逆定理对任意正实数都适用，不仅限于整数。

✏️ 练习

1. 设三角形的三边长分别等于下列各组数，试判断各三角形是不是直角三角形。若是，指出哪一条边所对的角是直角。

    (1) $12$，$16$，$20$；    (2) $1.5$，$2$，$2.5$

解：根据勾股定理的逆定理，先确定最长边，再验证两较短边的平方和是否等于最长边的平方。

(1) 三边为 $12$，$16$，$20$。

$12^2 + 16^2 = 144 + 256 = 400$，而 $20^2 = 400$。

所以 $12^2 + 16^2 = 20^2$，满足勾股定理的逆定理。

结论：该三角形是直角三角形，边长 $20$ 所对的角是直角。（注：是 $(3,4,5)$ 的 $4$ 倍）

(2) 三边为 $1.5$，$2$，$2.5$。

$1.5^2 + 2^2 = 2.25 + 4 = 6.25$，而 $2.5^2 = 6.25$。

所以 $1.5^2 + 2^2 = 2.5^2$，满足勾股定理的逆定理。

结论：该三角形是直角三角形，边长 $2.5$ 所对的角是直角。（注：是 $(3,4,5)$ 的 $0.5$ 倍）

2. 若一个三角形的三边 $a$、$b$、$c$ 满足 $\displaystyle \frac{a}{c-b}=\frac{c+b}{a}$，则这个三角形是直角三角形吗？

解：由已知条件 $\displaystyle \frac{a}{c-b}=\frac{c+b}{a}$，交叉相乘得：

$$a^2 = (c-b)(c+b)$$

$$a^2 = c^2 - b^2$$

移项得：$$a^2 + b^2 = c^2$$

这正好满足勾股定理的逆定理的条件（$c$ 为最长边）。

答：这个三角形是直角三角形，且 $c$ 边所对的角为直角。

3. 若一个三角形的三边 $a$、$b$、$c$ 满足 $a^2 + b^2 > c^2$，这个三角形一定是锐角三角形吗？如果满足 $a^2 + b^2 < c^2$ 呢？请举例说明。

答：

若 $a^2+b^2 > c^2$（其中 $c$ 为最长边），则该三角形是锐角三角形（$c$ 边所对的角为锐角）。

反例验证：取三边 $3, 3, 3$（等边三角形），$3^2+3^2=18 > 9=3^2$，每个角都是 $60°$（锐角）✓

若 $a^2 + b^2 < c^2$（其中 $c$ 为最长边），则该三角形是钝角三角形（$c$ 边所对的角为钝角）。

例子：取三边 $2, 3, 4$，其中 $4$ 是最长边。$2^2+3^2=4+9=13 < 16=4^2$，这是钝角三角形 ✓

4. 想一想：有多少种方法可以判定一个三角形是直角三角形？

判定直角三角形的方法主要有以下几种：

  1. 角的定义：有一个内角等于 $90°$（直接测量或证明）
  2. 勾股定理的逆定理：三边 $a,b,c$ 满足 $a^2+b^2=c^2$
  3. 利用已知直角关系：如两直线垂直、直径所对的圆周角等
  4. 特殊图形性质：如矩形的四个角都是直角

在本节中，我们主要学习的是方法②——勾股定理的逆定理，它是通过三边数量关系来判定形状的重要工具。

📖 读一读——勾股数的历史与趣谈

古埃及人的方法：据记载，古埃及人用绳子来构造直角。他们取一根绳子打上等距离的结，将绳子分成12段，然后围成一个三边分别是3段、4段、5段的三角形，这样就能得到一个精确的直角三角形。这种方法被称为"埃及绳法"或"拉绳者"技术。

《九章算术》：我国古代数学名著《九章算术》中就有关于勾股数的记载："今有池方一丈，葭生其中央，出水一尺。引葭赴岸，适与岸齐。问水深、葭长各几何？"这道经典问题就用到了勾股定理及其逆定理的思想。

📌 课堂小结

• 勾股定理的逆定理：若 $a^2+b^2=c^2$，则该三角形是以 $c$ 为斜边的直角三角形。
• 勾股数：满足 $a^2+b^2=c^2$ 的正整数 $(a,b,c)$ 称为勾股数。
• 用逆定理判定直角三角形时，先确定最长边 $c$，再验证 $a^2+b^2=c^2$ 是否成立。
• 勾股定理与其逆定理互为逆命题，前者用于"算边"，后者用于"定形"。
