📚 复习回顾——矩形的两个重要性质

上节课我们学习了矩形的两条基本性质，它们是解决今天例题的关键工具：

🔷 性质定理1

矩形的四个角都是直角。

若四边形ABCD是矩形，则
$\angle A = \angle B = \angle C = \angle D = 90°$

🔶 性质定理2

矩形的对角线相等。

若四边形ABCD是矩形，则
对角线 $AC = BD$

💎 重要推论：由于平行四边形的对角线互相平分，结合矩形对角线相等的性质可得：
矩形的对角线相等且互相平分。即 $OA = OB = OC = OD = \frac{1}{2}AC$

📝 例题2：利用面积法求垂线段长

【问题】如图，在矩形ABCD中，BE ⊥ AC 于点E，已知 AC = 6，BC = 4，求 BE 的长。

图1：例题2图形示意图

🔍 分析思路

第一步：观察已知条件，确定解题方向

已知：矩形ABCD中，AC = 6，BC = 4

目标：求 BE 的长（BE 是点 B 到对角线 AC 的距离）

💡 关键：要求垂线段 BE，可以考虑用等面积法

第二步：先求出矩形的另一条边 AB

∵ 四边形 ABCD 是矩形，∴ $\angle ABC = 90°$

在 Rt△ABC 中，由勾股定理：

$AB^2 + BC^2 = AC^2$

$AB^2 + 4^2 = 6^2$

$AB^2 + 16 = 36$

$AB^2 = 20$，∴ $AB = \sqrt{20} = 2\sqrt{5}$

第三步：利用面积法建立等式

△ABC 的面积可以用两种方式表示：

方式一：以 BC 为底，AB 为高
$S_{\triangle ABC} = \frac{1}{2} \times AB \times BC$

方式二：以 AC 为底，BE 为高（∵ BE ⊥ AC）
$S_{\triangle ABC} = \frac{1}{2} \times AC \times BE$

∵ 面积相等，∴ $\frac{1}{2} \times AB \times BC = \frac{1}{2} \times AC \times BE$

第四步：代入数据求解

$AB \times BC = AC \times BE$

$2\sqrt{5} \times 4 = 6 \times BE$

$8\sqrt{5} = 6BE$

$BE = \frac{8\sqrt{5}}{6} = \frac{4\sqrt{5}}{3}$

✅ 答案：$BE = \frac{4\sqrt{5}}{3}$

【完整解答】

解：∵ 四边形ABCD是矩形

∴ $\angle ABC = 90°$ （矩形的四个角都是直角）

在Rt△ABC中，根据勾股定理：

$AB = \sqrt{AC^2 - BC^2} = \sqrt{6^2 - 4^2} = \sqrt{36 - 16} = \sqrt{20} = 2\sqrt{5}$

∵ $S_{\triangle ABC} = \frac{1}{2} \times AB \times BC = \frac{1}{2} \times AC \times BE$

∴ $AB \times BC = AC \times BE$

$2\sqrt{5} \times 4 = 6 \times BE$

$BE = \frac{8\sqrt{5}}{6} = \frac{4\sqrt{5}}{3}$

答：BE的长为 $\frac{4\sqrt{5}}{3}$。

🔑 面积法（等面积法）的核心思想：

同一个三角形的面积可以用不同的"底×高÷2"来表示，由此建立等式求解未知量。

适用场景：求三角形一边上的高或点到直线的距离时，当直接计算困难时，可尝试用面积法。

关键公式：$S = \frac{1}{2}a \cdot h_a = \frac{1}{2}b \cdot h_b$   →   $a \cdot h_a = b \cdot h_b$

📝 例题3：综合运用矩形性质

【问题】如图，在矩形ABCD中，对角线AC、BD相交于点O，EF 垂直且平分线段 OA，垂足为 E，EF = $\frac{7}{2}$，求 CE 的长。

图2：例题3图形示意图

🔍 分析思路

第一步：分析已知条件中的隐含信息

已知：矩形ABCD → 对角线 AC、BD 相等且互相平分于 O 点

→ $OA = OC = \frac{1}{2}AC$，$OB = OD = \frac{1}{2}BD$，且 $OA = OC = OB = OD$

又知：EF 垂直且平分 OA，E为垂足，EF = $\frac{7}{2}$

💡 垂直平分线的性质：垂直平分线上的任意一点到线段两端点的距离相等

第二步：利用垂直平分线的性质确定 F 点位置

∵ EF 垂直平分 OA，F 在 EF 上

∴ 根据垂直平分线的性质：FA = FO

即 △AOF 是等腰三角形，F 到 A、O 两点的距离相等

设 OE = EA = x，则 OA = 2x

第三步：建立方程关系

在 Rt△OEF 中（∵ EF ⊥ OA）：

$OF^2 = OE^2 + EF^2$

又 ∵ FA = OF（垂直平分线性质），且 FA = OA - ? 等等...

⚠️ 注意：需要明确 F 点的位置！F 在 OA 的哪一侧？

从图中可以看出，F 在矩形内部，EF 与 AC 交于 E

设 $OE = x$，则 $EA = x$，$OA = 2x$

在 Rt△OEF 中：$OF = \sqrt{x^2 + (\frac{7}{2})^2} = \sqrt{x^2 + \frac{49}{4}}$

而 FA = OF，同时 FA 可以通过坐标表示...

第四步：完整推导过程

设 OE = EA = x，则 OA = 2x

由勾股定理：$OF = \sqrt{OE^2 + EF^2} = \sqrt{x^2 + (\frac{7}{2})^2} = \sqrt{x^2 + \frac{49}{4}}$

∵ FA = OF，且 F 在 OA 上方，E 在 OA 上

考虑 F 在矩形内部，通过坐标系分析可知：设 O 为原点，C 在正 x 轴方向

则 E 的坐标为 (-x, 0)，F 的坐标为 (-x, 7/2)

$OF^2 = x^2 + \frac{49}{4}$，$AF^2 = (2x - x)^2 + (\frac{7}{2})^2 = x^2 + \frac{49}{4}$ ✓

C 的坐标为 (2x, 0)，所以 CE = $\sqrt{(2x+x)^2 + (0-\frac{7}{2})^2}$ ...

💡 实际上，此题需要结合具体数值条件来确定 x 的值

【完整解答】

解：∵ 四边形ABCD是矩形，对角线AC、BD交于点O

∴ $OA = OC$ （平行四边形对角线互相平分）

∴ $OA = OC = OB = OD$ （矩形对角线相等）

设 $OA = OC = a$

∵ EF 垂直且平分线段 OA，垂足为E

∴ $OE = EA = \frac{1}{2}OA = \frac{a}{2}$

∵ F 在 OA 的垂直平分线上

∴ $FA = FO$ （垂直平分线上的点到两端距离相等）

在 Rt△OEF 中，由勾股定理：

$FO^2 = OE^2 + EF^2$

$FO^2 = (\frac{a}{2})^2 + (\frac{7}{2})^2 = \frac{a^2}{4} + \frac{49}{4}$

∵ $FA = FO$，设 C 为原点向右的方向，O 在 CA 上

以 O 为原点，OC 方向为 x 轴正方向建立坐标系

则 $C(a, 0)$，$E(-\frac{a}{2}, 0)$，$F(-\frac{a}{2}, \frac{7}{2})$

$$CE = \sqrt{(a - (-\frac{a}{2}))^2 + (0 - \frac{7}{2})^2} = \sqrt{(\frac{3a}{2})^2 + \frac{49}{4}} = \sqrt{\frac{9a^2+49}{4}} = \frac{\sqrt{9a^2+49}}{2}$$

答：CE的长为 $\frac{\sqrt{9a^2+49}}{2}$（其中 a = $\frac{1}{2}AC$）。若给出AC的具体值，即可求得CE的确切数值。

💡 数学思想方法总结

  • 转化思想：将求垂线段问题转化为三角形面积问题；将复杂的几何关系转化为代数方程
  • 方程思想：通过设未知数，利用勾股定理建立方程，把几何量的计算转化为代数运算
  • 面积法：同一三角形的面积可用不同的底和高表示，从而建立等式求解未知量
  • 数形结合：借助坐标系将几何问题转化为代数计算，直观明了
  • 整体思想：利用矩形整体性质（对角线相等且平分），避免繁琐的局部证明

✏️ 课堂练习

练习1：在矩形ABCD中，AB = 5，BC = 12，求对角线AC的长以及AC边上的高（B到AC的距离）。

解：

(1) 由勾股定理：$AC = \sqrt{AB^2 + BC^2} = \sqrt{5^2 + 12^2} = \sqrt{169} = 13$

(2) 设 B 到 AC 的距离为 h，由面积法：

$\frac{1}{2} \times AB \times BC = \frac{1}{2} \times AC \times h$

$5 \times 12 = 13h$

$h = \frac{60}{13}$

答：AC = 13，AC边上的高为 $\frac{60}{13}$。

练习2：如图，矩形ABCD的对角线AC、BD相交于点O，AB = 6，AC = 10，过点O作 OE ⊥ AB 于 E，求 OE 的长。

解：

∵ 矩形ABCD的对角线AC = 10

∴ 对角线互相平分，$OA = OC = \frac{1}{2}AC = 5$

∵ AB = 6，$\angle ABC = 90°$

∴ $BC = \sqrt{AC^2 - AB^2} = \sqrt{100 - 36} = \sqrt{64} = 8$

∵ O 是 AC 中点，OE ⊥ AB，且 AB ∥ CD

∴ E 是 AB 的中点（O到两平行线距离相等）

$OE = \frac{1}{2}BC = \frac{1}{2} \times 8 = 4$

答：OE 的长为 4。

练习3（拓展）：在矩形ABCD中，M是BC的中点，MC = 3，MA = 5，求矩形ABCD的周长和面积。

解：

∵ M 是 BC 的中点，MC = 3

∴ BM = MC = 3，$BC = 2MC = 6$

在 Rt△ABM 中（∠B = 90°）：$AM^2 = AB^2 + BM^2$

$5^2 = AB^2 + 3^2$

$25 = AB^2 + 9$

$AB^2 = 16$，∴ $AB = 4$

矩形周长 = $2(AB + BC) = 2(4 + 6) = 20$

矩形面积 = $AB \times BC = 4 \times 6 = 24$

答：矩形ABCD的周长为20，面积为24。

📝 课堂小结

  • 矩形性质的灵活运用：矩形的四个角都是直角（用于构造直角三角形）；对角线相等且互相平分（$OA=OB=OC=OD$）
  • 面积法（核心方法）：同一三角形的面积等于任意底与对应高的乘积的一半，即 $S = \frac{1}{2}ah$。利用这一性质可以建立等式求高或底边
  • 垂直平分线的应用：垂直平分线上的任意一点到线段两端点的距离相等，这是连接代数与几何的重要桥梁
  • 勾股定理的作用：在含直角的图形中，勾股定理是计算边长的有力工具
  • 综合解题策略：①审清题意找条件 → ②联想性质建联系 → ③选择方法列算式 → ④严谨计算得结果
