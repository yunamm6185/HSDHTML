第十八章 矩形、菱形与正方形 18.1 矩形

18.1.2 矩形的判定

第 3 课时：直角三角形斜边上的中线
学习目标

本节课的学习目标：

1. 探索并掌握直角三角形斜边上的中线等于斜边的一半这一性质定理。

2. 能运用该性质定理进行简单的计算与证明。

3. 了解该定理的逆命题及其应用。

思考

我们已经知道，矩形的两条对角线相等且互相平分，如图 18.1.14① 所示。在矩形 $ABCD$ 中，$AC = BD$，$AO = OC$，$BO = OD$。

若擦去半个矩形，如图 18.1.14②，$BO$ 即是 Rt$\triangle ABC$ 斜边 $AC$ 上的中线，由此，你能发现 $BO$ 与斜边 $AC$ 的关系吗？

例题精讲

例 7

如图 18.1.15，在 Rt$\triangle ABC$ 中，$BO$ 为斜边 $AC$ 上的中线。

求证：$BO = \dfrac{1}{2}AC$。

将 $\triangle ABC$ 补成矩形，即可得到要证的结论。

证明：如图 18.1.16，延长 $BO$ 至点 $D$，使 $OD = OB$，连结 $AD$ 和 $CD$。

在四边形 $ABCD$ 中，

$\because OA = OC$，$OB = OD$，

$\therefore$ 四边形 $ABCD$ 是平行四边形。

又 $\because \angle ABC = 90^\circ$，

$\therefore$ 四边形 $ABCD$ 是矩形。

从而 $AC = BD$，$BO = \dfrac{1}{2}BD = \dfrac{1}{2}AC$。

定理
直角三角形斜边上的中线等于斜边的一半。

即：在 Rt$\triangle ABC$ 中，$\angle ABC = 90^\circ$，$O$ 是 $AC$ 的中点，则 $BO = \dfrac{1}{2}AC$。

试一试

写出上述结论的逆命题，观察图 18.1.17，试判断该逆命题是否成立。

逆命题：如果一个三角形一边上的中线等于该边的一半，那么这个三角形是直角三角形。

图 18.1.17 中，$CD = AD = BD$。即边 $AB$ 上的中线 $CD$ 将整个三角形分成了两个等腰三角形，利用等腰三角形两底角相等的性质，容易证明 $\angle ACD$ 与 $\angle BCD$ 的和为 $90^\circ$，即该三角形确实是一个直角三角形。

于是有：

如果一个三角形一边上的中线等于该边的一半，那么这个三角形是一个直角三角形。

练习

1. 如图，在 Rt$\triangle ABC$ 中，$\angle ACB = 90^\circ$，点 $D$ 为斜边 $AB$ 的中点，$AC = 6$，$BC = 8$，求 $CD$ 的长。

解：$\because \angle ACB = 90^\circ$，$AC = 6$，$BC = 8$，

$\therefore AB = \sqrt{AC^2 + BC^2} = \sqrt{6^2 + 8^2} = \sqrt{36 + 64} = \sqrt{100} = 10$。

$\because$ 点 $D$ 为斜边 $AB$ 的中点，

$\therefore CD = \dfrac{1}{2}AB = \dfrac{1}{2} \times 10 = 5$。

2. 在 Rt$\triangle ABC$ 中，$\angle C = 90^\circ$，$AB = 2AC$，求 $\angle A$ 和 $\angle B$ 的度数。

解：$\because \angle C = 90^\circ$，点 $C$ 为直角顶点，设 $D$ 为 $AB$ 的中点，

则 $CD = \dfrac{1}{2}AB = AD = BD$（直角三角形斜边中线等于斜边一半）。

$\because AB = 2AC$，$\therefore CD = AC = AD$。

$\therefore \triangle ACD$ 为等边三角形，$\angle A = 60^\circ$。

$\therefore \angle B = 90^\circ - 60^\circ = 30^\circ$。

课堂小结
  • 直角三角形斜边上的中线等于斜边的一半：$BO = \dfrac{1}{2}AC$。
  • 证明方法：将直角三角形补成矩形，利用矩形对角线相等且互相平分的性质。
  • 逆命题也成立：若三角形一边上的中线等于该边的一半，则该三角形为直角三角形。
核心素养

📐 数学素养提升点

逻辑推理：通过补形法（将直角三角形补成矩形）证明斜边中线定理，体会转化思想。

数学运算：运用定理进行直角三角形中的线段长度计算。

直观想象：从矩形的对角线性质中"擦去"一半，直观发现直角三角形斜边中线的性质。

矩形与直角三角形，看似不同，却有深刻的联系。当你学会"补形"与"分割"，你就掌握了几何证明的一把钥匙！
