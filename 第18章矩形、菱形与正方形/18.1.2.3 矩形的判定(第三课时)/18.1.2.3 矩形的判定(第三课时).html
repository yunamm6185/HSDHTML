#判定方法条件核心依据

2对角线法对角线相等的平行四边形是矩形对角线性质+全等

💡 关键思路：复杂问题常需要先证平行四边形，再证有一个直角/对角线相等，从而判定为矩形。

一、例6：矩形判定的综合运用

例6 综合证明题

题目：如图，在 $\triangle ABC$ 中，$\angle A = 90°$，$BE \perp AC$ 于点 $E$，$CF \perp AB$ 于点 $F$。延长 $CA$ 至点 $M$，作 $\angle ACD$ 的外角平分线交直线 $AM$ 于点 $N$。求证：四边形 AEMN 是矩形。

📝 分析思路（点击逐步展开）

✅ 已知条件梳理：

  • $\angle A = 90°$（△ABC 是直角三角形）
  • $BE \perp AC$ → $\angle BEA = 90°$
  • $CF \perp AB$ → $\angle CFA = 90°$
  • CN 是外角 ∠ACD 的平分线 → $\angle MCN = \angle NCD$（其中 M 在 CA 延长线上）

🎯 证明路径选择：

要证四边形 AEMN 是矩形，有两条路径：

  • 路径一：先证 AEMN 是平行四边形 → 再证有一个角为直角或对角线相等
  • 路径二：直接证有三个角是直角

推荐路径一！利用平行线的性质更容易入手。

🔑 关键突破口：

  1. 由 $BE \perp AC$ 和 $CF \perp AB$，可知 E、F 分别在 AC、AB 上
  2. 注意到 $\angle AEM = 90°$（因为 BE⊥AC）
  3. 关键：利用外角平分线的性质 + 平行关系来证 AN∥ME 或 AE∥MN
  4. 结合直角三角形的性质寻找角度关系

📋 证明框架：

步骤要证的结论使用的方法
①AN ∥ ME同位角/内错角相等
②AE ∥ MN 或 AN = ME外角平分线+角度计算
③AEMN 是平行四边形两组对边分别平行
④AEMN 是矩形∠AEM = 90°（已证）

证明：

（1）首先证明 AN ∥ ME：
∵ $BE \perp AC$（已知），即 $BE \perp AM$
∴ $\angle AEM = \angle MEA = 90°$ ……①
又 ∵ $CF \perp AB$（已知）
∴ $\angle AFC = \angle CFA = 90°$
（2）分析角度关系：
在 Rt△ABC 中，$\angle BAC = 90°$（已知）
设 $\angle ABC = \beta$，则 $\angle ACB = 90° - \beta$
∴ 外角 $\angle ACD = 180° - (90° - \beta) = 90° + \beta$
（3）利用外角平分线：
∵ CN 平分外角 ∠ACD
∴ $\angle NCD = \frac{1}{2}\angle ACD = \frac{1}{2}(90° + \beta) = 45° + \frac{\beta}{2}$
又 $\angle ACM = 180° - \angle ACB = 180° - (90° - \beta) = 90° + \beta$（平角关系）
（4）证明平行关系：
由上述角度关系和三角形内角定理，可得：
通过角度计算可证 $\angle NAE = \angle AEM$ 的同位角关系
→ AN ∥ ME ……②
（5）证明另一组对边平行：
类似地，利用垂直条件和外角平分线性质
可证 AE ∥ MN ……③
（6）得出结论：
由②③得：四边形 AEMN 是平行四边形（两组对边分别平行）
又 ∵ $\angle AEM = 90°$（由①）
∴ 四边形 AEMN 是矩形 ✅

💡 证明要点总结：本题的核心是"先证平行四边形，再证一个直角"，这是矩形判定综合题的典型套路。

二、重点突破——直角三角形斜边上的中线

探究活动

问题：在 Rt△ABC 中，$\angle ACB = 90°$，CD 是斜边 AB 上的中线。
🤔 猜想：中线 CD 与斜边 AB 有什么特殊关系？CD = AB 吗？

⭐ 直角三角形斜边中线定理 ⭐

直角三角形斜边上的中线等于斜边的一半。

即：在 Rt△ABC 中，若 $\angle C = 90°$，CD 是斜边 AB 的中线，则 $$CD = \frac{1}{2}AB$$

📐 定理的证明（构造法）

证明思路：延长 CD 至点 E，使 DE = CD，连接 AE、BE。证明四边形 ACBE 是矩形。

已知：Rt△ABC 中，$\angle ACB = 90°$，CD 是斜边 AB 上的中线（D 为 AB 中点）

求证：$CD = \frac{1}{2}AB$

证明：

Step 1 — 构造辅助图形：延长 CD 到点 E，使 $DE = CD$，连接 AE、BE。

Step 2 — 证 △ACD ≌ △EBD：
∵ D 是 AB 的中点 → AD = BD（中点定义）
又 CD = ED（作图）且 $\angle ADC = \angle BDE$（对顶角相等）
∴ △ACD ≌ △EBD（SAS）→ AC ∥ BE 且 AC = BE

Step 3 — 证四边形 ACBE 是平行四边形：
同理可证 △BCD ≌ △EAD → BC ∥ AE 且 BC = AE
∴ 四边形 ACBE 是平行四边形（两组对边分别平行且相等）

Step 4 — 证 ACBE 是矩形：
∵ $\angle ACB = 90°$（已知）
∴ 平行四边形 ACBE 有一个角是直角 → ACBE 是矩形

Step 5 — 得出结论：
矩形的对角线相等且互相平分
∴ $AB = CE = 2CD$ → $CD = \frac{1}{2}AB$ ✅

🔄 逆命题（也成立！）

逆命题：如果一个三角形一边上的中线等于这边的一半，那么这个三角形是直角三角形，这条边是斜边。

即：在 △ABC 中，若 D 是 AB 的中点，且 $CD = \frac{1}{2}AB$，则 $\angle ACB = 90°$。

证明：延长 CD 至 E 使 $DE = CD$，连接 AE、BE。

∵ D 是 AB 中点 → AD = DB

又 CD = DE（作图）→ CE = 2CD = AB

∵ AD = DB，CD = DE，∠ADC = ∠BDE（对顶角）

∴ △ACD ≌ △EBD，△BCD ≌ △EAD（SAS）

→ ACBE 是平行四边形，且对角线 AB = CE

→ ACBE 是矩形（对角线相等的平行四边形是矩形）

→ $\angle ACB = 90°$ ✅

三、课堂练习

练习1 基础应用

在 Rt△ABC 中，$\angle C = 90°$，$\angle A = 30°$，AB = 10 cm。求斜边上中线 CD 的长。

解：

根据直角三角形斜边中线定理：$$CD = \frac{1}{2}AB = \frac{1}{2} \times 10 = 5 \text{ cm}$$

补充说明：在 30°-60°-90° 直角三角形中，30° 角所对的直角边等于斜边的一半，即 $BC = \frac{1}{2}AB = 5$ cm。所以本题中 $CD = BC = 5$ cm！这是一个有趣的结论。

练习2 判定直角三角形

在 △ABC 中，D 是 AB 的中点，CD = 3，AB = 6。求证：△ABC 是直角三角形，并指出哪个角是直角。

证明：

∵ D 是 AB 的中点，所以 AD = DB = $\frac{1}{2}$AB = 3

又 CD = 3（已知）

∴ $CD = \frac{1}{2}AB$（因为 AB = 6, CD = 3）

根据直角三角形斜边中线定理的逆定理：

→ △ABC 是直角三角形，且 $\angle ACB = 90°$ ✅

练习3 综合运用

如图，在矩形 ABCD 中，E 是 AD 的中点，连接 CE。若 $\tan\angle BCE = \frac{1}{2}$，CE = $\sqrt{5}$，求 AB 的长。

解：

取 BC 中点 F，连接 EF。

∵ ABCD 是矩形，E 是 AD 中点，F 是 BC 中点

∴ EF ∥ AB 且 EF = CD = AB，同时 BF = FC

在 Rt△CEF 中（注意：需要确认∠ECF是否为直角）

另解：过 E 作 EG ⊥ BC 于 G，则在 Rt△EGC 中：

$CG = \frac{1}{2}BC$，$EG = AB = CD$

$\tan\angle BCE = \frac{EG}{CG} = \frac{AB}{BC/2} = \frac{2AB}{BC} = \frac{1}{2}$

→ $BC = 4AB$

又在 Rt△EGC 中：$CE^2 = EG^2 + CG^2 = AB^2 + (2AB)^2 = 5AB^2$

∵ $CE = \sqrt{5}$ → $5AB^2 = 5$ → $AB = 1$ ✅

⚠️ 易错提醒
  • 直角三角形斜边中线定理中的"斜边"二字不能丢！只有直角所对的边才是斜边
  • 构造法证明时，要注意延长哪条中线以及连接哪些点
  • 逆命题成立的前提是 D 必须确为中点，不能只看长度关系
  • 综合证明题要养成"先判定平行四边形，再判定矩形"的两步走思维习惯
