第十八章 矩形、菱形与正方形 18.1 矩形

18.1.2 矩形的判定

第 2 课时：判定定理的应用（例5~例6）
学习目标

本节课的学习目标：

1. 熟练运用矩形的三个判定方法（定义法、角判定法、对角线判定法）解综合题。

2. 掌握复杂图形中证明四边形为矩形的推理方法。

知识铺垫 / 复习回顾

矩形的判定方法

① 定义法：有一个角是直角的平行四边形是矩形。

② 角判定法（判定定理1）：有三个角是直角的四边形是矩形。

③ 对角线判定法（判定定理2）：对角线相等的平行四边形是矩形。

例题解析

例 5

如图 18.1.12，四边形 $ABCD$ 是由两个全等的正三角形 $ABD$ 和 $BCD$ 组成的，$M$、$N$ 分别为 $BC$、$AD$ 的中点。

求证：四边形 $BMDN$ 是矩形。

证明：$\because \triangle ABD$ 和 $\triangle BCD$ 是全等的正三角形，

$\therefore \angle ADB = \angle CDB = 60^\circ$。

又 $\because M$、$N$ 分别为 $BC$、$AD$ 的中点，

$\therefore BN \perp AD$，$DM \perp BC$，$\angle BDM = \frac{1}{2}\angle BDC = 30^\circ$。

$\therefore \angle DNB = \angle DMB = 90^\circ$，

$\angle MDN = \angle ADB + \angle BDM = 90^\circ$。

$\therefore$ 四边形 $BMDN$ 是矩形（有三个角是直角的四边形是矩形）。

例 6

如图 18.1.13，在 $\triangle ABC$ 中，$AB = AC$，$AD \perp BC$，垂足为点 $D$，$AG$ 是 $\triangle ABC$ 的外角 $\angle FAC$ 的平分线，$DE \parallel AB$，交 $AG$ 于点 $E$。

求证：四边形 $ADCE$ 是矩形。

证明：$\because AB = AC$，$AD \perp BC$，

$\therefore \angle B = \angle ACB$，$BD = DC$。

又 $\because AE$ 是 $\triangle ABC$ 的外角 $\angle CAF$ 的平分线，

$\therefore \angle FAE = \frac{1}{2}\angle CAF = \frac{1}{2}(\angle B + \angle ACB) = \angle B$。

$\therefore AE \parallel BC$。

又 $\because DE \parallel AB$，

$\therefore$ 四边形 $ABDE$ 是平行四边形。

$\therefore AE = BD$，$AB = DE$。

$\therefore AC = DE$，$AE = DC$。

又 $\because AE \parallel DC$，

$\therefore$ 四边形 $ADCE$ 是平行四边形。

$\therefore$ 四边形 $ADCE$ 是矩形（对角线相等的平行四边形是矩形）。

练习

1. 如图，$AD$、$AE$ 分别是 $\triangle ABC$ 的内角 $\angle BAC$ 和外角 $\angle BAF$ 的平分线，$BE \perp AE$，$DA \perp BC$。
求证：四边形 $AEBD$ 是矩形。

证明：$\because AD$ 平分 $\angle BAC$，$AE$ 平分 $\angle BAF$，

$\therefore \angle DAB = \frac{1}{2}\angle BAC$，$\angle BAE = \frac{1}{2}\angle BAF$。

$\therefore \angle DAE = \angle DAB + \angle BAE = \frac{1}{2}(\angle BAC + \angle BAF) = \frac{1}{2} \times 180^\circ = 90^\circ$。

$\because BE \perp AE$，$DA \perp BC$，

$\therefore \angle AEB = 90^\circ$，$\angle ADB = 90^\circ$。

$\therefore$ 四边形 $AEBD$ 是矩形（有三个角是直角的四边形是矩形）。

2. 一个四边形满足：它的每个顶点到其他三个顶点的距离之和相等。试证明该四边形为矩形。

证明：设四边形为 $ABCD$，由题意得

$AB + AC + AD = BA + BC + BD = CA + CB + CD = DA + DB + DC$。

由 $AB + AC + AD = BA + BC + BD$，得 $AC + AD = BC + BD$。$\cdots$①

由 $CA + CB + CD = DA + DB + DC$，得 $CA + CB = DA + DB$。$\cdots$②

①$-$②，得 $AD - CB = BC - AD$，$\therefore AD = BC$。

同理可证 $AB = CD$，$AC = BD$。

$\therefore$ 四边形 $ABCD$ 是平行四边形且对角线相等，

$\therefore$ 四边形 $ABCD$ 是矩形。

3. 如图，将 $\square ABCD$ 的边 $DC$ 延长到点 $E$，使 $CE = DC$，连结 $AE$，交 $BC$ 于点 $F$，$\angle AFC = 2\angle D$，连结 $AC$、$BE$。
求证：四边形 $ABEC$ 是矩形。

证明：$\because$ 四边形 $ABCD$ 是平行四边形，

$\therefore AB \parallel DC$，$AB = DC$，$\angle ABC = \angle D$。

$\because CE = DC$，$\therefore AB = CE$。

又 $\because AB \parallel CE$，

$\therefore$ 四边形 $ABEC$ 是平行四边形（一组对边平行且相等）。

$\because \angle AFC = \angle ABC + \angle BAF = 2\angle D = 2\angle ABC$，

$\therefore \angle ABC = \angle BAF$。

$\therefore FA = FB$。

又 $\because AB \parallel CE$，$BC$ 与 $AE$ 交于 $F$，

$\therefore \triangle ABF \cong \triangle ECF$（ASA），$\therefore AF = EF$，$BF = CF$。

$\therefore AE = 2AF$，$BC = 2BF = 2AF$，$\therefore AE = BC$。

$\therefore$ 四边形 $ABEC$ 是矩形（对角线相等的平行四边形是矩形）。

课堂小结
  • 巩固了矩形的三种判定方法：定义法、角判定法、对角线判定法。
  • 学会在复杂图形中灵活选择合适的判定方法。
  • 掌握"先证平行四边形，再证矩形"的典型解题路径。
