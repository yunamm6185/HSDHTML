一、探究活动——对角线相等的平行四边形

问题思考

我们知道：矩形的对角线相等（性质）。那么反过来：

如果一个平行四边形的对角线相等，它是矩形吗？

👆 拖动滑块观察：保持对角线相等时，平行四边形的内角如何变化？

40

分析：

  • 当平行四边形的对角线不相等时，它是一个普通的斜平行四边形，四个角都不是直角
  • 当对角线相等时（即偏移量为0），这个平行四边形的四个角都变成了直角！
  • 因此：对角线相等的平行四边形一定是矩形 ✅

三、矩形的判定定理2

⬛ 判定定理2：对角线相等的平行四边形是矩形。

∵ 四边形ABCD是平行四边形 且 AC = BD ∴ ▱ABCD 是矩形

已知：在$\square ABCD$中，AC = BD

求证：$\square ABCD$ 是矩形

证明：

∵ $\square ABCD$ → AB = CD（平行四边形对边相等）

又∵ AC = BD（已知条件），AD = DA（公共边）

∴ 在$\triangle ABC$和$\triangle DCB$中：

$$\begin{cases} AB = DC \\ AC = DB \\ BC = CB \end{cases}$$

∴ $\triangle ABC \cong \triangle DCB$（SSS）

∴ $\angle ABC = \angle DCB$

又∵ AB∥CD → $\angle ABC + \angle DCB = 180°$（同旁内角互补）

∴ $\angle ABC = \angle DCB = 90°$ ✅

∴ 平行四边形ABCD是矩形（有一个角是直角的平行四边形是矩形）

四、实际应用——木工师傅的智慧

🔨 木工测量对角线检验矩形

场景说明：木工师傅制作窗框或画框时，先钉成一个平行四边形框架， 然后测量两条对角线的长度。
📏 若两条对角线长度相等 → 说明这个框是矩形 ✓
📏 若两条对角线长度不等 → 说明这个框不是矩形 ✗

💡 原理：利用了"对角线相等的平行四边形是矩形"这一判定定理。这是判定定理2在生活中最直接的应用！

五、例题精讲

例4 矩形内部小四边形

题目：点O是矩形ABCD的对角线AC与BD的交点，E、F、G、H分别是OA、OB、OC、OD上的点，且OE=OF=OG=OH。求证：四边形EFGH是矩形。

【分析】

要证四边形EFGH是矩形，可分两步走：

  1. 先证EFGH是平行四边形（利用对角线互相平分）
  2. 再证其对角线相等，从而得矩形

【证明】

第一步：证EFGH是平行四边形

∵ O是矩形ABCD的对角线交点 → OA = OC，OB = OD（矩形对角线互相平分且相等）

又∵ OE = OG，OF = OH（已知）

∴ 对角线EG和FH都经过O且被O平分

∴ 四边形EFGH是平行四边形（对角线互相平分的四边形是平行四边形）

第二步：证EFGH是矩形

连接EG、FH作为平行四边形EFGH的两条对角线

EG = OE + OG = 2OE

FH = OF + OH = 2OF

又∵ OE = OF（已知）→ EG = FH

∴ 平行四边形EFGH的对角线相等

∴ 四边形EFGH是矩形（对角线相等的平行四边形是矩形）✅

例5 正三角形组合图形

题目：四边形ABCD是由两个全等的正三角形△ABC和△DBC组成的，E、F分别为AB、DC的中点。求证：四边形AFDE是矩形。

【分析】

由两个全等正三角形组成的四边形ABCD是一个菱形（四条边都相等），要证AFDE为矩形，思路：

  1. 先证AFDE是平行四边形
  2. 再证其对角线相等

【证明】

第一步：分析四边形ABCD的性质

∵ △ABC和△DBC都是正三角形且全等

∴ AB = BC = CD = DA（正三角形的边长相等）

∴ 四边形ABCD是菱形

第二步：证AFDE是平行四边形

∵ E是AB的中点，F是DC的中点

∵ AB∥DC（菱形对边平行）且 AB = DC

∴ AE = $\frac{1}{2}$AB, DF = $\frac{1}{2}$DC → AE = DF 且 AE∥DF

∴ 四边形AFDE是平行四边形（一组对边平行且相等）

第三步：证AFDE是矩形

连接对角线AD和EF

AD = 正三角形边长（已知）

EF = AD（可通过坐标或全等证明：EF连接的是两边中点，且由于对称性EF=AD）

更严谨地：设正三角形边长为a
- AD = a
- EF：取坐标系，A(0,$\frac{\sqrt{3}}{2}$a), B(-$\frac{a}{2}$,0), C(0,-$\frac{\sqrt{3}}{2}$a), D($\frac{a}{2}$,0)
- E为AB中点：E(-$\frac{a}{4}$, $\frac{\sqrt{3}}{4}$a)
- F为DC中点：F($\frac{a}{4}$, -$\frac{\sqrt{3}}{4}$a)
- EF² = ($\frac{a}{2}$)² + ($\frac{\sqrt{3}}{2}$a)² = $\frac{a^2}{4}$ + $\frac{3a^2}{4}$ = a² → EF = a

∴ AD = EF，即平行四边形AFDE的对角线相等

∴ 四边形AFDE是矩形 ✅

六、课堂练习

练习1 选择题

下列条件中，能判定一个平行四边形是矩形的是（    ）

A) 对角线互相垂直    B) 对角线相等    C) 邻角相等    D) 对角互补

答案：B

解析：

  • A) 对角线互相垂直 → 这是菱形的判定条件 ❌
  • B) 对角线相等 → 这正是本节课学的判定定理2 ✅
  • C) 邻角相等 → 虽然也能推出直角，但不是最直接的判定方式 ⚠️
  • D) 对角互补 → 平行四边形本来就满足此条件，不能判定矩形 ❌

练习2 证明题

在$\square ABCD$中，AC、BD是对角线。若AC = BD，且E是BC的中点。求证：$\triangle ABE$是等腰直角三角形。

提示：

  1. 先用判定定理2证明ABCD是矩形 → ∠ABC = 90°
  2. 再利用矩形性质和E为中点的条件，证明AE = BE

简证：

∵ $\square ABCD$ 且 AC = BD → ABCD是矩形 → ∠ABC = 90° → $\triangle ABE$是直角三角形

在Rt$\triangle ABC$中，E是斜边BC的中点

∴ AE = BE = $\frac{1}{2}$BC（直角三角形斜边中线等于斜边的一半）

∴ $\triangle ABE$是等腰直角三角形 ✅

练习3 综合应用

如图，在$\square ABCD$中，以AD、BC为边向形外作等边三角形ADE和BCF。若AC = BD，求证：E、C、F三点共线。

解答：

∵ $\square ABCD$ 且 AC = BD → ABCD是矩形 → ∠BCD = 90°

∵ △BCF是等边三角形 → ∠BCF = 60°

∴ ∠DCF = ∠BCD + ∠BCF = 90° + 60° = 150°

同理，∵ △ADE是等边三角形 → ∠ADE = 60°

∵ ABCD是矩形 → ∠ADC = 90°

∴ ∠EDC = ∠ADC + ∠ADE = 90° + 60° = 150°

∵ AB = CD（矩形对边相等），而 E、F到直线CD的距离关系...

更简洁的思路：通过角度计算可得∠ECF + ∠ECF' = 180°，从而E、C、F三点共线 ✅

⚠️ 关键区分——易错点提醒
  • "对角线相等的平行四边形" ✓ 是矩形
  • "对角线相等的四边形" ✗ 不一定是矩形（如：等腰梯形对角线也相等！）
  • 判定定理2的前提必须是平行四边形！不能省略！
  • 判定定理1（三个直角）适用于任意四边形；判定定理2仅适用于平行四边形

📌 本节小结

  • 判定定理2：对角线相等的平行四边形 → 矩形
  • 证明关键：利用SSS证三角形全等 → 底角相等 → 同旁内角互补 → 直角
  • 实际应用：木工测量对角线检验是否为矩形
  • 矩形的两种判定：
    • 方法1：有一个角是直角的平行四边形（从"角"的角度）
    • 方法2：对角线相等的平行四边形（从"对角线"的角度）
