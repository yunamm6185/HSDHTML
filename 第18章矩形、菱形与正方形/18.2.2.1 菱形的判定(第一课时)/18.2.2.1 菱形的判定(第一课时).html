第18章 矩形、菱形与正方形

18.2.2 菱形的判定（第一课时）——四条边都相等的四边形是菱形

🎯 学习目标

1掌握菱形的定义判定法：有一组邻边相等的平行四边形是菱形
2掌握判定定理1：四条边都相等的四边形是菱形，并理解其证明思路
3会用尺规作图作一个四条边都相等的四边形
4能运用菱形的判定方法进行证明和计算
一、复习回顾——菱形的性质

📌 菱形的性质回顾

在上一节课中，我们学习了菱形的两个重要性质：

性质名称 内容表述
性质定理1 菱形的四条边都相等
$\because$ 四边形ABCD是菱形 $\therefore AB = BC = CD = DA$
性质定理2 菱形的对角线互相垂直平分
$\because$ 四边形ABCD是菱形 $\therefore AC \perp BD$ 且互相平分

菱形ABCD：四边相等 + 对角线互相垂直平分

二、情境导入——从性质到判定

💭 逆向思考：如何判断一个四边形是菱形？

性质 ⇄ 判定 的思维转换

我们已经知道：

• 菱形 → 四条边相等 ✅ （这是性质）

• 菱形 → 对角线互相垂直 ✅ （这也是性质）

那么反过来呢？

• 如果一个四边形四条边都相等，它一定是菱形吗？❓

• 如果一个四边形三条边相等呢？❓❓

🔑 这就是今天我们要研究的核心问题：
"满足什么条件的四边形是菱形？"

三、探究活动——几条边相等能确定菱形？

🔬 探究：边数与菱形的关系

我们知道"菱形的四条边相等"，但反过来：需要几条边相等才能确定它是菱形呢？

✅ 四条边相等 → 是菱形！

❌ 三条边相等 → 不一定！

⚠️ 重要发现：
  • 四条边都相等的四边形一定是菱形 ✓
  • 只有三条边相等的四边形不一定是菱形！（如右图反例：筝形/凸四边形，三边相等但对边不一定平行）
  • 关键区别：四条边相等 ⇒ 对边也相等 ⇒ 平行四边形 ⇒ 而且邻边也相等 ⇒ 菱形
四、菱形的判定方法（一）——定义法

⭐ 定义判定法

有一组邻边相等的平行四边形叫做菱形。

几何语言：

∵ 四边形ABCD是平行四边形，且 $AB = AD$（或任意一组邻边相等）

∴ ▱ABCD是菱形 🟣

💡 思路分析：

  1. 先证平行四边形（两组对边分别平行 / 一组对边平行且相等 / 对角线互相平分等）
  2. 再证一组邻边相等
  3. 根据定义即可得出菱形的结论
五、菱形的判定方法（二）——判定定理1

⭐ 判定定理1：四条边都相等的四边形是菱形。

几何语言：

∵ $AB = BC = CD = DA$

∴ 四边形ABCD是菱形

📝 定理1的证明思路

已知：在四边形ABCD中，$AB = BC = CD = DA$

求证：四边形ABCD是菱形

证明：

∵ $AB = CD$，$BC = DA$（已知四条边相等）

∴ 四边形ABCD是平行四边形
（两组对边分别相等的四边形是平行四边形 —— 平行四边形判定定理2）

又∵ $AB = BC$（已知四条边相等）

即：▱ABCD有一组邻边相等

∴ ▱ABCD是菱形
（有一组邻边相等的平行四边形是菱形 —— 定义法） ✅

💡 证明路线总结：
四边相等 → 两组对边分别相等 → 平行四边形 → 再加邻边相等 → 菱形

七、例题精讲

例4 矩形中点连线问题

题目：如图，在矩形ABCD中，E、F、G、H分别是四条边的中点。连接EF、FG、GH、HE。问：四边形EFGH是什么形状？请说明理由。

解：四边形EFGH是菱形。

理由如下：

第一步：证明EFGH是平行四边形

连接矩形ABCD的两条对角线AC和BD，交于点O。

∵ E、F分别是AD、CD的中点

∴ EF是△ACD的中位线（三角形中位线定理）

∴ $EF = \frac{1}{2}AC$ 且 $EF \parallel AC$

同理，GH是△ABC的中位线，∴ $GH = \frac{1}{2}AC$ 且 $GH \parallel AC$

∴ $EF = GH$ 且 $EF \parallel GH$

∴ 四边形EFGH是平行四边形（一组对边平行且相等）

第二步：证明邻边相等

∵ E、H分别是AD、AB的中点

∴ EH是△ABD的中位线

∴ $EH = \frac{1}{2}BD$ 且 $EH \parallel BD$

又∵ 四边形ABCD是矩形

∴ $AC = BD$（矩形的对角线相等）

∴ $EF = \frac{1}{2}AC = \frac{1}{2}BD =$ EH

∴ ▱EFGH有一组邻边EF = EH

∴ ▱EFGH是菱形（定义法） ✅

💡 关键知识点链：
矩形对角线相等 (AC=BD) → 三角形中位线定理 ($EF=\frac{1}{2}AC$, $EH=\frac{1}{2}BD$) → EF=EH → 邻边相等的平行四边形是菱形

八、课堂练习

练习1 ⭐

如图，在平行四边形ABCD中，已知 $AB = BC$。求证：平行四边形ABCD是菱形。

证明：

∵ 四边形ABCD是平行四边形（已知），且 $AB = BC$（已知）

即：平行四边形ABCD有一组邻边相等

∴ 平行四边形ABCD是菱形（根据菱形的定义）✅

（此题为最基础的定义法应用）

练习2 ⭐⭐

已知：在四边形ABCD中，$AB=BC=CD=DA=5\,\text{cm}$。判断四边形ABCD的形状，并说明理由。

解：四边形ABCD是菱形。

理由：

∵ $AB = CD = 5\,\text{cm}$，$BC = DA = 5\,\text{cm}$（已知）

∴ $AB = CD$，$BC = DA$

∴ 四边形ABCD是平行四边形（两组对边分别相等）

又∵ $AB = BC$（已知）

∴ ▱ABCD是菱形（有一组邻边相等的平行四边形是菱形）✅

📝 另一种表述方式：直接使用判定定理1——
∵ $AB = BC = CD = DA$ ∴ 四边形ABCD是菱形（四条边都相等的四边形是菱形）✅

练习3 ⭐⭐⭐

如图，在□ABCD中，对角线AC的垂直平分线分别交AD于点E，交BC于点F。求证：四边形AFCE是菱形。

证明：

设对角线AC的垂直平分线与AC交于点O。

∵ EF是AC的垂直平分线

∴ OA = OC（垂直平分线的性质），且 $EF \perp AC$

又∵ 四边形ABCD是平行四边形

∴ $AD \parallel BC$

∴ ∠EAO = ∠FCO，∠AEO = ∠CFO

∴ △AOE ≌ △COF（AAS）

∴ OE = OF

又∵ OA = OC

∴ 四边形AFCE的对角线AC与EF互相平分

∴ 四边形AFCE是平行四边形

又∵ $EF \perp AC$（EF是AC的垂直平分线）

∴ 平行四边形AFCE是菱形（对角线互相垂直的平行四边形是菱形）✅

💡 提示：本题也可通过证明EA=EC来利用定义法。因为E在AC的垂直平分线上，所以EA=EC；同理FA=FC。

⚠️ 易错点提醒
  • "四边相等"≠"三边相等" —— 只有四条边都相等才是菱形的充分条件，三条边相等不够！必须画出反例图形加深理解
  • 判定时注意前提条件 —— "一组邻边相等的平行四边形是菱形"，前提必须是平行四边形！如果只是普通四边形，即使邻边相等也不一定是菱形
  • 不要混淆性质和判定 —— 性质是"菱形→..."（正向推导），判定是"...→菱形"（逆向推理），两者方向不同！
  • 多种判定方法的灵活选择 —— 根据已知条件灵活选择定义法或定理1，哪种方便用哪种

🧠 数学思想·核心素养

1. 逆向思维：从"菱形的性质是什么"反向思考"什么条件可以推出菱形"，体会数学命题与其逆命题的关系。性质定理的逆命题往往成为判定定理。

2. 类比迁移：类比矩形的判定方法学习菱形的判定，体会特殊四边形判定的共性思路：要么基于定义（加条件到基本图形），要么直接给出充分条件。

3. 反例意识：通过构造"三条边相等但不是菱形"的反例，培养严谨的数学论证习惯，理解"必要但不充分"的概念雏形。

4. 数形结合：尺规作图中将代数条件"边长为a"转化为几何操作（圆弧截取等长线段），体现了数与形的完美统一。
