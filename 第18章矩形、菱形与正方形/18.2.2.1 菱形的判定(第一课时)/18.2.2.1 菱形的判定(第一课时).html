第十八章 矩形、菱形与正方形 18.2 菱形

18.2.2 菱形的判定

第 1 课时：判定定理 1
学习目标

本节课的学习目标：

1. 理解菱形的判定定理1：四条边都相等的四边形是菱形。

2. 能运用判定定理1解决简单问题。

3. 经历探索菱形判定方法的过程，培养逻辑推理能力。

复习回顾

我们已经知道，有一组邻边相等的平行四边形是菱形，这是菱形的定义。

我们可以根据定义来判定一个四边形是不是菱形。除此之外，还能找到其他的判定方法吗？

菱形是特殊的平行四边形，具有如下性质：

1. 四条边都相等；

2. 两条对角线互相垂直。

这些性质，对我们寻找判定菱形的方法有什么启示呢？

思考

对于一般的四边形，如何寻找判定它是不是菱形的方法呢？

由菱形的性质"四条边都相等"，你可能会想到：如果一个四边形的四条边都相等，那么它肯定是一个菱形。试着画一画，与周围的同学讨论，猜一猜结论是否成立。

试一试

如图18.2.10，作一个四条边都相等的四边形。

作法：

（1）作两条相等的线段 ABABAB、ADADAD；

（2）分别以点 BBB 和点 DDD 为圆心、ABABAB 长为半径作弧，两弧相交于点 CCC；

（3）连结 BCBCBC、CDCDCD。

四边形 ABCDABCDABCD 即为所要求作的四边形。

观察你所画的图形，它是菱形吗？

答：是菱形。

∵AB=CD\because AB = CD∵AB=CD，AD=BCAD = BCAD=BC（作图可知四边相等），

∴\therefore∴ 四边形 ABCDABCDABCD 是平行四边形（两组对边分别相等的四边形是平行四边形）。

又 ∵AB=BC\because AB = BC∵AB=BC（邻边相等），

∴\therefore∴ 平行四边形 ABCDABCDABCD 是菱形（菱形的定义）。

菱形的判定定理 1

判定定理 1

四条边都相等的四边形是菱形。

你能证明这个结论吗？

已知：在四边形 ABCDABCDABCD 中，AB=BC=CD=DAAB = BC = CD = DAAB=BC=CD=DA。

求证：四边形 ABCDABCDABCD 是菱形。

证明：∵AB=CD\because AB = CD∵AB=CD，BC=DABC = DABC=DA，

∴\therefore∴ 四边形 ABCDABCDABCD 是平行四边形（两组对边分别相等的四边形是平行四边形）。

又 ∵AB=BC\because AB = BC∵AB=BC（邻边相等），

∴\therefore∴ 平行四边形 ABCDABCDABCD 是菱形（菱形的定义）。

这里的条件能否再减少一些呢？有三条边相等的四边形是菱形吗？试着画一画，相信你很快会发现，这个结论是不成立的。

例题解析

例 4

如图18.2.11，在矩形 ABCDABCDABCD 中，点 EEE、FFF、GGG、HHH 分别是四条边的中点。试问：四边形 EFGHEFGHEFGH 是什么图形？并说明理由。

分析　四边形 EFGHEFGHEFGH 的四条边分别属于矩形四个角上的三角形，如果能够证明这四个三角形全等，那么就可以利用菱形的判定定理1，得出四边形 EFGHEFGHEFGH 是菱形。

你能说出完整的证明过程吗？

证明：∵\because∵ 四边形 ABCDABCDABCD 是矩形，

∴AB=CD\therefore AB = CD∴AB=CD，AD=BCAD = BCAD=BC，∠A=∠B=∠C=∠D=90∘\angle A = \angle B = \angle C = \angle D = 90^{\circ}∠A=∠B=∠C=∠D=90∘。

∵E\because E∵E、FFF、GGG、HHH 分别是四条边的中点，

∴AE=BE=CG=DG\therefore AE = BE = CG = DG∴AE=BE=CG=DG，AH=DH=BF=CFAH = DH = BF = CFAH=DH=BF=CF。

∴△AEH≅△BFE≅△CGF≅△DHG\therefore \triangle AEH \cong \triangle BFE \cong \triangle CGF \cong \triangle DHG∴△AEH≅△BFE≅△CGF≅△DHG（SAS）。

∴EH=FE=GF=HG\therefore EH = FE = GF = HG∴EH=FE=GF=HG。

∴\therefore∴ 四边形 EFGHEFGHEFGH 是菱形（四条边都相等的四边形是菱形）。

练习

1. 你还记得做过的剪纸探索吗？如图，将一张矩形的纸对折，再对折，然后沿着虚线剪下，打开，你发现这是一个特殊的平行四边形——菱形。现在你能说明其中的理由吗？

理由：将矩形纸对折再对折后，剪下的四边形四条边是由同一条折痕剪出的，

∴\therefore∴ 打开后四边形的四条边都相等。

∴\therefore∴ 这个四边形是菱形（四条边都相等的四边形是菱形）。

2. 如图，在四边形 ABCDABCDABCD 中，AD∥BCAD \parallel BCAD∥BC，AB=ADAB = ADAB=AD，∠BAD\angle BAD∠BAD 的平分线 AEAEAE 交 BCBCBC 于点 EEE，连结 DEDEDE。求证：四边形 ABEDABEDABED 是菱形。

证明：∵AE\because AE∵AE 平分 ∠BAD\angle BAD∠BAD，∴∠BAE=∠DAE\therefore \angle BAE = \angle DAE∴∠BAE=∠DAE。

∵AD∥BC\because AD \parallel BC∵AD∥BC，∴∠DAE=∠AEB\therefore \angle DAE = \angle AEB∴∠DAE=∠AEB。

∴∠BAE=∠AEB\therefore \angle BAE = \angle AEB∴∠BAE=∠AEB，∴AB=BE\therefore AB = BE∴AB=BE。

又 ∵AB=AD\because AB = AD∵AB=AD，∴AD=BE\therefore AD = BE∴AD=BE。

又 ∵AD∥BE\because AD \parallel BE∵AD∥BE，∴\therefore∴ 四边形 ABEDABEDABED 是平行四边形。

又 ∵AB=AD\because AB = AD∵AB=AD，∴\therefore∴ 平行四边形 ABEDABEDABED 是菱形。

3. 如图，在 □ABCD\square ABCD□ABCD 中，点 PPP 是对角线 ACACAC 上的一点，PE⊥ABPE \perp ABPE⊥AB，PF⊥ADPF \perp ADPF⊥AD，垂足分别为点 EEE、FFF，且 PE=PFPE = PFPE=PF。问：□ABCD\square ABCD□ABCD 是菱形吗？为什么？

答：□ABCD\square ABCD□ABCD 是菱形。

理由：∵PE⊥AB\because PE \perp AB∵PE⊥AB，PF⊥ADPF \perp ADPF⊥AD，且 PE=PFPE = PFPE=PF，

∴\therefore∴ 点 PPP 在 ∠BAD\angle BAD∠BAD 的平分线上（角平分线性质的逆定理）。

∴AC\therefore AC∴AC 平分 ∠BAD\angle BAD∠BAD，即 ∠BAC=∠DAC\angle BAC = \angle DAC∠BAC=∠DAC。

∵\because∵ 四边形 ABCDABCDABCD 是平行四边形，∴AB∥CD\therefore AB \parallel CD∴AB∥CD，AD∥BCAD \parallel BCAD∥BC。

∴∠BAC=∠DCA\therefore \angle BAC = \angle DCA∴∠BAC=∠DCA，∠DAC=∠BCA\angle DAC = \angle BCA∠DAC=∠BCA。

∴∠DCA=∠DAC\therefore \angle DCA = \angle DAC∴∠DCA=∠DAC，∴AD=CD\therefore AD = CD∴AD=CD。

∴□ABCD\therefore \square ABCD∴□ABCD 是菱形（有一组邻边相等的平行四边形是菱形）。

课堂小结
  • 菱形的定义：有一组邻边相等的平行四边形是菱形。
  • 菱形的判定定理1：四条边都相等的四边形是菱形。
  • 判定定理1可通过"先证平行四边形，再证邻边相等"来证明。
