📐 18.1.2.1 矩形的判定（第一课时）——三个角是直角的四边形

🌟 情境导入：木工师傅的智慧

在日常生活中，我们经常见到各种矩形的物品，如门窗、桌面、书本等。木工师傅在制作这些矩形物件时，有一套独特的检验方法。


🔨 木工师傅的经验：

制作一个门框时，木工师傅先钉好四条边，然后用角尺（直角尺）去测量三个角。如果三个角都是直角，他就可以断定这个四边形是矩形了。


❓ 思考：为什么只需要检验三个角是直角就能确定是矩形？这背后蕴含着怎样的数学道理？今天我们就来探究这个问题！

🎯 学习目标

  • 理解并掌握矩形的判定方法："有三个角是直角的四边形是矩形"
  • 能够运用该判定定理进行简单的推理和证明
  • 了解尺规作图方法：作一个三个角都是直角的四边形
  • 体会从"性质"到"判定"的逆向思维过程

📚 复习回顾：矩形的性质 → 判定

我们已经学过的矩形性质：
  1. 定义：有一个角是直角的平行四边形叫做矩形
  2. 性质：矩形的四个角都是直角
  3. 性质：矩形的对角线相等

逆向思考：既然"矩形→四个角都是直角"，那么反过来呢？

  • 四个角都是直角的四边形 → 是矩形吗？（显然是的）
  • 三个角都是直角的四边形 → 是矩形吗？（？）
  • 两个角都是直角的四边形 → 是矩形吗？（？）

这就是我们今天要研究的问题：用几个直角来判定矩形。

🔬 探究活动：三个直角能决定第四个吗？

图1：探索三个直角对四边形的影响

关键发现：

在四边形ABCD中，若∠A = ∠B = ∠D = 90°：

  • ∠A + ∠D = 180° → AD ∥ BC（同旁内角互补，两直线平行）
  • ∠A + ∠B = 180° → AB ∥ DC（同旁内角互补，两直线平行）
  • 两组对边分别平行 → 四边形ABCD是平行四边形
  • 又∵ ∠A = 90° → 平行四边形ABCD是矩形
  • 同时可得 ∠C = 90°（平行四边形邻角互补）

📐 尺规作图：作一个三个角都是直角的四边形

下面我们用尺规作图的方法，逐步作出一个三个角为直角的四边形。

图2：尺规作图分步动画演示

步骤 操作说明 几何原理
①画射线AB，过A作AD⊥AB∠DAB = 90°
②在AB上取点B，过B作BC⊥AB∠ABC = 90°
③在AD上取点D，过D作DC⊥AD∠ADC = 90°
④BC与DC相交于点C，连接各边四边形ABCD有三个直角→是矩形
⭐ 矩形判定定理1

📜 定理内容

有三个角是直角的四边形是矩形

📝 几何语言

在四边形 $ABCD$ 中，
$\because \angle A = \angle B = \angle D = 90^{\circ}$
$\therefore$ 四边形 $ABCD$ 是矩形

✍️ 完整证明

已知：四边形$ABCD$中，$\angle A = \angle B = \angle D = 90^{\circ}$

求证：四边形$ABCD$是矩形

证明：

$\because \angle A + \angle D = 90^{\circ} + 90^{\circ} = 180^{\circ}$

$\therefore AD \parallel BC$ （同旁内角互补，两直线平行）   ⋯⋯①

$\because \angle A + \angle B = 90^{\circ} + 90^{\circ} = 180^{\circ}$

$\therefore AB \parallel DC$ （同旁内角互补，两直线平行）   ⋯⋯②

由①、②可知，四边形$ABCD$是平行四边形（两组对边分别平行的四边形是平行四边形）

又$\because \angle A = 90^{\circ}$

$\therefore$ 平行四边形$ABCD$是矩形（有一个角是直角的平行四边形是矩形）

证毕。✅

图3：证明示意图

📝 例题精讲

例1：如图，在四边形$ABCD$中，$\angle ABC = \angle ADC = 90^{\circ}$，且$AB \parallel CD$，$AB = CD$。求证：四边形$ABCD$是矩形。

例1 示意图

证明：

$\because AB \parallel CD$ 且 $AB = CD$

$\therefore$ 四边形$ABCD$是平行四边形（一组对边平行且相等的四边形是平行四边形）

$\because \angle ABC = 90^{\circ}$

$\therefore$ 平行四边形$ABCD$是矩形（有一个角是直角的平行四边形是矩形）

证毕。✅

💡 方法提示：本题也可以通过"三直角"来证：先证出$\angle BAD=90^\circ$，则已有三个直角，从而得矩形。两种思路都可行！

✏️ 课堂练习

练习1：判断下列说法是否正确，并说明理由：

（1）有三个角是直角的四边形是矩形；

（2）有两个角是直角的四边形是矩形；

（3）有三个角是直角的平行四边形是矩形；

（4）四个角都是直角的四边形是矩形。

答案：

（1）正确。这正是本节课学习的判定定理1。

（2）不正确。反例：直角梯形有2个直角但不是矩形。

（3）正确。但这是多余的，因为平行四边形只要一个直角就够了。

（4）正确。四直角→三直角→满足判定定理1。

练习2：如图，在四边形$ABCD$中，$O$是对角线$AC$与$BD$的交点，$\angle DAB = \angle ABC = 90^{\circ}$，$BO = DO$，$AO = CO$。求证：四边形$ABCD$是矩形。

证明：

$\because AO = CO$ 且 $BO = DO$

$\therefore$ 四边形$ABCD$是平行四边形（对角线互相平分的四边形是平行四边形）

$\because \angle DAB = 90^{\circ}$

$\therefore$ 平行四边形$ABCD$是矩形（有一个角是直角的平行四边形是矩形）

另解：也可利用∠DAB=∠ABC=90°，结合平行四边形性质推出∠ADC=90°，得到三直角。

练习3：如图所示，工人师傅在做窗框时，先做好框架$ABCD$，然后测量了$\angle A$、$\angle B$、$\angle C$三个角，发现都是$90^{\circ}$。请问：他能确定这个窗框是矩形吗？为什么？

答：能确定。

因为四边形$ABCD$中，$\angle A = \angle B = \angle C = 90^{\circ}$，即有三个角是直角。

根据矩形的判定定理1：有三个角是直角的四边形是矩形，所以这个窗框是矩形。

这就是木工师傅在实际操作中使用的方法！

📝 课堂小结

  • 矩形的判定方法（本课重点）： 有三个角是直角的四边形 → 矩形
  • 证明思路链： 三直角 → 两组同旁内角互补 → 两组对边平行 → 平行四边形 → 有一直角 → 矩形
  • 矩形的两种判定途径：
    1. 定义法：先证平行四边形，再证有一个直角
    2. 判定定理1：直接证三个角为直角
  • 实际应用：木工师傅测量三个角检验矩形工件
