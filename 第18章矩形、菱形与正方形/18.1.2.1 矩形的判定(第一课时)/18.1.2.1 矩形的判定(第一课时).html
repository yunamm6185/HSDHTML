第十八章 矩形、菱形与正方形 18.1 矩形

18.1.2 矩形的判定

第 1 课时：判定定理 1 与判定定理 2
学习目标

本节课的学习目标：

1. 探索并掌握矩形的判定定理1：有三个角是直角的四边形是矩形。

2. 探索并掌握矩形的判定定理2：对角线相等的平行四边形是矩形。

3. 能运用矩形的判定方法解决简单问题。

情景导入

我们已经知道，有一个角是直角的平行四边形是矩形——这是矩形的定义。我们可以依此判定一个四边形是不是矩形。除此之外，我们能否找到其他判定矩形的方法呢？

矩形具有如下特殊性质：① 四个角都是直角；② 两条对角线相等。这些性质对我们寻找判定矩形的方法有什么启示？

知识铺垫 / 复习回顾

1. 矩形定义：有一个角是直角的平行四边形是矩形。

2. 矩形性质定理1：矩形的四个角都是直角。

3. 矩形性质定理2：矩形的对角线相等。

思考：这些性质的逆命题是否都成立？

新知探索：判定定理 1

试一试

由矩形的性质"四个角都是直角"反思：如果一个四边形的四个角都是直角，那它肯定是矩形。但条件能否再减少一些？三个角是直角的四边形是矩形吗？

作图——作一个三个角都是直角的四边形：

(1) 任意作两条互相垂直的线段 $AB$、$AD$；

(2) 过点 $B$ 作垂直于 $AB$ 的直线 $l$；

(3) 过点 $D$ 作垂直于 $AD$ 的直线 $m$，与直线 $l$ 相交于点 $C$。

四边形 $ABCD$ 即为所要求作的四边形。观察你所作的图形，它是一个矩形吗？

四边形 $ABCD$ 中，$\angle A = \angle B = \angle C = 90^\circ$，由四边形内角和为 $360^\circ$ 可得 $\angle D = 90^\circ$。且由垂直关系可证对边平行，从而是平行四边形。因此它是矩形。
矩形的判定定理 1

判定定理 1

有三个角是直角的四边形是矩形。

已知：在四边形 $ABCD$ 中，$\angle A = \angle B = \angle C = 90^\circ$。

求证：四边形 $ABCD$ 是矩形。

证明：$\because \angle A = \angle B = 90^\circ$，$\therefore \angle A + \angle B = 180^\circ$。

$\therefore AD \parallel BC$（同旁内角互补，两直线平行）。

同理，$\because \angle B = \angle C = 90^\circ$，$\therefore AB \parallel DC$。

$\therefore$ 四边形 $ABCD$ 是平行四边形。

又 $\because \angle A = 90^\circ$，$\therefore$ 四边形 $ABCD$ 是矩形（矩形的定义）。

新知探索：判定定理 2

思考：从对角线的角度

对角线相等，是矩形所特有的性质。那么从对角线的角度，你可以得到关于矩形判定的什么猜想？

猜想：如果一个平行四边形的两条对角线相等，那么这个平行四边形是矩形。

试一试

作一个对角线相等的平行四边形：

(1) 任意作两条相交的直线，交点记为 $O$；

(2) 以点 $O$ 为圆心、适当长为半径作弧，在两条直线上分别截取相等的四条线段 $OA$、$OB$、$OC$、$OD$；

(3) 顺次连结所得的四点。

四边形 $ABCD$ 的两条对角线相等且互相平分，即 $AC = BD$。

和你的同伴交流一下，看看这个平行四边形是不是矩形。

矩形的判定定理 2

判定定理 2

对角线相等的平行四边形是矩形。

已知：如图 18.1.10，四边形 $ABCD$ 是平行四边形，$AC = BD$。

求证：四边形 $ABCD$ 是矩形。

证明：$\because$ 四边形 $ABCD$ 是平行四边形，

$\therefore AB = DC$（平行四边形对边相等）。

又 $\because AC = BD$（已知），$BC = CB$（公共边），

$\therefore \triangle ABC \cong \triangle DCB$（SSS）。

$\therefore \angle ABC = \angle DCB$。

又 $\because AB \parallel DC$，$\therefore \angle ABC + \angle DCB = 180^\circ$。

$\therefore \angle ABC = \angle DCB = 90^\circ$。

$\therefore$ 四边形 $ABCD$ 是矩形（有一个角是直角的平行四边形是矩形）。

实际应用：这一判定方法在日常生活中经常被应用。例如，木工师傅在制作门框或其他矩形形状的物体时，常用测量对角线的方法来检验产品是否符合要求。

例题精讲

例4 如图 18.1.11，点 $O$ 是矩形 $ABCD$ 的对角线 $AC$ 与 $BD$ 的交点，$E$、$F$、$G$、$H$ 分别是 $AO$、$BO$、$CO$、$DO$ 上的一点，且 $AE = BF = CG = DH$。
求证：四边形 $EFGH$ 是矩形。

证明：$\because$ 四边形 $ABCD$ 是矩形，

$\therefore AC = BD$；

$AO = CO = \frac{1}{2}AC$，$BO = DO = \frac{1}{2}BD$。

$\therefore AO = BO = CO = DO$。

$\because AE = BF = CG = DH$，

$\therefore OE = OF = OG = OH$。

$\therefore$ 四边形 $EFGH$ 是平行四边形。

$\because EO + OG = FO + OH$，

$\therefore EG = FH$。

$\therefore$ 四边形 $EFGH$ 是矩形（对角线相等的平行四边形是矩形）。

方法归纳

矩形的判定方法总结

现在我们有三种方法判定矩形：

1. 定义法：有一个角是直角的平行四边形是矩形。

2. 角判定法：有三个角是直角的四边形是矩形。（判定定理1）

3. 对角线判定法：对角线相等的平行四边形是矩形。（判定定理2）

练一练

1. 如图，$AB$、$CD$ 是 $⊙O$ 的两条直径，四边形 $ACBD$ 是矩形吗？证明你的结论。

是矩形。$\because AB$、$CD$ 是直径，$\therefore OA = OB = OC = OD$。

$\therefore$ 四边形 $ACBD$ 是平行四边形（对角线互相平分）。

又 $\because AB = CD$（均为直径），$\therefore$ 四边形 $ACBD$ 是矩形（对角线相等的平行四边形是矩形）。

2. 如图，在 $\square ABCD$ 中，$\angle 1 = \angle 2$。此时，四边形 $ABCD$ 是矩形吗？为什么？

是矩形。$\because$ 四边形 $ABCD$ 是平行四边形，对角线互相平分，

$\therefore OA = OC$，$OB = OD$。

$\because \angle 1 = \angle 2$，即 $\angle OAB = \angle OBA$，

$\therefore OA = OB$（等角对等边）。

$\therefore OA = OB = OC = OD$，即 $AC = BD$。

$\therefore$ 四边形 $ABCD$ 是矩形（对角线相等的平行四边形是矩形）。

3. 如图，在四边形 $ABCD$ 中，$BF = DE$，$AC$ 与 $EF$ 互相平分并相交于点 $O$，$\angle B = 90^\circ$。
求证：四边形 $ABCD$ 是矩形。

证明：$\because AC$ 与 $EF$ 互相平分于点 $O$，

$\therefore$ 四边形 $AFCE$ 是平行四边形（对角线互相平分的四边形是平行四边形）。

$\therefore AF = CE$，$AF \parallel CE$。

$\because BF = DE$，

$\therefore AF + BF = CE + DE$，即 $AB = CD$。

又 $\because AF \parallel CE$，$\therefore AB \parallel CD$。

$\therefore$ 四边形 $ABCD$ 是平行四边形（一组对边平行且相等）。

又 $\because \angle B = 90^\circ$，

$\therefore$ 四边形 $ABCD$ 是矩形（有一个角是直角的平行四边形是矩形）。

  • 掌握了判定定理1：有三个角是直角的四边形是矩形。
  • 掌握了判定定理2：对角线相等的平行四边形是矩形。
  • 能从角的角和对角线的角度灵活判定矩形。
