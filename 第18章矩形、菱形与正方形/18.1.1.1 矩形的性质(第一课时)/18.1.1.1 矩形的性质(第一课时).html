18.1.1 矩形的性质

第 1 课时：矩形的定义与对称性、性质定理
学习目标

本节课的学习目标：

1. 理解矩形的定义，知道矩形是特殊的平行四边形。

2. 探索并掌握矩形的对称性（中心对称 + 轴对称）。

3. 掌握矩形的两个性质定理：四个角都是直角；对角线相等。

4. 能运用矩形性质进行简单的计算和推理。

情景导入

在日常生活中，我们随处可见矩形的身影——黑板、门窗、课桌、书本……它们都是我们早已熟悉的长方形，即矩形。矩形具有哪些独特的性质？它与平行四边形有什么关系？让我们一起来探索吧！

新知探索

试一试

给你一个平行四边形相邻两边的长，你能利用尺规作图作出这个平行四边形吗？现在将你与同伴所作的图形放在一起，发现它们都是平行四边形，相邻两边的长也一样，但两邻边之间的夹角有大有小。

我们若改变平行四边形的内角，使其一个内角恰好为直角，就得到一种特殊的平行四边形——也就是我们早已熟悉的长方形，即矩形。

矩形的定义：有一个角是直角的平行四边形叫做矩形。
$\because$ 四边形 $ABCD$ 是平行四边形，且 $\angle A = 90^\circ$，$\therefore$ 四边形 $ABCD$ 是矩形。
做一做

动手操作

用一张长方形纸片，通过对折观察：

(1) 矩形有几条对称轴？分别在哪里？

(2) 对称轴的交点在哪里？

发现：矩形有2条对称轴，分别是通过两组对边中点的直线；对称轴的交点就是对角线的交点——矩形的对称中心。
思考：探索矩形的性质

观察与发现

作为一种特殊的平行四边形，矩形具有平行四边形的一般性质，同时也具有一些特殊性质。观察下面的矩形，将你的发现填入下表：

对称性边角对角线
平行四边形一般性质中心对称对边平行且相等对角相等互相平分
矩形的特殊性质轴对称（2条）—四个角都是直角相等
矩形的性质定理 1

性质定理 1

矩形的四个角都是直角。

已知：如图，四边形 $ABCD$ 是矩形，$\angle A = 90^\circ$。

求证：$\angle A = \angle B = \angle C = \angle D = 90^\circ$。

证明：$\because$ 四边形 $ABCD$ 是矩形，

$\therefore$ 四边形 $ABCD$ 是平行四边形（矩形定义）。

$\therefore AD \parallel BC$，$\therefore \angle A + \angle B = 180^\circ$（两直线平行，同旁内角互补）。

又 $\because \angle A = 90^\circ$，$\therefore \angle B = 90^\circ$。

同理，$\angle C = \angle A = 90^\circ$，$\angle D = \angle B = 90^\circ$（平行四边形对角相等）。

$\therefore \angle A = \angle B = \angle C = \angle D = 90^\circ$。

矩形的性质定理 2

性质定理 2

矩形的对角线相等。

已知：如图，在矩形 $ABCD$ 中，对角线 $AC$ 和 $BD$ 交于点 $O$。

求证：$AC = BD$。

证明：在矩形 $ABCD$ 中，

$\because \angle ABC = \angle DCB = 90^\circ$（性质定理1），

$AB = DC$（平行四边形对边相等），

$BC = CB$（公共边），

$\therefore \triangle ABC \cong \triangle DCB$（SAS）。

$\therefore AC = DB$（全等三角形对应边相等）。

推论：矩形的对角线相等且互相平分。

例题

例1 如图，矩形 $ABCD$ 被两条对角线分成四个小三角形，如果这四个小三角形周长的和是 $86\text{ cm}$，矩形的对角线长是 $13\text{ cm}$，那么该矩形的周长是多少？

分析：四个小三角形分别是 $\triangle AOB$、$\triangle BOC$、$\triangle COD$、$\triangle DOA$。

它们的周长之和可表示为矩形四边与对角线部分的和。

解：$\because \triangle AOB$、$\triangle BOC$、$\triangle COD$ 和 $\triangle DOA$ 这四个小三角形周长的和为 $86\text{ cm}$，

$\therefore AB + BC + CD + DA + 2(OA + OB + OC + OD) = 86$。

$\therefore AB + BC + CD + DA + 2(AC + BD) = 86$。

又 $\because AC = BD = 13$（矩形的对角线相等），

$\therefore AB + BC + CD + DA = 86 - 2(AC + BD)$

$= 86 - 4 \times 13 = 34\text{ cm}$。

答：该矩形的周长是 $34\text{ cm}$。

方法归纳

矩形的性质总结

矩形是特殊的平行四边形，它具有：

一般性质：对边平行且相等、对角相等、对角线互相平分、中心对称。

特殊性质：

① 四个角都是直角（性质定理1）

② 对角线相等（性质定理2）

③ 是轴对称图形，有2条对称轴（通过对边中点的直线）

练一练

1. 在矩形 $ABCD$ 中，对角线 $AC$ 与 $BD$ 相交于点 $O$，试找出图中相等的线段和相等的角。

相等的线段：$AB = CD$，$AD = BC$；$AC = BD$；$OA = OC = OB = OD$。

相等的角：$\angle A = \angle B = \angle C = \angle D = 90^\circ$；对顶角相等；内错角相等等。

2. 矩形 $ABCD$ 的两条对角线相交于点 $O$，$\angle AOD = 120^\circ$。求证：$AC = 2AB$。

证明：$\because$ 矩形的对角线相等且互相平分，$\therefore OA = OB = OC = OD$。

$\because \angle AOD = 120^\circ$，$\therefore \angle AOB = 60^\circ$（邻补角）。

$\therefore \triangle AOB$ 是等边三角形，$\therefore OA = OB = AB$。

$\because AC = 2OA$，$\therefore AC = 2AB$。

3. 如图，在矩形 $ABCD$ 中，点 $E$ 在边 $CD$ 上。将该矩形沿 $AE$ 折叠，恰好使点 $D$ 落在边 $BC$ 上的点 $F$ 处。如果 $\angle BAF = 60^\circ$，求 $\angle DAE$ 的大小。

解：$\because$ 四边形 $ABCD$ 是矩形，$\therefore \angle DAB = 90^\circ$。

$\because \angle BAF = 60^\circ$，$\therefore \angle DAF = \angle DAB - \angle BAF = 30^\circ$。

$\because$ 将矩形沿 $AE$ 折叠，点 $D$ 落在点 $F$ 处，

$\therefore AE$ 是 $\angle DAF$ 的平分线（折叠对称性）。

$\therefore \angle DAE = \dfrac{1}{2} \angle DAF = \dfrac{1}{2} \times 30^\circ = 15^\circ$。

课堂小结
  • 矩形定义：有一个角是直角的平行四边形叫做矩形。
  • 对称性：矩形既是中心对称图形，也是轴对称图形（2条对称轴）。
  • 性质定理1：矩形的四个角都是直角。
  • 性质定理2：矩形的对角线相等。
