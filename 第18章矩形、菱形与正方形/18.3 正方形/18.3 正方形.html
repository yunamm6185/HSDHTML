第18章 矩形、菱形与正方形

🔷 18.3 正方形 —— 特殊的平行四边形

🎯 学习目标

1理解正方形的两种定义方式，知道正方形是最特殊的平行四边形
2掌握正方形的性质：边、角、对角线、对称性
3掌握正方形的判定方法
4理解平行四边形、矩形、菱形、正方形之间的包含关系
5能运用正方形的性质解决相关计算和证明问题
一、情境导入——矩形与菱形的"相遇"

🤔 一个有趣的思考：

矩形 + 菱形 = ？

回顾我们已经学过的图形：

📐 矩形：有一个角是直角的平行四边形
→ 特征：四个角都是直角，对角线相等

💎 菱形：有一组邻边相等的平行四边形
→ 特征：四条边都相等，对角线互相垂直

💡 关键问题：如果一个四边形既是矩形又是菱形，它会有什么特征？

→ 四个直角 + 四边相等 = 🔷 正方形！

二、正方形的定义

📌 正方形的定义（两种等价表述）

定义一：

有一个角是直角的菱形叫做正方形。

菱形 → 加上"有一个直角" → 正方形

定义二：

有一组邻边相等的矩形叫做正方形。

矩形 → 加上"邻边相等" → 正方形

从菱形或矩形出发得到正方形

⚠️ 要点理解：

  1. 正方形首先必须是平行四边形（这是基础）
  2. 定义一：在菱形的基础上增加"有一个角是直角"
    → 菱形已有四边相等，加上直角后自然成为正方形
  3. 定义二：在矩形的基础上增加"有一组邻边相等"
    → 矩形已有四个直角，加上邻边相等后自然成为正方形
  4. 两种定义是等价的！它们描述的是同一个图形
  5. 正方形是最特殊的平行四边形，它同时具备矩形和菱形的所有性质
三、正方形的性质

🔬 探究：正方形有哪些性质？

既然正方形既是矩形又是菱形，那么它应该同时具有两者的所有性质。让我们来系统梳理：

性质类别 矩形的性质 菱形的性质 ✨ 正方形（兼具两者）
边的性质 对边平行且相等 四条边都相等 ✅ 四条边都相等
角的性质 四个角都是直角 对角相等，邻角互补 ✅ 四个角都是直角
对角线 相等且互相平分 互相垂直平分 ✅ 相等、垂直、互相平分
对称性 轴对称（2条对称轴）
中心对称		 轴对称（2条对称轴）
中心对称		 ✅ 轴对称（4条对称轴）
✅ 中心对称

正方形ABCD及其对角线（点击画布可切换显示模式）

🔷 正方形的性质总结 🔷

① 四条边都相等   |   ② 四个角都是直角（$90°$）
③ 对角线相等、互相垂直平分   |   ④ 是轴对称图形（有4条对称轴），也是中心对称图形

⚡ 特别注意：正方形的两条对角线把正方形分成四个全等的等腰直角三角形！

每个等腰直角三角形的锐角都是 $\angle OAB = \angle OBA = \angle OBC = \angle OCD = \angle ODA = 45°$

四、知识结构图——四边形的"家族谱"

🌳 平行四边形的分类体系

正方形处于整个体系的"最核心"位置——它是矩形和菱形的交集！

平行四边形分类关系图（正方形 = 矩形 ∩ 菱形）

🔍 关系解读：

  • 平行四边形是最大的集合，包含了矩形和菱形
  • 矩形：有一个角是直角的平行四边形（强调"角"的特征）
  • 菱形：有一组邻边相等的平行四边形（强调"边"的特征）
  • 正方形：既是矩形又是菱形（同时满足"角"和"边"的特殊条件）
  • 因此：所有的正方形都是矩形，也都是菱形，但反之不成立
五、例题精讲

📝 例1：求正方形中对角线形成的角度

【题目】已知正方形 $ABCD$，对角线 $AC$ 与 $BD$ 相交于点 $O$。
求 $\angle OAB$ 和 $\angle OBA$ 的度数。

正方形ABCD及对角线交点O

【解答】

解：∵ 四边形 $ABCD$ 是正方形

∴ 对角线 $AC$ 平分 $\angle DAB$（正方形的对角线平分内角）

又∵ $\angle DAB = 90°$（正方形的四个角都是直角）

∴ $\angle OAB = \frac{1}{2}\angle DAB = \frac{1}{2} \times 90° = \mathbf{45°}$

同理，$\angle OBA = \frac{1}{2}\angle ABC = \frac{1}{2} \times 90° = \mathbf{45°}$

【另解】（利用三角形性质）：

∵ $OA = OB$（正方形对角线互相平分且相等，故 $OA = OB = OC = OD$）

∴ $\triangle OAB$ 是等腰三角形

又∵ $\angle OAB + \angle OBA + \angle AOB = 180°$ 且 $\angle AOB = 90°$（对角线互相垂直）

∴ $\angle OAB = \angle OBA = \frac{180° - 90°}{2} = \mathbf{45°}$

答：$\angle OAB = 45°$，$\angle OBA = 45°$

六、讨论活动——如何检验一个四边形是正方形？

🗣️ 课堂讨论：检验正方形的方法

问题情境：小明用纸剪了一个四边形，他想知道这个四边形是否为正方形。三位同学提出了不同的检验方法……

👦 小明的方法："用尺子量一下四条边，如果四条边长度一样，就是正方形！"

❌ 分析：不可靠！菱形的四条边也相等，但不一定是正方形。缺少了"直角"条件。

👦 小兵的方法："量一下两条对角线的长度，如果对角线相等，就是正方形！"

❌ 分析：也不可靠！矩形的对角线也相等，但不一定是正方形。缺少了"邻边相等"条件。

👧 小英的方法："先比较对角线是否相等（判断是不是矩形），再比较对角线相互分成的四条线段是否相等（判断是不是菱形）。两个条件都满足才是正方形！"

✅ 分析：正确！这实际上是在验证既是矩形又是菱形。

✅ 结论：要证明一个四边形是正方形，需要证明它同时满足矩形和菱形的判定条件。
检验方法 操作步骤 能否判定正方形？ 原因
只量边长 四条边相等？ ❌ 不能 菱形也满足
只量对角线 对角线相等？ ❌ 不能 矩形也满足
先证矩形再证菱形 ① 对角线相等 → 矩形
② 邻边相等 → 菱形		 ✅ 能 同时满足两者
直接测量 ① 四边相等 + ② 一个角为90° ✅ 能 符合定义

📋 正方形的判定方法

判定方法一（从菱形出发）：

有一个角是直角的菱形是正方形

判定方法二（从矩形出发）：

有一组邻边相等的矩形是正方形

判定方法三（从平行四边形出发）：

既是矩形（对角线相等/有一个直角）又是菱形（邻边相等/对角线垂直）的平行四边形是正方形

判定方法四（从一般四边形出发）：

四条边相等且有一个角是直角的四边形是正方形

八、课堂练习

✏️ 练习1

题目：下列说法中，正确的个数是（    ）

① 所有的正方形都是矩形；   ② 所有的正方形都是菱形；
③ 所有的矩形都是正方形；   ④ 所有的菱形都是正方形。

A. 0个    B. 1个    C. 2个    D. 4个

答案：C（2个正确）

• ① 正确 —— 正方形是特殊的矩形（有一个直角的菱形=正方形，而菱形是平行四边形，所以正方形也是矩形）

• ② 正确 —— 正方形也是特殊的菱形

• ③ 错误 —— 矩形的邻边不一定相等（如长方形不是正方形）

• ④ 错误 —— 菱形的角不一定是直角

✏️ 练习2

题目：正方形的对角线长为 $4\sqrt{2}$ cm，则它的边长为 _______ cm，面积为 _______ cm²。

答案：边长 = 4 cm，面积 = 16 cm²

解析：

设正方形边长为 $a$，根据勾股定理：

$a^2 + a^2 = (对角线)^2$

$2a^2 = (4\sqrt{2})^2 = 32$

$a^2 = 16$ → $a = 4$ (cm)

面积 $S = a^2 = 4^2 = 16$ (cm²)

✏️ 练习3

题目：如图，正方形 $ABCD$ 中，$E$ 是 $BC$ 上的一点，$AE \perp CE$ 于点 $O$。
若 $BE = 3$，$CE = 4$，求正方形的边长。

答案：正方形的边长为 7

解析：

在 Rt△BEC 中，由勾股定理得：

$BC = \sqrt{BE^2 + CE^2} = \sqrt{3^2 + 4^2} = \sqrt{9+16} = \sqrt{25} = 5$？❌ 不对！

注意：这里 $E$ 在 $BC$ 上，所以 $BE + EC = BC$（如果E在B、C之间）

但实际上本题的几何关系：由于 $AE \perp CE$，结合正方形性质...

正确思路：设正方形边长为 $a$，则 $BC = a$，且 $BE = 3$，$EC = a - 3$（假设E靠近B）
在 Rt△AEC 中：$AE^2 = AB^2 + BE^2 = a^2 + 9$
又 $AE^2 + EC^2 = AC^2$（Rt△AEO）... 此题需根据具体图形分析。

💡 提示：这是一道综合应用题，建议课后仔细推导。关键在于利用正方形的边长相等和对角线性质。

九、课堂小结

📝 本课知识梳理

项目 内容要点
定义 ① 有一个角是直角的菱形  |  ② 有一组邻边相等的矩形
边的性质 四条边都相等（继承自菱形）
角的性质 四个角都是直角（继承自矩形）
对角线 相等、互相垂直平分、每条对角线平分一组对角
对称性 轴对称（4条对称轴：两条对角线 + 两组对边中点连线）+ 中心对称
判定方法 证矩形 + 证菱形（即同时满足两者的条件）
特殊三角形 对角线分成4个全等的等腰直角三角形（每个锐角45°）

💡 数学思想方法：特殊与一般的辩证关系

🔄 从一般到特殊：

平行四边形 → （加直角条件）→ 矩形

平行四边形 → （加邻边相等）→ 菱形

矩形 ∩ 菱形 = 正方形

📐 特殊图形的一般性：

正方形具备平行四边形的所有性质

正方形额外拥有矩形和菱形的全部特性

✨ 这种"层层递进、逐步特殊化"的思想是数学中非常重要的思维方式！
