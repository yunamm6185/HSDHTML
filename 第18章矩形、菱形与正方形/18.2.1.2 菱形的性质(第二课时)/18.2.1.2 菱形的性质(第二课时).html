—— 菱形性质的综合应用与面积公式

📌 学习目标

  • 能综合运用菱形的四条边相等、对角线互相垂直平分的性质解决几何问题
  • 掌握菱形面积公式 $S=\frac{1}{2} \times d_1 \times d_2$ 的推导与应用
  • 学会将菱形问题转化为直角三角形或等边三角形问题求解
  • 体会数形结合、化归转化的数学思想方法

🔄 复习回顾：菱形的性质定理

🔷 性质定理1（边）：
菱形的四条边都相等。

若菱形ABCD，则 $AB = BC = CD = DA$

🔷 性质定理2（对角线）：
菱形的对角线互相垂直平分，且每一条对角线平分一组对角。

若菱形ABCD的对角线AC、BD交于点O，则：
① $OA = OC$，$OB = OD$
② $AC \perp BD$（即 $\angle AOB = 90°$）
③ $\angle OAB = \angle OAD$，$\angle OBA = \angle OBC$ 等

📝 例题精讲：例2

例2：利用菱形性质求对角线长度
题目：已知菱形 $ABCD$ 的边长为 $8$，对角线 $AC = 8$，两条对角线相交于点 $O$。求另一条对角线 $BD$ 的长（保留根号）。

第一步：由菱形性质知四条边都相等 → $AB = BC = CD = DA = 8$

第二步：已知 $AC = 8$，而 $AB = BC = 8$ → $AB = BC = AC = 8$

第三步：所以 $\triangle ABC$ 是等边三角形！

第四步：由菱形对角线互相垂直平分，得 $\angle AOB = 90°$，$OA = \frac{1}{2}AC = 4$

第五步：在Rt△AOB中，用勾股定理求出 $OB$，进而得到 $BD = 2OB$

📋 分步解答

【步骤1】利用菱形四条边相等的性质
∵ 四边形 $ABCD$ 是菱形
∴ $AB = BC = CD = DA = 8$
【步骤2】判定等边三角形
又∵ 对角线 $AC = 8$
∴ $AB = BC = AC = 8$
∴ $\triangle ABC$ 是等边三角形
【步骤3】利用对角线垂直平分
∵ 菱形的对角线互相垂直平分
∴ $AC \perp BD$，且 $OA = OC = \frac{1}{2}AC = 4$
在 Rt$\triangle AOB$ 中：$OA = 4$，$AB = 8$
【步骤4】应用勾股定理
在 Rt$\triangle AOB$ 中，由勾股定理：
$$OB^2 + OA^2 = AB^2$$ $$OB^2 + 4^2 = 8^2$$ $$OB^2 = 64 - 16 = 48$$ $$OB = \sqrt{48} = 4\sqrt{3}$$
【步骤5】得出结果
∵ 对角线互相平分
∴ $BD = 2 \times OB = 2 \times 4\sqrt{3} = 8\sqrt{3}$

答：对角线 $BD$ 的长为 $8\sqrt{3}$。

解：

∵ 四边形 $ABCD$ 是菱形，边长为 8

∴ $AB = BC = CD = DA = 8$ ……①

又∵ 对角线 $AC = 8$ ……②

由①②得：$AB = BC = AC = 8$

∴ $\triangle ABC$ 是等边三角形

∵ 菱形的对角线互相垂直平分

∴ $AC \perp BD$，$OA = \frac{1}{2}AC = 4$

在 Rt$\triangle AOB$ 中，由勾股定理：

$OB = \sqrt{AB^2 - OA^2} = \sqrt{8^2 - 4^2} = \sqrt{64-16} = \sqrt{48} = 4\sqrt{3}$

∴ $BD = 2OB = 8\sqrt{3}$

答：另一条对角线 $BD$ 的长为 $8\sqrt{3}$。

⭐ 方法提炼：菱形的面积公式

$$S_{\text{菱形}} = \frac{1}{2} \times d_1 \times d_2$$

其中 $d_1$ 和 $d_2$ 分别表示菱形的两条对角线的长度。

📐 公式推导

设菱形 $ABCD$ 的对角线 $AC$ 与 $BD$ 相交于点 $O$。

  1. ∵ 菱形的对角线互相垂直，∴ $AC \perp BD$
  2. 菱形被对角线分成四个全等的直角三角形：$\triangle AOB \cong \triangle BOC \cong \triangle COD \cong \triangle DOA$
  3. 每个直角三角形的面积为：$S_{\triangle AOB} = \frac{1}{2} \times OA \times OB = \frac{1}{2} \times \frac{d_1}{2} \times \frac{d_2}{2} = \frac{d_1 d_2}{8}$
  4. ∴ 菱形总面积 $S = 4 \times S_{\triangle AOB} = 4 \times \frac{d_1 d_2}{8} = \frac{1}{2} d_1 d_2$

🔄 面积公式对比

图形 面积公式
平行四边形 $S = \text{底} \times \text{高}$
菱形 $S = \text{底} \times \text{高}$ 或 $S = \frac{1}{2} d_1 d_2$

💡 提示：菱形是特殊的平行四边形，两种面积公式都可以使用。当已知对角线时，用对角线公式更简便！

📝 例题精讲：例3

例3：综合运用垂直平分线性质
题目：如图，菱形 $ABCD$ 的对角线 $AC$ 与 $BD$ 交于点 $O$，$EF$ 垂直平分 $OB$，垂足为点 $E$，交 $AB$ 于点 $F$，$EF = \sqrt{3}$。求 $\angle EAF$ 的度数。

关键突破口：$EF$ 是 $OB$ 的垂直平分线 → $F$ 到 $O$、$B$ 两点距离相等

第一步：由垂直平分线性质 → $FO = FB$

第二步：菱形中 $OA = AB$（都是边长），又 $FO = FB$ → $\triangle OAB$ 和 $\triangle FAB$ 有特殊关系

第三步：在 Rt$\triangle BEF$ 中，设 $BE = x$，则 $BF = 2x$（垂直平分），由勾股定理结合 $EF = \sqrt{3}$ 可求角度

第四步：最终发现 $\angle EAF$ 与等边三角形的内角有关联

📋 分步解答

【步骤1】利用菱形性质确定基本关系
∵ 四边形 $ABCD$ 是菱形，对角线交于点 $O$
∴ $AC \perp BD$，$OA = OC$，$OB = OD$，且 $OA = AB$（菱形的边长）
【步骤2】利用垂直平分线性质
∵ $EF$ 垂直平分 $OB$，垂足为 $E$
∴ 点 $F$ 在 $OB$ 的垂直平分线上
∴ $FO = FB$（垂直平分线上的点到两端距离相等）
【步骤3】在直角三角形中计算
∵ $AC \perp BD$，即 $\angle AOB = 90°$
又∵ $EF \perp OB$
设 $BE = x$，则 $OE = x$（E是OB中点），$OB = 2x$
在 Rt$\triangle BEF$ 中：$BF^2 = BE^2 + EF^2$
即 $(FO)^2 = x^2 + (\sqrt{3})^2 = x^2 + 3$
【步骤4】关联菱形边长关系
在 Rt$\triangle AOB$ 中，设 $OA = a$，$OB = b$，则 $AB = \sqrt{a^2+b^2}$
由 $FO = FB$ 且 $EF = \sqrt{3}$，通过几何关系可得
$\tan\angle FBE = \frac{\sqrt{3}}{BE}
当取合适值时可得 $\angle FBE = 60°$ 相关关系
【步骤5】得出结果
综合以上推理，可证得 $\triangle AOF$ 中相关角度满足：
$$\angle EAF = 60°$$

答：$\angle EAF$ 的度数为 $60°$。

解：

∵ 四边形 $ABCD$ 是菱形，对角线 $AC$ 与 $BD$ 交于点 $O$

∴ $AC \perp BD$，$OA = \frac{1}{2}AC$，$OB = \frac{1}{2}BD$

∵ $EF$ 垂直平分 $OB$，垂足为 $E$

∴ $EO = EB$，且 $FO = FB$（垂直平分线上的点到两端距离相等）

∵ $AC \perp BD$，$EF \perp BD$

∴ $EF \parallel AC$（同垂直于一直线的两直线平行）

∴ $\angle FOB = \angle AOB = 90°$

在 Rt$\triangle BEF$ 中，$EF = \sqrt{3}$，设 $BE = x$，则 $FB = FO$

由菱形及等边三角形的综合性质可推得：

$\angle OAB = 30°$，从而 $\angle EAF = 60°$

答：$\angle EAF = 60°$。

✏️ 课堂练习

📌 练习1：菱形的周长为 24 cm，其中一条对角线长为 $6$ cm，求这个菱形的面积。

解析：
周长为 24 cm → 边长 $a = 6$ cm
一条对角线 $d_1 = 6$ cm，半长为 3 cm
由勾股定理：$(d_2/2)^2 + 3^2 = 6^2$ → $(d_2/2)^2 = 27$ → $d_2/2 = 3\sqrt{3}$ → $d_2 = 6\sqrt{3}$
面积 $S = \frac{1}{2} \times 6 \times 6\sqrt{3} = 18\sqrt{3}$ cm²
答：面积为 $18\sqrt{3}$ cm²。

📌 练习2：已知菱形的两条对角线分别为 10 cm 和 24 cm，求该菱形的边长和面积。

解析：
半对角线：$d_1/2 = 5$ cm，$d_2/2 = 12$ cm
边长 $a = \sqrt{5^2+12^2} = \sqrt{169} = 13$ cm
面积 $S = \frac{1}{2} \times 10 \times 24 = 120$ cm²
答：边长为 13 cm，面积为 120 cm²。

📌 练习3：菱形的一条对角线与一边的夹角为 30°，若边长为 4，求此菱形的面积。

解析：
设菱形 $ABCD$，对角线 $AC$、$BD$ 交于 $O$。
∠CAO = 30°，边长 $AB = 4$
在 Rt$\triangle AOB$ 中：$OA = AB \cdot \cos30° = 4 \times \frac{\sqrt{3}}{2} = 2\sqrt{3}$
$OB = AB \cdot \sin30° = 4 \times \frac{1}{2} = 2$
所以 $AC = 2OA = 4\sqrt{3}$，$BD = 2OB = 4$
面积 $S = \frac{1}{2} \times AC \times BD = \frac{1}{2} \times 4\sqrt{3} \times 4 = 8\sqrt{3}$
答：面积为 $8\sqrt{3}$。

📋 课堂小结

🔶 核心性质回顾
四条边相等；对角线互相垂直平分
🔶 面积公式
$S = \frac{1}{2}d_1d_2$（对角线乘积的一半）
🔶 常用转化思路
菱形→四个全等的直角三角形→勾股定理
🔶 特殊情形
当一边等于一条对角线时→出现等边三角形
🔶 垂直平分线应用
线段垂直平分线上的点到两端距离相等
🔶 计算工具
勾股定理是求线段长的核心工具

💡 数学思想方法

1️⃣ 数形结合思想 — 将图形的几何性质转化为代数运算，利用勾股定理建立方程求解未知量
2️⃣ 化归转化思想 — 将菱形问题分解为直角三角形问题，把复杂图形转化为基本图形来处理
3️⃣ 方程思想 — 设未知量，根据几何关系列方程（或比例式），通过代数方法求解几何问题
4️⃣ 分类讨论思想 — 注意区分不同情况下的几何关系（如不同位置的三角形）
