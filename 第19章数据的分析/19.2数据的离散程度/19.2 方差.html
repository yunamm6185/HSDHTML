从表19.2.1可以看出，天津与新加坡的气温相比，有4个时刻的气温相对高些，有4个时刻的气温相对低些。

比较两地气温的高低，求平均气温是一种常用的方法。

经计算可知，这两地的平均气温相等，都是18.625°C。这能否说明两地的气温情况总体上没有什么差异呢？

直观感受 — 气温波动对比

观察图中的气温变化，你感觉它们有没有差异呢？

🔍 观察要点：

  • 天津的气温波动范围比较大，从11°C到21°C，相差10°C
  • 新加坡的气温波动范围比较小，从25°C到28°C，相差仅3°C
  • 虽然两地平均气温相同，但波动程度明显不同！

📌 概括

  • 比较两组数据时，通常可以先画图，直观地感受一下两组数据的整体特点
  • 即便两组数据的平均数相等，它们还可能在数据的波动大小上表现出差异
  • 因此，不能只限于比较平均数。数据波动小，则平均数更具有代表性

🔍 分析要点：

  • 两人测试成绩的平均数都是8分
  • 成绩的最大值与最小值也都相差4分
  • 但从图可以看出，相比之下，小明的成绩大部分集中在其平均数附近
  • 而小兵的成绩与其平均数的离散程度略大
  • 结论：小明的成绩比较稳定

💭 思考：如何衡量离散程度？

我们已经看出，小兵的测试成绩与其平均数的偏差与小明相比略大。那么如何加以说明呢？

可以直接将各数据与其平均数的差进行累加吗？

📐 离差平方和

尝试计算

在下面的表格中，填写每次成绩与平均成绩的差：

第1次 第2次 第3次 第4次 第5次 求和
小明成绩 7 8 9 8 8 -
每次成绩-平均成绩 7-8=-1 8-8=0 9-8=+1 8-8=0 8-8=0 0
小兵成绩 6 9 7 10 6 -
每次成绩-平均成绩 6-8=-2 9-8=+1 7-8=-1 10-8=+2 6-8=-2 0

发现：两组数据偏差求和的结果都是0！

事实上，一组数据中的每个数与这组数据平均数的差相加之和始终等于0。

⚠️ 既然直接求和不行，那么用什么办法可以从整体上反映各个数据远离平均数的情况呢？

💡 解决方案：平方后再求和

我们可以用"先平均，再求差，然后平方，最后求和"所得到的结果反映一组数据与其平均数的离散程度。

这个结果称为这组数据的离差平方和。

离差平方和的计算

📐 方差的定义

📌 方差的定义

在计算一组数据的离差平方和时，随着数据个数的增多，和通常也会增大。

因此，当两组数据所含数据的个数不同时，直接比较离差平方和显得不公平，还需要平均化。

这样得到的结果称为方差，通常记为 $s^2$。我们通常用方差来衡量一组数据偏离其平均数的情况。

方差公式：

$$s^2 = \frac{1}{n}\left[(x_1-\bar{x})^2 + (x_2-\bar{x})^2 + \cdots + (x_n-\bar{x})^2\right]$$

简记为：$$s^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2$$

📝 计算小明和小兵成绩的方差

小明5次测试成绩的方差：

$s^2 = \frac{1}{5}\left[(7-8)^2 + (8-8)^2 + (9-8)^2 + (8-8)^2 + (8-8)^2\right]$

$= \frac{1}{5}\left[1 + 0 + 1 + 0 + 0\right] = \frac{2}{5} = \textbf{0.4}$

小兵5次测试成绩的方差：

$s^2 = \frac{1}{5}\left[(6-8)^2 + (9-8)^2 + (7-8)^2 + (10-8)^2 + (6-8)^2\right]$

$= \frac{1}{5}\left[4 + 1 + 1 + 4 + 4\right] = \frac{14}{5} = \textbf{2.8}$

结论：小明的成绩比较稳定（方差0.4 < 2.8）

📐 标准差

📌 标准差

方差的算术平方根称为标准差，通常记为 $s$。

$$s = \sqrt{s^2} = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2}$$

标准差也是用来衡量一组数据波动大小的量，它的单位和原数据的单位相同。

📊 极差、方差、标准差的比较

三种度量方式的对比

度量方式 计算方法 特点
极差 最大值 - 最小值 只考虑两端数据，简单但不够全面
方差 $\frac{1}{n}\sum(x_i-\bar{x})^2$ 考虑所有数据的偏离程度，较全面
标准差 $\sqrt{方差}$ 单位与原数据一致，实际应用更方便

🎯 方差的意义

  • 方差越大，数据的波动越大，越不稳定
  • 方差越小，数据的波动越小，越稳定
  • 方差为0，说明所有数据都相等，没有波动
  • 方差和标准差都是用来描述数据离散程度的统计量

练习1：比较方差

比较下列两组数据的方差：

A组：5, 5, 5, 5, 5, 5, 5, 5

B组：2, 3, 5, 6, 4, 7, 5, 8

A组：所有数据都相等，方差 = 0

B组：

平均数 = (2+3+5+6+4+7+5+8)÷8 = 5

方差 = [(2-5)²+(3-5)²+(5-5)²+(6-5)²+(4-5)²+(7-5)²+(5-5)²+(8-5)²]÷8

= [9+4+0+1+1+4+0+9]÷8 = 28÷8 = 3.5

结论：A组的方差 < B组的方差

练习2：计算气温的方差

算一算，第20页问题1中哪个城市该时段气温的离散程度较大？与你从图19.2.1中直观看出的结果一致吗？

天津气温方差：

平均数 = 18.625

方差 = [(11-18.625)²+(14-18.625)²+...+(14-18.625)²]÷8 ≈ 11.7

新加坡气温方差：

平均数 = 18.625

方差 = [(25-18.625)²+(26-18.625)²+...+(26-18.625)²]÷8 ≈ 0.7

结论：天津的方差（11.7）> 新加坡的方差（0.7）

天津气温的离散程度较大，与直观看出的结果一致。

📝 课堂小结

  1. 方差是用来衡量一组数据偏离其平均数的程度的统计量
  2. 方差公式：$s^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2$
  3. 方差的意义：
    • 方差越大，数据波动越大，越不稳定
    • 方差越小，数据波动越小，越稳定
  4. 标准差是方差的算术平方根，单位与原数据相同
  5. 比较两组数据的波动程度，可以用方差或标准差
