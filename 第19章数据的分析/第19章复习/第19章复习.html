二、核心概念与公式

平均数

$\bar{x} = \dfrac{x_1 + x_2 + \cdots + x_n}{n}$

加权平均数

$\bar{x} = \dfrac{x_1f_1 + x_2f_2 + \cdots + x_kf_k}{f_1 + f_2 + \cdots + f_k}$

中位数

按大小排列后，第 $\dfrac{n+1}{2}$ 位的数

方差

$s^2 = \dfrac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2$

标准差

$s = \sqrt{\dfrac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2}$

四、易错点提醒

⚠️ 注意事项

1. 平均数的局限性：当数据中有异常值时，平均数不能很好地代表这组数据。

• 例如：5个人的年龄分别是 8, 8, 9, 9, 80 岁，平均年龄为 22.8 岁，但这不能代表这组人。

2. 中位数的计算：必须先将数据按大小排序。

• 当 n 为偶数时，中位数是中间两个数的平均数。

• 当 n 为奇数时，中位数就是正中间的那个数。

3. 众数的特点：

• 一组数据可能没有众数

• 一组数据可能有一个众数

• 一组数据可能有多个众数

4. 方差的意义：

• 方差越大，数据波动越大

• 方差越小，数据越稳定

• 方差的单位是原数据单位的平方

• 标准差的单位与原数据相同

5. 四分位数：

• 下四分位数（Q₁）处于总体 25% 的位置

• 中位数（Q₂）处于总体 50% 的位置

• 上四分位数（Q₃）处于总体 75% 的位置

五、综合例题

例1：计算统计量（复习题A组第1题）

某班30名学生的考试成绩如下：

67, 78, 86, 75, 75, 78, 68, 70, 80, 69, 75, 86, 82, 74, 79, 75, 88, 82, 86, 74, 82, 83, 75, 88, 80, 85, 88, 80, 76, 100

请计算这次考试全班学生成绩的平均数、中位数、众数和方差。

解：

（1）平均数：

$\bar{x} = \dfrac{67 + 78 + \cdots + 100}{30} = \dfrac{2400}{30} = 80$（分）

（2）中位数：

将数据从小到大排列后，第15、16位都是80分

中位数 $= \dfrac{80 + 80}{2} = 80$（分）

（3）众数：

统计各分数出现次数：75分出现5次，80分出现3次，86分出现3次，88分出现3次

众数 $= 75$（分）

（4）方差：

$s^2 = \dfrac{1}{30}[(67-80)^2 + (78-80)^2 + \cdots + (100-80)^2] \approx 48.67$

例2：平均数与加权平均数（复习题A组第2题）

有两组数据：第一组数据是 2, 4, 6, 8, 10；第二组数据是 1, 3, 5, 7, 9, 11。

（1）先分别求出这两组数据的平均数；

（2）再将这两组数据合并在一起，求合并后这组数据的平均数；

（3）想一想，它是前两个平均数的平均数吗？

解：

（1）第一组的平均数：$\bar{x}_1 = \dfrac{2+4+6+8+10}{5} = \dfrac{30}{5} = 6$

第二组的平均数：$\bar{x}_2 = \dfrac{1+3+5+7+9+11}{6} = \dfrac{36}{6} = 6$

（2）合并后的平均数：

$\bar{x} = \dfrac{2+4+6+8+10+1+3+5+7+9+11}{11} = \dfrac{66}{11} = 6$

（3）不是前两个平均数的平均数。

若用两个平均数的平均数计算：$\dfrac{6+6}{2} = 6$，恰好相等。

但这只是特例！一般情况下，合并后数据的平均数应该是所有数据的总和除以总个数。

正确公式：$\bar{x} = \dfrac{n_1\bar{x}_1 + n_2\bar{x}_2}{n_1 + n_2}$（加权平均）

例3：工程问题（复习题A组第6题）

某个工程队正在修建道路，有2天每天修7m，有3天每天修8m，有4天每天修9m，有1天修10m。

这10天中该工程队平均每天修建道路多少米？

解：

这是求加权平均数的问题：

$\bar{x} = \dfrac{7 \times 2 + 8 \times 3 + 9 \times 4 + 10 \times 1}{2 + 3 + 4 + 1}$

$= \dfrac{14 + 24 + 36 + 10}{10}$

$= \dfrac{84}{10} = 8.4$（米）

答：这10天中该工程队平均每天修建道路8.4米。

七、复习题分层练习

判断题（复习题A组第3题）

判断下列说法是否正确，若不正确，请举出反例。

(1) n个数的平均数就是把这n个数的总和除以n所得的数。

(2) n个数的平均数一定是这n个数中的某一个。

(3) 将n个数由小到大排列后，如果n是奇数，位置在正中间的数就是这n个数的中位数；如果n是偶数，位置在正中间的那两个数的平均数才是这n个数的中位数。

(4) n个数的中位数一定是这n个数中的某一个。

(5) 如果在n个数中某个或某几个数出现的频数最大，那么这个或这几个数就是这n个数的众数；如果找不出这样的数，那么这n个数就没有众数。

(6) 如果n个数中存在众数，那么该众数一定是这n个数中的某一个。

(1) 正确。这就是平均数的定义。

(2) 错误。反例：1, 2, 3 的平均数是2，是其中一数；但 1, 2, 4 的平均数是 $\dfrac{7}{3}$，不是其中任何一个数。

(3) 正确。这就是中位数的定义。

(4) 错误。反例：当n为偶数时，如 1, 2, 3, 4 的中位数是 $\dfrac{2+3}{2}=2.5$，不是其中任何一个数。

(5) 正确。这就是众数的定义。

(6) 正确。众数必须是数据中存在的数。

平均数的性质（复习题A组第5题）

已知一组数据的平均数等于60，判断下列说法是否正确，若不正确，请举出一个反例。

(1) 如果这组数据共有三个，且其中一个大于60，那么必有一个小于60。

(2) 如果这组数据共有四个，且其中两个小于60，那么必有两个大于60。

(1) 正确。

设三个数为 a, b, c，平均数为60，则 a + b + c = 180。

若 a > 60，则 b + c = 180 - a < 120。

假设 b ≥ 60 且 c ≥ 60，则 b + c ≥ 120，矛盾！

所以 b 和 c 中至少有一个小于60。

(2) 错误。

反例：40, 50, 70, 80，平均数为 $\dfrac{40+50+70+80}{4} = 60$

有两个小于60（40和50），但只有两个大于60（70和80），正确。

但如果反例是：30, 50, 70, 90，平均数为60，两个小于60，两个大于60，也是正确的。

实际上这个命题是正确的！反例不存在。

身高比较（复习题B组第2题）

下表给出了某校七年级和九年级部分学生的身高（单位：cm）：

七年级：160, 163, 158, 161, 162, 157, 160, 161, 159, 160, 167, 168, 161, 163, 159, 166, 161, 159, 162, 160

九年级：162, 161, 166, 160, 168, 171, 168, 164, 169, 167, 164, 160, 162, 167, 166, 167, 170, 166, 164, 167

在这些学生中，哪个年级的学生平均身高较高？哪个年级的学生身高的方差较大？

解：

七年级平均身高：$\bar{x}_7 = \dfrac{160+163+\cdots+160}{20} \approx 162.0$（cm）

九年级平均身高：$\bar{x}_9 = \dfrac{162+161+\cdots+167}{20} \approx 166.1$（cm）

结论：九年级学生平均身高较高。

七年级方差：$s_7^2 \approx 8.1$

九年级方差：$s_9^2 \approx 8.4$

结论：九年级学生身高的方差较大，说明九年级学生身高差异略大。

气温变化幅度比较（复习题B组第3题）

比较2021年乌鲁木齐和南宁月平均气温的变化幅度。

城市 1月2月3月4月5月6月 7月8月9月10月11月12月
乌鲁木齐 -12-92121823 2423178-2-10
南宁 131417222627 282827241915

解：

乌鲁木齐：

平均气温 $\bar{x} = \dfrac{-12-9+2+12+18+23+24+23+17+8-2-10}{12} \approx 9.3$（℃）

极差 = 24 - (-12) = 36（℃）

方差 $s^2 \approx 186.2$

南宁：

平均气温 $\bar{x} \approx 21.7$（℃）

极差 = 28 - 13 = 15（℃）

方差 $s^2 \approx 30.2$

结论：乌鲁木齐的月平均气温变化幅度明显大于南宁，这与两地的气候特点有关。乌鲁木齐属于温带大陆性气候，冬冷夏热；南宁属于亚热带季风气候，全年温暖。

成绩总评计算（复习题B组第5题）

某同学这学期四次数学测验成绩依次为90分、76分、88分和92分，期中考试成绩为82分，数学老师说这学期的总评成绩的权重将按平时测验、期中考试和期末考试依次占30%、30%和40%计算。这位同学希望总评成绩能够达到或超过85分，那么期末考试这位同学至少要考多少分？（取整数）

解：

平时测验平均分：$\bar{x}_{平时} = \dfrac{90+76+88+92}{4} = \dfrac{346}{4} = 86.5$（分）

设期末考试成绩为 $x$ 分，则：

总评成绩 $= 86.5 \times 30\% + 82 \times 30\% + x \times 40\%$

$= 25.95 + 24.6 + 0.4x$

$= 50.55 + 0.4x$

要使总评成绩 $\geq 85$：

$50.55 + 0.4x \geq 85$

$0.4x \geq 34.45$

$x \geq 86.125$

答：这位同学期末考试至少要考 87分。

GDP分析（复习题C组第1题）

分析2021年我国31个省市自治区人均GDP的分布情况，计算平均数、中位数和众数，并思考用什么指标来代表比较合适。

分析思路：

（1）利用频数分布表中各组的组中值乘以频数，再除以总数，可估算平均数。

（2）中位数是第16个数据所在组的组中值。

（3）众数是频数最大组的组中值。

结果：

平均数约为 8.5 万元左右

中位数约为 7.2 万元左右

众数约为 6.8 万元左右（频数最大组）

用什么指标代表？

由于数据分布呈现右偏（有少数省市人均GDP特别高，如北京、上海等），平均数会受这些极端值影响而偏大。

因此，用中位数来代表更合适，它能更好地反映大多数省市的一般水平。

八、本章知识要点

📋 数据分析的核心思想

  • 数据的作用：数据对我们了解所考察的对象非常重要，但过多的数据有时反而让我们无法把握数据的重点。
  • 数据分析的两件事：
    ① 制作形象的统计图表，对数据形成整体印象
    ② 计算代表数据的平均数、中位数和众数，以这几个指标概括数据
  • 集中趋势：通过平均数、中位数和众数来反映数据的集中趋势。
  • 离散程度：通过方差、标准差等来反映数据的离散程度。
  • 数据分布：借助四分位数和箱线图，了解数据的分布情况。
  • 工具选择：选取一组数据的代表时，要注意平均数、中位数和众数的适用范围。
九、数学文化

📖 数据可视化的重要性

各式各样的统计图表是分析数据必不可少的工具。只有直观地呈现数据，展示其变化规律和相互关系等特征，才能让阅读的人一目了然，准确高效地获取信息。

常用的统计图表包括：

  • 茎叶图：保留原始数据，便于观察分布
  • 条形图：比较各类别的数量
  • 扇形图：显示各部分占总体的比例
  • 折线图：显示数据的变化趋势
  • 箱线图：显示数据的分布特征和异常值
  • 频数分布直方图：显示数据的分布情况
