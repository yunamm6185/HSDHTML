第19章 数据的分析

19.1.2 加权平均数

🎯 学习目标

1理解加权平均数的概念和计算公式
2理解"权"的意义和作用
3能根据实际问题确定权重并计算
4区分算术平均与加权平均的适用场景

🌟 情境导入

在日常生活中，我们经常会遇到平均数，但发现在有些情况下以前计算平均数的方法并不适用。比如，当不同数据的重要程度不一样时……

📋 问题情境：学期总评成绩

老师在计算学生每学期的总评成绩时，并不是简单地将一个学生的平时成绩与考试成绩相加除以2，而是突出考试成绩的重要性。比如，按照"平时成绩占40%，考试成绩占60%"的比例计算。

如果一个学生某学期的平时成绩为70分，考试成绩为90分，那么他该学期的总评成绩应该是多少？

权重
平时成绩 40% 考试成绩 60%

学期总评成绩 = $70 \times 40\% + 90 \times 60\%$

$= 70 \times 0.4 + 90 \times 0.6 = 28 + 54 = \mathbf{82}$（分）

如果简单求算术平均：$(70+90) \div 2 = 80$（分）—— 没有考虑权重，结果不同！

📖 加权平均数的定义

一般地，若$n$个数$x_1, x_2, \cdots, x_n$的权分别是$w_1, w_2, \cdots, w_n$，则

$\bar{x} = \dfrac{w_1x_1 + w_2x_2 + \cdots + w_nx_n}{w_1 + w_2 + \cdots + w_n}$

叫做这$n$个数的加权平均数。

权的意义：权重反映了各个数据在总体中的重要程度。权重越大，该数据对平均数的影响越大。

算术平均数

(70+90)÷2 = 80

每个数据权重相同（各1/2）

加权平均数

70×0.4+90×0.6 = 82

权重不同：40%、60%

📝 试一试

小青某学期的数学成绩情况为：测验一得89分，测验二得78分，测验三得85分，期中考试得90分，期末考试得87分。如果按照平时成绩占10%、期中成绩占30%、期末成绩占60%的权重，那么小青该学期的总评成绩是多少分？

平时成绩 = (89+78+85)÷3 = 252÷3 = 84分

总评成绩 = $84 \times 10\% + 90 \times 30\% + 87 \times 60\%$

$= 84 \times 0.1 + 90 \times 0.3 + 87 \times 0.6 = 8.4 + 27 + 52.2 = \mathbf{87.6}$（分）

问题1：招聘面试评分

某公司对应聘者A、B、C、D进行面试，并按三个方面给应聘者打分，每个方面满分为20分，最后打分结果如表所示。如果你是人事主管，会录用哪一位应聘者？

应聘者专业知识工作经验仪表形象
A141812
B181611
C171414
D161614

甲同学说：看谁的总分高就录用谁。通过计算可以发现D的总分最高（16+16+14=46分），应被录用。

乙同学说：我有不同意见。三个方面满分都是20分，但按理这三个方面的重要性应该有所不同，比如专业知识就应该比仪表形象更重要。

假设上述三个方面的重要性之比为6:3:1，那么应该录用谁呢？

因为 6:3:1 = 60%:30%:10%，所以专业知识、工作经验与仪表形象这三个方面的权重分别是60%、30%与10%。

这样A的最后得分为

$14 \times 60\% + 18 \times 30\% + 12 \times 10\% = 8.4 + 5.4 + 1.2 = \mathbf{15}$（分）

请你根据这样的权重要求，继续算出另三位应聘者的最后得分。从你的计算结果看，谁应被录用？

B：$(18\times6+16\times3+11\times1)\div10=(108+48+11)\div10=\mathbf{16.7}$（分）

C：$(17\times6+14\times3+14\times1)\div10=(102+42+14)\div10=\mathbf{15.8}$（分）

D：$(16\times6+16\times3+14\times1)\div10=(96+48+14)\div10=\mathbf{15.8}$（分）

结论：B的加权平均分最高（16.7分），应录用B。

权重体现了不同指标的重要程度。专业知识权重最大（60%），B在该项得分最高（18分），因此胜出。

💡 思考：如果这三个方面的重要性之比为10:7:3，此时三个方面的权重各是多少？哪一位应被录用呢？

总份数 = 10+7+3 = 20

专业知识权重 = 10÷20 = 50%

工作经验权重 = 7÷20 = 35%

仪表形象权重 = 3÷20 = 15%

A：$14 \times 50\% + 18 \times 35\% + 12 \times 15\% = 7 + 6.3 + 1.8 = \mathbf{15.1}$（分）

B：$18 \times 50\% + 16 \times 35\% + 11 \times 15\% = 9 + 5.6 + 1.65 = \mathbf{16.25}$（分）

C：$17 \times 50\% + 14 \times 35\% + 14 \times 15\% = 8.5 + 4.9 + 2.1 = \mathbf{15.5}$（分）

D：$16 \times 50\% + 16 \times 35\% + 14 \times 15\% = 8 + 5.6 + 2.1 = \mathbf{15.7}$（分）

结论：B的加权平均分仍然最高（16.25分），应录用B。

问题2：学生睡眠时间调查

某所初中学校通过调查了解到，该校七、八、九年级学生平均每天的睡眠时间依次为9h、8.5h和8h。

(1) 根据这些信息，能否求出该校学生平均每天的睡眠时间？

分析 (1)：如果该校三个年级的学生人数相同，可以求出该校学生平均每天的睡眠时间为

$$\dfrac{9 + 8.5 + 8}{3} = 8.5 \text{（h）}$$

如果该校三个年级的学生人数不相同，用上面的算法就得不到问题的答案了。要想求出该校学生平均每天的睡眠时间，还需要更多的信息。

(2) 如果已知该校七、八、九年级的学生人数分别为350、330、320，能否求出该校学生平均每天的睡眠时间？

(2) 有了各年级的学生人数，就可以求出该校学生每天睡眠的总时间，再除以学生的总人数：

$$\dfrac{9 \times 350 + 8.5 \times 330 + 8 \times 320}{350 + 330 + 320} = \dfrac{3150 + 2805 + 2560}{1000} = \dfrac{8515}{1000} = \mathbf{8.515} \text{（h）}$$

其实，在求解这个问题时，也可以分成三部分计算，即

$$9 \times \dfrac{350}{1000} + 8.5 \times \dfrac{330}{1000} + 8 \times \dfrac{320}{1000} = 8.515 \text{（h）}$$

这是加权平均数，各年级学生人数占总人数的比例就是权重。

(3) 如果已知该校七、八、九年级的学生人数比为4:3:3，能否求出该校学生平均每天的睡眠时间？

(3) 已知各年级的学生人数之比，可以通过加权计算得到该校学生平均每天的睡眠时间为

$$9 \times \dfrac{4}{4+3+3} + 8.5 \times \dfrac{3}{4+3+3} + 8 \times \dfrac{3}{4+3+3} = \mathbf{8.55} \text{（h）}$$

例题：苹果的平均价格

超市里有两种苹果，一种单价为15元/kg，另一种单价为18元/kg。小明妈妈买了单价为15元/kg的苹果1kg，单价为18元/kg的苹果3kg。你认为应该如何计算所买苹果的平均价格？

15元

18元/kg
买了 3kg
54元
=
共 4kg
69元
均价 17.25元/kg

总金额：1×15 + 3×18 = 15 + 54 = 69元

总重量：1 + 3 = 4kg

平均价格 = 69÷4 = 17.25元/kg

❌ 错误做法：(15+18)÷2 = 16.5元/kg —— 没有考虑每次购买量不同！
每次的"购买量"就是权重。
✏️ 练习

1. 本节例1中求"平均每人植树多少棵"还可以通过加权计算，请列式并计算。

假设第一组2人，平均每人植树4棵；第二组3人，平均每人植树3棵；第三组5人，平均每人植树6棵。

加权平均 = $\dfrac{4 \times 2 + 3 \times 3 + 6 \times 5}{2 + 3 + 5} = \dfrac{8 + 9 + 30}{10} = \dfrac{47}{10} = \mathbf{4.7}$（棵）

答：平均每人植树4.7棵。

2. 某人在A商店买了2包饼干，单价是6.20元。走了没多远，看见B商店也有卖这种饼干的，每包5.80元，于是他又买了3包。请先估计一下他买5包饼干的平均价格是小于、等于还是大于6元，然后再算出5包饼干的平均价格，看看你的估计对不对。

估计：因为便宜的买了3包，贵的买了2包，平均价格应该小于6元。

平均价格 = $\dfrac{6.20 \times 2 + 5.80 \times 3}{2+3} = \dfrac{12.4 + 17.4}{5} = \dfrac{29.8}{5} = \mathbf{5.96}$（元）

确实小于6元，估计正确！

3. 一部电梯的最大载重是1000kg。现有13位乘客要搭乘这部电梯，已知其中11位先生的平均体重是80kg，2位女士的平均体重是70kg。请问：他们能否一起安全地搭乘这部电梯？他们的平均体重是多少千克？

总体重 = $11 \times 80 + 2 \times 70 = 880 + 140 = \mathbf{1020}$（kg）

因为1020kg > 1000kg，所以不能一起安全搭乘这部电梯。

平均体重 = $1020 \div 13 \approx \mathbf{78.5}$（kg）

4. 一家小吃店原有三个品种的馄饨，其中菜馅馄饨的售价为10元/碗，鸡蛋馅馄饨的售价为12元/碗，肉馅馄饨的售价为16元/碗。每碗均有10个馄饨。该店老板准备推出混合馄饨，请帮她解决以下问题：

(1) 如果每碗有3个菜馅馄饨、3个鸡蛋馅馄饨和4个肉馅馄饨，那么混合馄饨每碗定价多少合适？

(2) 如果菜馅馄饨、鸡蛋馅馄饨、肉馅馄饨的个数之比为3:2:5，那么混合馄饨每碗定价多少合适？

(3) 如果菜馅馄饨、鸡蛋馅馄饨、肉馅馄饨的个数之比为1:1:3，那么混合馄饨每碗定价多少合适？

每个馄饨的价格：菜馅1元，鸡蛋馅1.2元，肉馅1.6元。

(1) 每碗定价 = $3 \times 1 + 3 \times 1.2 + 4 \times 1.6 = 3 + 3.6 + 6.4 = \mathbf{13}$（元）

(2) 总份数=10，每碗定价 = $3 \times 1 + 2 \times 1.2 + 5 \times 1.6 = 3 + 2.4 + 8 = \mathbf{13.4}$（元）

(3) 总份数=5，每碗定价 = $1 \times 1 + 1 \times 1.2 + 3 \times 1.6 = 1 + 1.2 + 4.8 = \mathbf{7}$（元）—— 但每碗应有10个馄饨，所以实际应为2碗的比例，定价 = $2 \times 7 = \mathbf{14}$（元），或直接按10个计算：2个菜馅、2个鸡蛋馅、6个肉馅，定价 = $2 \times 1 + 2 \times 1.2 + 6 \times 1.6 = 2 + 2.4 + 9.6 = \mathbf{14}$（元）

📝 课堂小结

  • 定义：$\bar{x} = \dfrac{w_1x_1 + w_2x_2 + \cdots + w_nx_n}{w_1 + w_2 + \cdots + w_n}$
  • 权的意义：反映各数据在总体中的重要程度
  • 权的形式：可以是百分比、比例、频数等
  • 算术平均 vs 加权平均：当所有权重相等时，加权平均数 = 算术平均数
