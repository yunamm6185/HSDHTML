📊 19.1.2 加权平均数

🌟 情境导入

在日常生活中，我们经常会与平均数打交道，但发现在有些情况下以前计算平均数的方法并不适用。请看下面的例子：

📋 问题情境

情境：小明的学期总评成绩是这样计算的：平时成绩占30%，期中考试占30%，期末考试占40%。

如果小明的平时成绩是85分，期中考试78分，期末考试82分，那么他的学期总评成绩是多少？

权重比例示意

分析：

这不是简单的算术平均！每个成绩的重要程度（权重）不同。

学期总评成绩 = $85 \times 30\% + 78 \times 30\% + 82 \times 40\%$

$= 85 \times 0.3 + 78 \times 0.3 + 82 \times 0.4$

$= 25.5 + 23.4 + 32.8$

$= 81.7$（分）

📖 加权平均数的定义

一般地，若$n$个数$x_1, x_2, \cdots, x_n$的权分别是$w_1, w_2, \cdots, w_n$，则

$\bar{x} = \frac{w_1x_1 + w_2x_2 + \cdots + w_nx_n}{w_1 + w_2 + \cdots + w_n} = \frac{\sum_{i=1}^{n}w_ix_i}{\sum_{i=1}^{n}w_i}$

叫做这$n$个数的加权平均数。

权的意义：权重反映了各个数据在总体中的重要程度。权重越大，该数据对平均数的影响越大。

算术平均数

$\bar{x} = \frac{85+78+82}{3} = \frac{245}{3} \approx 81.67$

每个数据的权重相同（都是$\frac{1}{3}$）

加权平均数

$\bar{x} = 85 \times 0.3 + 78 \times 0.3 + 82 \times 0.4 = 81.7$

权重不同：30%、30%、40%

例1：学生睡眠时间调查

问题：某所初中学校通过调查了解到，该校七、八、九年级学生平均每天的睡眠时间分别是9小时、8.5小时、8小时。

(1) 如果三个年级的学生人数相同，能否求出该校学生平均每天的睡眠时间？

(2) 如果三个年级的人数比是4:3:3，求该校学生平均每天的睡眠时间。

解：

(1) 如果三个年级人数相同，则权重相同，都是$\frac{1}{3}$。

平均睡眠时间 = $\frac{9 + 8.5 + 8}{3} = \frac{25.5}{3} = 8.5$（小时）

(2) 如果人数比是4:3:3，则权重不同。

平均睡眠时间 = $\frac{9 \times 4 + 8.5 \times 3 + 8 \times 3}{4 + 3 + 3}$

$= \frac{36 + 25.5 + 24}{10} = \frac{85.5}{10} = 8.55$（小时）

例2：苹果的平均价格

问题：小明去超市买了三次苹果：第一次买了10千克，每千克2元；第二次买了5千克，每千克3元；第三次买了5千克，每千克4元。求小明购买苹果的平均价格。

三次购买苹果情况

分析：

总金额 = $10 \times 2 + 5 \times 3 + 5 \times 4 = 20 + 15 + 20 = 55$（元）

总重量 = $10 + 5 + 5 = 20$（千克）

平均价格 = $\frac{55}{20} = 2.75$（元/千克）

对比：如果直接求三种价格的平均：

$\frac{2 + 3 + 4}{3} = 3$（元/千克）❌ 错误！

因为每次购买的重量不同，应该用重量作为权重计算加权平均。

购买次数 价格（元/kg） 重量（kg） 金额（元） 权重
第一次 2 10 20 $\frac{10}{20} = 50\%$
第二次 3 5 15 $\frac{5}{20} = 25\%$
第三次 4 5 20 $\frac{5}{20} = 25\%$
合计 - 20 55 100%

✏️ 课堂练习

练习：某公司招聘员工，笔试成绩占40%，面试成绩占60%。甲、乙两位应聘者的成绩如下表：

应聘者 笔试成绩 面试成绩
甲 90 75
乙 80 85

谁的最终成绩更高？

解：

甲的最终成绩 = $90 \times 0.4 + 75 \times 0.6 = 36 + 45 = 81$（分）

乙的最终成绩 = $80 \times 0.4 + 85 \times 0.6 = 32 + 51 = 83$（分）

因此，乙的最终成绩更高。

📝 课堂小结

  • 加权平均数的定义：$\bar{x} = \frac{w_1x_1 + w_2x_2 + \cdots + w_nx_n}{w_1 + w_2 + \cdots + w_n}$
  • 权的意义：反映各数据在总体中的重要程度
  • 权的形式：可以是百分比、比例、频数等
  • 与算术平均数的关系：当所有权重相等时，加权平均数 = 算术平均数
  • 应用场景：成绩计算、商品平均价格、人员调配等
