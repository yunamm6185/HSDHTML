第12章 全等三角形

12.2.5 斜边直角边 (HL)

🎯 学习目标

✔掌握直角三角形全等的“斜边直角边”判定方法
✔能运用HL进行几何推理证明
✔理解HL是SSA的特殊情形

我们已经知道，对于两个三角形，如果有“边边角”分别相等，那么不能保证这两个三角形全等。在两个直角三角形中，当斜边和一条直角边分别相等时，也属于“边边角”相等的情况，这时这两个直角三角形是否全等呢？

做一做

如图12.2.24，已知线段 $a, b(b > a)$，试作 $\mathrm{Rt}\triangle ABC$，使 $\angle B = 90^{\circ}$，$BC = a$，$AC = b$。

图12.2.24 已知线段 $a, b$
图12.2.25 作法示意图

作法：
(1) 作线段 $BC$，使 $BC = a$；
(2) 作 $\angle CBM = 90^{\circ}$；
(3) 以点 $C$ 为圆心、线段 $b$ 的长为半径作圆弧，交射线 $BM$ 于点 $A$；
(4) 连结 $AC$。
如图12.2.25，$\triangle ABC$ 即为所要求作的直角三角形。

把你作的直角三角形与其他同学作的直角三角形进行比较，或剪下你作的直角三角形，放到其他同学作的直角三角形上，看看是否完全重合。所作的直角三角形都全等吗？
换两条线段，试试看，是否有同样的结论？

由以上操作，可以发现它们完全重合，所作的直角三角形都全等。

等待比较...
💡 分别调整两个直角三角形的直角边和斜边(cm)，系统自动计算另一直角边。当斜边和任意一条直角边分别相等(误差<0.08)时，根据HL判定全等。试试看：让两个三角形的斜边相等，且一条直角边相等(包括交叉相等，如△ABC的AB=△DEF的DF)，即可验证全等！

通过动态实验可以发现：当两个直角三角形的斜边和一条直角边分别相等时，它们能够完全重合，形状大小完全相同。这验证了定理“斜边和一条直角边分别相等的两个直角三角形全等”（HL）。交互工具支持交叉相等（如左边直角边等于右边另一直角边），更全面体现HL的本质。

于是可得判定直角三角形全等的又一个定理！

斜边和一条直角边分别相等的两个直角三角形全等。（简写成“斜边直角边”或“HL”）

这是一个定理，以后会给出它的证明。

例8

如图12.2.26，$AC = BD$，$\angle C = \angle D = 90^{\circ}$。求证：$BC = AD$。

图12.2.26

证明：
$\because \angle C = \angle D = 90^{\circ}$ (已知)，
$\therefore \triangle ABC$ 和 $\triangle BAD$ 都是直角三角形。
在 $\mathrm{Rt}\triangle ABC$ 和 $\mathrm{Rt}\triangle BAD$ 中，
$\because AB = BA$ (公共边)，
$AC = BD$ (已知)，
$\therefore \mathrm{Rt}\triangle ABC \cong \mathrm{Rt}\triangle BAD$ (HL)。
$\therefore BC = AD$ (全等三角形的对应边相等)。

思考

为什么HL能判定直角三角形全等？你能结合前面学习的“边角边”或“角角边”给出证明吗？

事实上，HL定理的严格证明需要用到一些新的知识，我们将在后续学习中逐步揭示。目前，我们可以通过实验操作和直观感受来接受这一结论。这体现了直角三角形在全等判定中的特殊性——正是由于直角的存在，使“边边角”这种一般情形下不成立的判定在直角三角形中成了真命题。具体证明将在我们学习更多几何知识后给出。

📖 读一读

至此，我们已经学习了关于全等三角形的四个判定方法：SAS、ASA、AAS、SSS，以及直角三角形的特殊判定HL。HL是“边边角”在直角三角形中成立的特例，体现了数学中“特殊化”的思想。

✏️ 练习

1. 如图，在 $\triangle ABC$ 中，点 $D$ 为 $BC$ 的中点，$DE \perp AB$，$DF \perp AC$，点 $E$、$F$ 为垂足，$DE = DF$。求证：$\mathrm{Rt}\triangle BED \cong \mathrm{Rt}\triangle CFD$。

证明：
由 $D$ 是 $BC$ 中点得 $BD = CD$。
在 $\mathrm{Rt}\triangle BED$ 和 $\mathrm{Rt}\triangle CFD$ 中，
$\because DE = DF$ (已知)，$BD = CD$ (已证)，
$\therefore \mathrm{Rt}\triangle BED \cong \mathrm{Rt}\triangle CFD$ (HL)。

2. 如图，$AC = AD$，$\angle C = \angle D = 90^{\circ}$。求证：$BC = BD$。

证明：
在 $\mathrm{Rt}\triangle ABC$ 和 $\mathrm{Rt}\triangle ABD$ 中，
$\because AC = AD$ (已知)，$AB = AB$ (公共边)，
$\therefore \mathrm{Rt}\triangle ABC \cong \mathrm{Rt}\triangle ABD$ (HL)。
$\therefore BC = BD$。

3. 如图，有两个长度相同的滑梯，左边滑梯的高度 $AC$ 与右边滑梯水平方向的跨度 $DF$ 相等，这两个滑梯的倾斜角 $\angle CBA$ 与 $\angle EFD$ 的大小有什么关系？说说你的想法和理由。

$\angle CBA = \angle EFD$。
理由：由题意，$BC = EF$（滑梯长度相同），$AC = DF$，且$\angle BAC = \angle EDF = 90^{\circ}$。
在 $\mathrm{Rt}\triangle ABC$ 和 $\mathrm{Rt}\triangle DEF$ 中，
$\because BC = EF$，$AC = DF$，
$\therefore \mathrm{Rt}\triangle ABC \cong \mathrm{Rt}\triangle DEF$ (HL)。
$\therefore \angle CBA = \angle EFD$。

📌 课堂小结

• 斜边和一条直角边分别相等的两个直角三角形全等（HL）。
• HL是直角三角形特有的判定方法，也是“边边角”在直角三角形中成立的特例。
• 运用HL时要注意：必须指明是直角三角形，且对应边是斜边和直角边。

🧠 数学思想·核心素养

特殊化思想： HL是一般SSA在直角三角形中的特殊化，体现了从一般到特殊的数学思想。
转化思想： 利用HL可将直角三角形全等问题转化为简单条件。
推理意识： 严谨的演绎推理是几何学习的核心。
